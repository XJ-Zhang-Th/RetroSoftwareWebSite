让你的图标动起来

　　桌面主题的工具大家知道的肯定不少，这次我向大家介绍的桌面主题――Icon Hear－it,可有很多“新”的地方哦，且听我一一道来。

　　Theme Changer（改变主题）

　　“Theme name”在此列表中“Iconhearit98”和“Sports”两个主题是自带的，其余是对硬盘上已有的各种桌面主题进行扫描后，加入到列表中的。在此可以设置一定时间间隔更换主题，也可以设置为顺序的或者是随机的。

　　Wallpager Changer（改变壁纸）、Screen Savers（改变屏保）

　　分别是设置每隔一段时间更换不同的壁纸、屏保，并可以对壁纸、屏幕保护等内容进行预览。

　　System Options（系统选项）

　　Icon Animation：让“我的电脑”、“网上邻居”等系统图标会动。
　　Bubble Help：当选定一个文件（这个文件最好在桌面上）时，显示该文件名与文件大小。
　　Wallpaper Changer：每隔一定时间自动更换壁纸。
　　Screen Saver Changer：每隔一定时间自动更换屏保。
　　Icon In Tray：在任务栏上显示图标。
　　Run At Startup：每次启动时就运行该程序。
　　Icon Launch Animati：点击桌面上的图标时，图标就会上下翻转。
　　Flvover Effects：点击桌面上的图标时，图标就会凹下去。
　　Icon Shortcut Overl：不显示快捷图标上的小箭头。
　　Minimize/Restore Eff：最小化/还原
　　Start Menu Animati：把开始菜单的标题栏变成活动的。
　　Start Menu Help：当你的鼠标指向“运行”菜单等内容时，会使鼠标所指的文字在标题栏上以滚动方式显示。
　　Menu Always on Top：菜单总出现在最前面。
　　

　　其他心得

　　1.报时的声音文件存放在“安装路径：\tasounds”中，你可以把它们录成中文，只要不改变文件名，这样对那些不懂英语的就方便多了。
　　2.图标会动，是由于除了有一个图标文件外（C:\Program Files\Plus!\Themesiconhearit98），还有一个与它文件名相同的以动画光标（.ani）。只要把动画光标的文件更换就达到了改变图标样式的目的。
　　这个软件可以在(http://www.iconhearit.com)下载。
