歪歪线报――欢乐色彩

　　大家好，歪歪又有一段时间没和大家见面了。原因是到上海去逛了一圈。除了见到以陈三公子、范军为首的系列帅哥外，还在上海的聊天室里遇到了一个叫阿紫的姑娘。可惜歪歪的名气在上海并不大，阿紫非但全无崇拜之意，反以为歪歪是个“黑客”或“大灰狼”，第二天就改掉了她在网上的全部信息（哎！歪歪太没面子了）。
　　闲话不说了，不然头头又要批评我浪费版面了。这期的主题是欢乐色彩。要欢乐，要色彩，当然要用歪歪推荐的图形图像软件了。 Adobe Dimensions (3.2M,试用30天，http://www.adobe.com)是由Adobe开发出来与Photoshop以及Illustrator相搭配的3D绘图软件。它具有如下特性：首先可以快速地制作出基本的3D图像，将Adobe Dimensions中内建的一些3D模型如Square、Sphere、cone等直接加入到3D图像中，再加上一些手动的参数调整，很快就得到一幅栩栩如生的3D图片。Adobe Dimensions的最大好处是与Illustrator等Adobe的软件配合相当好，向量图片可以直接输入到Adobe Dimensions中使用，或许Adobe Dimensions并不是目前在3D图形图像制作上的主流软件，而且效果也不是最顶尖的，但是对习惯使用Adobe Photoshop和Illustrator的朋友们来说是个不错的选择，不需要再花费大把时间去学习如何操作使用，只要将过去使用Adobe产品的经验直接照搬来就可以很快上手了。Canoma（7.2M,试用版只能使用软件中所附的图片，http://www.metacreations.com/）是一个可将照片中的实景变成3D动画的梦幻软件。它能将平面的图像轻易地转换为3D 模型，无论图像是来自于扫描的图片或者数码相机，Canoma都可以将它们在平面图像中所呈现出来的属性，快速转变为3D模型，所以不需要花费很多时间，就可以将整个场景中所需要的模型建立起来。由于Canoma与网页相结合的特性，可以将你的作品通过Metastram或者VRML的方式呈现在网页上，成为可以让更多人看到的画面。运用Canoma所制作出来的3D场景可以输出成OBJ、DXF等标准格式，让其他功能强大的3D软件便于处理。KPT Equalizer（3.4 M，免费但必须到原公司申请密码，http://www.metacreations.com/products/kptx/）这套免费的滤镜可以修整模糊的照片。歪歪一直很想给大家介绍Photoshop外挂滤镜的老大KPT5，可它实在是太贵了，这次介绍的产品还是那个公司出的，品质当然不差。KPT Equalizer之中提供了三组不同功能的Sharpen滤镜，1.Bounded Sharpen： 这个滤镜可以通过设置pixel值及其范围达到你所需要的清晰度，从而避免因为过度的清晰造成的图像失真。2.Contrast Sharpen： 这是一个可以增强局部图像反差效果的滤镜。 3.Equalizer：这个滤镜用于调整图像多个不同明暗层面的清晰程度。Equalizer包含了九种不同的pixel值供你选择：　1、 2、 4、 8、16、 32、 64、128 以及256，选择的数值越大，一次修改图像清晰程度的范围也就越大，它可以让你原本相当模糊的图像变得清晰、自然。PhotoSee Pro（1.3M,60天试用，http://www.acdsystems.com/)是一个专门管理相片光盘的软件，可以帮助你从千百张图片中挑选出特定的图片，它也是一个图库管理的工具软件。你可以用它来建立所有相片光盘的资料库，把光盘依照性质和内容分门别类归纳好。如果你要寻找图片的话，只要在这些建立好的资料库中查找就可以了，甚至连光盘都不必插入到光驱里就可以进行搜寻。对了，介绍了这么多的图形处理软件，最后歪歪还要给大家提供一个处理的素材：“范晓萱――别在我的背后说坏话”(桌面墙纸，http://pcsoft.cpcw.com)，大家不要把我们的小萱萱处理得太过火了，呵呵！

另类软件

　　歪歪上次推出的家庭游戏得到了大家喜欢，这次应大家要求介绍一款来自丹麦的新式俄罗斯方块Geetris（1.1M,30天试用，http://www.dmagic.dk/）。记得歪歪在学习编程的时候，编的第一个软件就是俄罗斯方块，那真是一个很有魅力的游戏。这个俄罗斯方块跟一般的俄罗斯方块差不多，主要就是要尽量消掉从上方不断落下的方块，但游戏规则就有所不同了。大多数落下来的方块都会附带一颗小泡泡，只要落地时在相同的颜色区块内包含有两个小泡泡，那这整个同色区块便可消除掉。有许多朋友来信问如何在Windows中收发传真，其实，Just The Fax（1.2M，30天试用http://www2.viaweb.com/rks/）就是一个在 Windows 下非常 容易使用 的个人传真软件。它使用所见即所得(WYSIWYG)的文字编辑器来帮你编辑 fax 文件及封面，接收及发送传真都非常容易，在接收传真的同时，马上就能快速打印出文件，就像是一台真的传真机一样。歪歪看到新闻说美国航空航天局计划在最近用“月球勘探者”号太空船猛烈撞击月球，以找寻月球的冰层。科学家相信当160公斤重的月球勘探者号以6115公里的时速撞向月球南极时，有很大机会能撞开一个洞。如果冰层确被压在冻土里，这撞击力足以释放出一团水蒸汽。一下子想起了一个小小的登月游戏Moonshot v 1.4 1.（4.7MB ，试用版有功能限制，ftp://zdftp.zdnet.com/pub/private/sWlIB/games/action_games/moonshot.zip）。这个游戏的主题是让你控制的飞船在月球上指定的地方登陆。作者在这个游戏的设计中加入了许多幽默动画效果，并且声音效果很出色。可以利用鼠标来控制游戏。不过这个游戏的难度非常大，歪歪在玩的时候，就发生了很多次太空船撞击月球的事故，可惜一次也没看到水蒸汽。

国产软件

　　软件评测室拿到了最新的WPS2000的家庭版，歪歪当然在第一时间试用了一下，在家庭版中，去掉了许多对于家庭用处不大的功能，价格也降到了让大家容易接受的98元，同时也增设了不少针对家庭用户的新功能：彩色通讯录、家庭自动记账簿、密码日记本、OEM版《金山词霸Ⅲ》英汉词典、数学化学电脑公式符号库、2200张图片库、彩色（黑白）浪漫信纸、动态明信片等15种特殊功能。可能是考虑到家庭用户水平参差不齐，在WPS 2000家庭版中还特地为大家内置了一套多媒体WPS2000教程 。最让歪歪心动的是“浪漫信纸”，当年追女孩的时候，为了找一张她没有见过的漂亮的信纸，歪歪曾经跑遍了整个城市。在WPS2000家庭版中不但提供了现成的彩色浪漫信纸，而且你可以随时用它提供的功能和图片设计新的信纸。嘿嘿，歪歪还在家庭版中发现了棋谱残局，这下可以好好研究研究了，下次好再把露一小脸杀个片甲不留。
　　歪歪成了编辑之后，很惨，每天要输入大量的文字，所以找到了很多输入法，其中就有全面重新改写代码的《智能陈桥》4.0版（http://chhu.yeah.net/），全面支持中文Windows NT系统。具有智能提示、智能自动选词、智能语句输入等智能功能。四万七千词组的海量词库和强大的词库管理功能使增加词组非常方便，还可定义各种混合词组、含GBK汉字词组等，方便用户输出，完善的GBK支持方案使其能输入各种生僻汉字。新增加的支持大五码汉字功能使它可用于港台地区，任意的挂接功能使它可挂接多种汉字输入方法，详尽的使用说明书可解答你在使用中遇到的每一个问题。关于拚音输入法，歪歪终于找到一个支持Windows NT，与《中文之星》带的新拼音类似的中文输入法自由拼音 (http://sww.newhua. com/down/freepy22.zip)。此软件支持中文Win95、Win98、WinNT。特点：1.提供类似《中文之星》带的新拼音的功能。2.支持GBK码，可以输入任何一个汉字。3.可以自己定制，输入各种符号特别方便。　前面讲到了WPS2000的家庭版，其实歪歪自己很喜欢用WPS2000，这篇文章就是在WPS2000上完成的，但很多没有安装WPS 2000的人，在电脑上就看不到歪歪的文章了，WPS2000阅读器（http://www.wps2000.com/）为大家浏览文件打开方便之门。无论你是否安装 WPS2000 ，只要安装了 WPS2000 阅读器，就可以对 WPS 格式的文件进行阅读。歪歪最后要重点推荐的是Magic Set 2.9（http://superr.yeah.net/），这是一个与Tweak UI类似的系统设置软件，所具备的功能比Tweak UI还要多，而且是全中文的，特别为中国用户设计的功能有：调整输入法的顺序、修改当前系统的语言、修改开始菜单的“程序”、“文档”等菜单的名字。可以使用Super Rabbit和Magic Set分别作为注册名和注册码试用，注册费10元,请支持国产共享软件！
