网海情报室9903公告

　　大家好！又见面了，不知上期给大家推荐的读书资源可否好用，有什么建议或要求，可千万别装在肚里，和水手(shuishou@cpcw.com)说吧，水手一定“万死不辞”，哇！太夸张了吧!:-)好了，言归正传，一起来瞧瞧这期水手都给大家带来些什么？
　　★ 九九昆明世博会网站(http://www.expo99.net/)于3月 11日开通，在这个网站中，你可以看到有关世博会的新闻、资料、图片和园艺知识等，对了，该站点还有中奖机会呢！快快行动吧！
　　★ 中国 1999 世界集邮展专辑站点(http://china99.cpi.com.cn/)已经开通。为了迎接8月21日在中国举行的世界集邮展，该站点介绍了邮展概况，参展组和参展商等情况。
　　★ 开发人员站点(http://www.developer.com/)日前开通，这个站点上可说是软件开发人员的乐园，不论你用何种编程语言进行软件开发，你都会在这里找到你感兴趣的东西。
　　★北京 Linux 俱乐部(http://mud.263.net.cn/~linux/aboutus.shtml)给 Linux 爱好者提供了一个交流和互相学习的环境，想找人交谈 Linux 话题，不妨来这里看看。

网上看甲 A

　　中国足球甲级联赛又开赛了，可乐坏了水手这个球迷，更值得高兴的是，今年水手可以不用像往年那样为看足球而发愁了，因为水手找到一个看甲 A 的新去处――因特网。现在就将水手发现的这些网站告诉大家，共同分享吧。
　　★ 新浪网和中国足协联合推出的“中国足球职业联赛”官方网站( http://www.c-league.com.cn/ ) 随着足球联赛的开赛也开通了，球迷们可在这里看到球赛的最新战况、比赛赛程、射手榜等，有关球赛的情况真是应有尽有。
　　★ 中国足球 (http://www.china-soccer.net/)，这里的东东可就多了，除了介绍甲A赛况外，还开设了欧美足坛、海外华将等。看看范志毅、孙继海在海外都“干些什么”。
　　★ 野虎足球场(http://www.5415.com/itsoccer/default.asp)，开设的“足坛快讯”、“资料简索”、“历史新闻”等栏目，内容相当丰富，可让你一饱眼福。
　　除了看以外，水手还给球迷们介绍几个可以自己参与的地方。
　　★ 搜狐举办’99甲A有奖竞猜(http://www.sohu.com/sports/football/luckydraw.htm)，竞猜规则很简单，只用填上每场比赛的比分即可。
　　★ 超级球迷网上联赛(http://www.wuhan.net.cn/wuhan/headnews/test/testindex.htm)只要参加一个小测试，你将有机会获得参赛的球迷金卡！
　　另外水手还给朋友们推荐一些足球俱乐部：
　　★ 大连万达：http://www.zg169.net/~xuyh/wanda.htm
　　★ 上海申花：http://www.netease.com/~shsoccer/
　　★ 北京国安(官方站点)：http://www.modernchina.com/guoan/
　　★ 济南泰山将军 ：http://www.online.jn.sd.cn/third_party/taishan/index.htm
　　★ 武汉红桃 K ：http://mail.wol.com.cn/htk/index.htm
　　水手想和大家说的东东还有很多很多，可是版面不允许再多说，真是意犹未尽，只好下次再和大家聊了。
