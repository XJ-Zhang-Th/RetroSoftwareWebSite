走进Windows 98（上）

　　中文Windows 98/95（以下简称WIN98/95）可以说是目前国内使用最多的电脑操作系统。它亲切的图形界面和高效的性能，使我们能够抛开DOS时代枯燥的命令和烦琐的配置,轻松地使用电脑。下面笔者简单地介绍一下有关WIN98的基本情况，希望能给首次面对WIN98的朋友一些帮助。中文WIN95的主要特点和操作与中文WIN98基本相同，使用WIN95的朋友也可以参考。
　　一、鼠标的使用
　　WIN98中的操作几乎是靠一“鼠”纵横天下。现在我们使用的鼠标以两键鼠标为多，分为左键和右键。在WIN98中鼠标有以下几种操作方法。
　　1.左键单击和双击
　　单击就是按下后立即松开，双击则是快速按两下左键再松开。WIN98中单击的作用是选择图标，双击的作用是打开文档或运行某个程序。
　　2.右键单击
　　在WIN98中，鼠标右键单击的作用是弹出“快捷菜单”。在桌面大部分地方右键单击都可以弹出相应的“快捷菜单”。
　　3.鼠标的“拖动”
　　方法是用鼠标指针点中图标或文件后，按住左或右键不松手直接向某处移动。“拖动”的目的有两种，一是移动程序窗口或图标；二是移动或拷贝文件（文件夹）。
　　另外，在以下的介绍中如果提到鼠标操作但不说明左、右键时，则就是指对左键的操作。
　　二、登录WIN98
　　“登录”是指打开电脑电源进入WIN98，获得对电脑软、硬件资源的使用权过程，简单说就是进入WIN98开始使用电脑的过程。
　　当打开电脑的电源后，电脑开始自检。自检通过后，如果上次是非正常关机，系统会自动运行磁盘扫描程序，修复磁盘错误。然后出现在“蓝天白云下飘扬着Windows 98的微软大旗”，在一声“太空音乐”声中，屏幕显示Windows 98的桌面。
　　WIN98从安全和保存个人桌面环境的设置角度出发，每次进入WIN98时会弹出一个对话框，要求输入用户名和密码，此时用户可以输入自己的信息。如果是新用户，WIN98会将新用户的信息保存起来，以后备用。如果单击一下“取消”按钮，也能登录进入WIN98，只是桌面环境按照系统的默认值设置。
　　有时，我们需要使用WIN98的多重启动菜单。方法是在系统启动时，按F8键将显示如下的多重启动菜单：
　　1.Normal（正常方式）
　　2.Logged（\BOOTLOG.TXT）（生成引导过程记录文件）
　　3.Safe mode(安全模式）
　　4.Safe mode with network support（支持网络的安全模式）
　　5.Step－by－Step confirmation（单步确认）
　　6.Command prompt only（进入命令提示符）
　　7.Safe mode command prompt only（安全模式下的命令提示符）
　　选择Normal方式与没有按下F8键的情况相同。当系统配置发生混乱时，试试选择Safe mode，但此方式不支持网络。如果你对系统配置文件有了解，可以使用Step－by－Step confirmation来确定启动时是否要调入这些文件，而选择Command prompt only则直接进入命令提示符。
　　三、WIN98的“桌面”
　　1.桌面
　　在进入WIN98后，首先看到的就是WIN98的“桌面”。“桌面”是WIN98的基本操作平台，它实际上就是我们运行各类程序、对系统进行各种管理的屏幕区域。在多用户使用时，如果每人的设置不同，“桌面”的外观也不尽相同。
　　2.图标
　　在桌面上分布着一些标有名称的小图案，这些小图案在WIN98中叫“图标”。桌面上的这些图标也叫“快捷方式”图标，因为只要双击它们就可以迅速打开对应的程序或任务。在WIN98桌面上的图标中，除了后来安装的一些应用软件的图标外，最基本的有“我的电脑”、“我的文档”、“回收站”、“Internet Explorer（因特网页浏览器）”、“网上邻居”和“Outlook Express（电子邮件管理器）”等图标。
　　3.任务栏
　　桌面下方是一个“任务栏”，在任务栏上又分为几个区域，从左至右分别是“开始”按钮、快速启动程序区、正在执行任务（程序）显示区和系统任务区。
　　其中“开始”按钮用于打开“开始菜单”；快速启动区存放着4个系统中最常用的功能和程序的图标；正在执行任务栏用于显示WIN98中所有正在运行的窗口最小化图标；系统任务栏用于显示WIN98各系统设备如声音、时钟、汉字输入法的状态以及部分外设（如打印机等）的运行状态。
　　四、WIN98中的“窗口”
　　在WIN98中所有打开的文件夹和正在运行的程序都将显示在一个尺寸可调的矩形区域中，这种矩形区域在WIN98中叫作“窗口”。例如图4就是WIN98系统工具“我的电脑”打开后的窗口。在这个窗口中从上至下分别有“标题栏”、“菜单栏”、“工具栏”、“地址栏”、“窗口内容”以及最下边的“状态栏”。它们的功能如下。
　　1.标题栏
　　用于表明是属于哪一项任务或程序的“窗口”，另外在标题栏的最右边还有三个窗口尺寸控制按钮，从左至右分别是“最小化”、“最大化”或“恢复”、“关闭”。如果单击“最小化”按钮，窗口将缩小成“图标”而只显示在任务栏上，对应的程序或任务自然也就转入后台运行，但并没被关闭。另三个按钮的作用与它们的名字相同。
　　2.菜单栏
　　菜单栏列出了一组功能项目名称，每个项目都能用鼠标拉出对应的一组操作命令，我们可以根据操作需要进行相应的选择。
　　3.工具栏
　　工具栏提供了对常用命令的快捷访问途径。在大多数程序中,如果将鼠标指针放在某个工具按钮上,关于此按钮的说明就会出现在按钮的右下侧。
　　4.地址或任务范围栏
　　在地址栏中一般显示当前文件、文件夹所在盘符和位置。
　　5.窗口内容
　　窗口中显示具体任务和程序运行情况，如使用的资源管理器，那么窗口中将列出一些磁盘或光盘驱动器的盘符、文件夹以及文件夹中的文件名等；如果是运行的是游戏“纸牌”，那么窗口中显示的则是相应的纸牌游戏过程。
　　6.状态栏
　　一般显示当前任务的执行情况，例如打开的文件数量、文件大小。当鼠标指向菜单命令时，状态栏还将显示命令的功能说明。
　　五、“我的电脑”
　　“我的电脑”是WIN98对电脑系统中软、硬件资源进行管理的主要手段。我们可以通过它去查看各磁盘、光盘上的信息或者使用各类系统工具。
　　用鼠标双击桌面上“我的电脑”图标打开窗口后。里面还有一些图标，它们的含义分别介绍如下。
　　1.驱动器图标
　　在“我的电脑”窗口中有分别标有“3.5英寸软盘”和“（C：）”、“（D：）”和“（E：）”的图标，代表系统中的软驱、硬盘和光驱（仔细看看，光驱的图标与硬盘图标不同喔）。可以通过双击这些图标来访问对应的磁盘或光盘。
　　2.打印机
　　用于安装和设置打印机，查看打印机的工作状态。
　　3.控制面板
　　控制面板是对WIN98中进行系统配置和软、硬件安装的地方。
　　4.拨号网络
　　用于查看通过MODEM连接的其它电脑上的共享资源情况以及建立新的连接。
　　5.计划任务
　　制定计划，让WIN98自动执行重复的任务（实际中很少用到）。
　　好了，以上介绍了WIN98的一些基础知识。也许你已经迫不及待想做点什么了，在下期我们将介绍WIN98的基本操作。
