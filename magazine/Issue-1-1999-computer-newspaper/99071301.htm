9种典型职业的软件选配套――之项目管理员篇


　　我们是项目管理员
　　随着社会经济的发展，制造与建设工程越来越大型化，系统化。例如三峡工程这样的跨世纪工程，包含了勘探、论证、决策、设计、施工、设备制造、安装、监理、工程 验收等等复杂的环节，还要考虑移民动迁、航道管理、文物保护、安全保卫、水文气象等重要工作。这些庞大和复杂的工作需要成千上万家公司、单位参与，而三峡工程的项目管理人员就是对整个工程进行统一的规划管理。我们项目管理人员负责工程项目的分化、定标、进度规划、协调参与单位，以及安排工程监理和验收等一系列管理工作，可以说，项目管理是三峡工程项目实施中的灵魂。
　　三峡工程的项目管理由三峡工程的业主单位中国长江三峡工程开发总公司（简称三峡总公司）负责实施，我们所在的总公司信息中心就是三峡工程项目管理的中心，负责建成联系宜昌总部和三峡工地以及各大施工单位的计算机广域网，同时,本中心与加拿大AGRA Monenco公司合作开发工程信息管理系统TGPMS。
　　我们的系统软件
　　由于三峡工程项目管理的复杂性，我们项目管理的电脑系统就是一个复杂的多层次计算机系统，软件的使用也就相当复杂，系统软件的使用情况如下：
　　1. 主机:
　　IBM AIX 4.3.1、 HACMP 4.2.2
　　2. PC服务器:
　　Windows NT4.0、NetWare 3.12
　　3. 数据库:
　　Oracle 7.3.2.3.1 for NT、Oracle 7.3.4.0.0 for AIX
　　4. 群件:
　　Lotus Notes Domino
　　5.用户:
　　因为公司内购买的基本都是品牌机,所以用户通用平台为随机的Microsoft Windows 95简体中文版。
一般应用软件
　　作为项目管理人员所使用的个人办公软件，目前公司已逐渐统一到使用Microsoft Office 97办公软件，用以打印编制往来公文、电子表格、演示图片等等。随着公司企业网建设的逐步完善,网络与大家的工作越来越密不可分，大家使用莲花公司的Notes来收发公司内部及来自因特网的电子邮件，公司内部的公共信息、工程信息也会发布在Notes上,而且大家还可以就感兴趣的话题在网上讨论区中发表言论,进行交流等。
工程项目管理应用软件
　　目前在Notes办公系统上发布工程形象和图纸主要使用AutoCAD LT 97。为在三峡工程中推广使用工程管理软件P3,公司先后购买了多套网络版及单机版P3,提供给监理及施工单位,并举行了多次P3软件培训班。公司内还　成立了相应的工程信息部来收集和发布工程信息, 编制进度计划。其它的应用软件还有：
　　1. 东大阿尔派的图纸管理系统。
　　2. 计算机专业人员目前从事的工作主要是开发公司的工程项目管理软件TGPMS(Three Gorges Project Management System)。开发工作即将结束,相信在不久的将来TGPMS将成为项目管理人员工作中密不可分的工具。
　　TGPMS系统开发专用软件
　　TGPMS是我们与加拿大AGRA Monenco公司合作开发的工程信息管理系统，采用Client/Server结构，后台的数据库系统为Oracle7.3.2.3.1，前台为简体中文Microsoft Windows 95。开发人员所使用的开发工具为Oracle Developer 2000(release 1.2 from Windows)以及加拿大AGRA Monenco公司所提供的Client/Server快速开发工具FormsBooster。该开发工具也是基于Form Designer 4.5开发的，其它使用到的软件有：
　　1. 由加拿大Neuma 公司开发的系统配置管理软件Neuma＋4.0来配置管理所有源程序,SQL脚本程序及所有的文档。
　　2. 统一使用Microsoft Office 95。
　　3. S－Designer 6.0, 用于TGPMS系统设计。
　　4. VISIO 4.0版,用于制作图表。
　　5. Doc－To－Help2.5版,用于将Word文件变成Microsoft的Help文件。
　　6. Dr.Solomon 7.75版,用于网络病毒防治。
　　7. RichWin4.2版。(湖北　刘益勇 陈　榕)

　　10：00～12：00

　　14：00～18：00

　　20：00～23：00
　　收发个人的电子信件，再抓紧时间看一看有关TGPMS的外文资料。
