Windows 95自测题


　　一、填空
　　1. Windows 95中既可以使用________个字符的长文件名，也可以支持以前________格式的文件名。
　　2. 多任务机制是指操作系统可以控制________时间分配，让计算机同时执行多个应用程序。
　　3. 在Windows 95的写字板文字处理器中，Ctrl＋Z实现的功能是________。
　　4. 如果要实现一次就把所有打开的窗口都最小化，应执行的操作为：在任务栏的空白区按鼠标右键，选择弹出菜单中的________即可实现。
　　5. 要想浏览Windows 95的基本系统文件，可以在“开始”菜单的“运行”命令中键入________即可启动系统编辑程序。
　　二、选择题
　　1. Windows 95的基本通信功能有________。
　　a. 两台计算机间电缆线的连接
　　b. 连接拨号网络
　　c. 不支持笔记本电脑文件的移动
　　d. 支持传真收发
　　2. 在Windows 95中，可以按________弹出“开始”菜单。
　　a. Ctrl＋Tab
　　b. Alt＋Tab
　　c. Alt＋Esc
　　d. Ctrl＋Esc
　　3. Windows 95的特点主要有_________。
　　a. 简单易用图形用户界面
　　b. 完整可靠的通信及网络环境
　　c. 面向对象系统的
　　d. 即插即用的硬件安装能力
　　e. 自动播放和执行多媒体及CD功能
　　f. 采用抢先式多任务机制
　　4. 滚动条的基本操作及相关功能有_________。
　　a. 按垂直滚动条的上方，滚动至上一页
　　b. 点击滚动箭头，滚动一行
　　c. 拖动滑块至最上方，到文件的开头
　　d. 拖动滑块可以到想要的位置
　　5. 要切换到上次使用的窗口应使用的快捷键为_________。
　　a. Ctrl＋Tab
　　b. Alt＋Tab
　　c. Alt＋Esc
　　d. Ctrl＋Esc
[答案]
　　一、 填空
　　1. 255，8.3 　　　　2. CPU
　　3. 取消上一次操作 　4. 最小化所有窗口
　　5. SYSEDIT
　　二、 选择题
　　1. a,b,d 　　　　2.d　
　　3. a,b,c,d,e,f 　4. a,b,c,d 5. b
