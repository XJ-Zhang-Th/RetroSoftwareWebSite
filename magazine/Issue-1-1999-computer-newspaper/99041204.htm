用MS Batch 98全自动轻松安装Win98


　　Win 98安装光盘里藏了不少好东西,近日我又发掘出一个MS Batch 98,这个东西有些什么作用,下面听我一一道来。
　　平时你是如何安装Win 98的呢?是不是觉得挺麻烦?如果偶尔装一下倒还罢了,若要经常安装或一下为几台、几十台机器安装Win 98你烦不烦?虽然在Win 95时代就已经可以利用自动安装脚本文件(.INF)来简化安装过程,但手工修改众多的安装选项仍然费时、费力、易出错。现在只要安装了MS Batch 98这些问题就迎刃而解了。在MS Batch 98漂亮的图形界面中,你只要用鼠标点几下就可以轻松地制作出包罗万象的自动安装脚本文件。有了这个文件,在安装Win 98时,安装程序将自动查询该文件而不向用户询问每条信息,真正实现安装时无人值守。下面看看具体如何使用。
　　1.安装MS Batch 98
　　MS Batch 98位于Win 98安装光盘的\TOOLS\RESKIT\BATCH目录下,运行其中SETUP程序后即可将Batch 98安装到硬盘中,单击“开始”，“程序”,“Microsoft Batch 98”就可运行它了。
　　2.创建自动安装脚本文件
　　运行“MicrosoftBatch 98”后,出现如图所示画面,单击“系统”框中的“一般安装选项”、“网络选项”、“可选的组件”、“Internet Explorer选项”、“高级选项”按钮可进行相应的设置。其中所包含的内容非常丰富,从产品序列号、Win 98安装目录到IE 4.0浏览器的起始页,甚至Windows从何处升级都能设置。对于一般用户来说“一般安装选项”和“可选的组件”按钮最为常用。设置好有关选项后,单击“保存设置到INF”按钮,程序将提示保存的路径与文件名,输入后单击“确定”按钮。
　　另外,单击“现在收集”按钮可以将本机注册表中的相关信息导入Batch 98以供用户使用。
　　3.使用自动安装脚本文件
　　在生成了自动安装脚本文件后,当需要安装Win 98时,输入″SETUP C:\MSBatch.INF″(假设自动安装脚本文件在C:\下名为MSBatch.INF)即可。
　　4.修改自动安装脚本文件
　　运行Batch 98后,单击菜单中的“文件”,“打开”后,选择.INF文件所在的目录、文件名,单击“确定”按钮,然后单击“系统”框中的相应按钮即可修改所设选项。
　　MS Batch 98的功能非常强大,还可以一次性生成多个供不同机器使用的自动安装脚本文件,在此就不再详述,大家可参考中帮助信息。（湖北　张杰）
