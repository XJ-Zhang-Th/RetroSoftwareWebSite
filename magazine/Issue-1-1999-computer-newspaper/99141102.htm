增强IE5的工具插件

　　微软似乎有个很特别的习惯，每推出一个新软件，就要悄悄地再编出一些为此软件增强或添加功能的小程序、小插件，虽然也是微软出品，但是对这些小软件、小插件却不提供技术支持，像Win95/98的Power Toys、Tweak UI、Office软件的Power Toys等都属于这类软件。这不，随着IE5.0正式版的发布，为其增强和添加功能的小插件程序也跟着悄悄地出现在微软的主页上了。

Internet Explorer 5.0 Toolbar Wallpaper

　　下载地址：http://joysoft.soim.net(171K)
　　这个小程序会在IE5.0的“工具”菜单增加一个“Toolbar Wallpaper”(工具栏壁纸）选项，通过它可以任意更改IE5.0工具栏的背景。选取该项弹出对话框，在“Select An image”下就可以选择列出的10个壁纸。另外，在对话框的最下面还有一个复选项“Show Desktop Wallpaper In List”，如果选上它，就可以将Windows的壁纸也加进来。如果自制壁纸或是想用自己作的图像来作为工具栏的背景，那么就将你喜欢的或是自制的图像文件拷贝到C:\Program Files\Internet Explorer\Bitmaps目录下，不过请注意文件的保存格式一定要是BMP的。

Internet Explorer 5.0 Power Tweaks Web Accessories

　　下载地址：http://joysoft.yeah.net(127K)
　　这个小工具所提供的功能会使你在操作IE5.0时变得更加轻松，它在工具栏上增加了一个“Offline/Online”按钮，使用者可以很容易地通过它在离线和在线状态间进行切换，这个功能按钮对那些使用笔记本电脑的朋友尤为有用。而它在IE5.0“工具”菜单中增加的“Add To Trusted Zone”和“Add To Restricted Zone”两个选项可以使你很容易地将当前的站点设置为可信站点或限制站点，不需要像以前那样到Internet选项中去设置了。另外，该程序还为鼠标右键菜单增加了“Copy An URL Location”功能，用户使用它可以方便地将当前页面的地址复制到E－Mail或文件中。

Web Accessories for Internet Explorer 5.0

　　下载地址：http://joysoft.yeah.net(137K)
　　这是一个IE5.0的工具包，它包含了八个提供给高级用户的功能，比如像IE4.0时的Power Toys中提供的图像放大、缩小、快速在多个搜索引擎中进行查找、图像显示开关等，其他还有选择“不显示网页图像”来加快下载速度、文本高亮显示、在新窗口中打开页面中的框架、列出页面中所有链接、列出页面中所有图片、使用默认搜索引擎查询网页中的文字等。比如文本高亮显示功能对于经常用网页作演示的朋友非常有用，利用它可以将所演示的重点文字标出来以提醒观众注意，它的使用方法也很简单，用鼠标左键选取页面上的一段文字然后点右键选择“Highlight”，这样所选文字就会显示鲜艳的黄色了。另外，使用默认搜索引擎查询网页中文字的功能也很不错，比如你在一个网页中想查询其中某个词，那么只要选取它并点击鼠标右键选择“Web Search”，IE5.0就会弹出一个新窗口并连接到Yahoo为IE5提供的特定搜索引擎上进行搜索，令人遗憾的是它不支持中文搜索。
