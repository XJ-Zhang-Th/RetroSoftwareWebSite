电脑中的密码设置

　　随着计算机应用的普及，许多用户都开始利用计算机处理一些涉及隐私甚至比较机密的信息，这就不可避免的会遇到如何保密的问题。相对而言，设置密码是运用最多，也是最方便有效的安全控制措施！相应的，如何在系统及应用程序中设置密码也就成了广大用户所关心的话题。

　　一、开机密码（又称CMOS密码）

　　一般来说，绝大多数计算机都允许同时设置两个不同的开机密码，其中一个为超级用户密码，它既可启动计算机，又可对BIOS进行设置，另外一个为普通密码，它只能启动计算机，而不能对BIOS进行设置。
　　设置：系统启动时按“DEL”键进入BIOS设置界面，然后分别利用主菜单中的“SUPERVISOR PASSWORD”和“USER PASSWORD”选项设置超级用户密码和普通密码。接着，我们应进入主菜单中的“BIOS FEATURES SETUP”（BIOS功能设置）窗口，然后在“Security Options”(安全选项)中选择“SYSTEM”选项（注意，“Security Options”主要有两个选项SYSTEM和SETUP，当我们在选择SETUP时，系统仅会在进入BIOS设置时要求用户输入密码，但正常的启动不受影响；而当我们选择SYSTEM时，系统一开机就会要求密码，无密码则无法启动计算机）。密码设置完毕之后我们只需按下F10键保存所作更改即可。
　　注:不同机器开机密码的设置可能与以上步骤不尽相同,应根据系统提示进行操作。

　　二、Windows密码

　　Windows密码并不能阻止他人使用你的计算机。只需要点击口令对话框中的“取消”按钮，他人就可以进入Windows桌面。但Windows密码可以防止他人修改你的Windows设置，例如可以防止别人删除你桌面上的快捷图标。
　　设置：在Windows 98的控制面板中双击“密码”图标。在“密码属性”窗口中单击“更改Windows密码”／“确定”按钮，出现“更改Windows密码”窗口。然后输入密码（如果你以前没有设置过密码，就不要填“旧密码”框）。然后选中“用户配置文件”选项卡中的“用户可自定义首选项及桌面设置，登录时，Windows将自动启用个人设置”选项。最后单击“确定”按钮并重新启动计算机即可。

　　三、屏幕保护密码

　　当你只是离开计算机一小会儿，你不想让别人偷窥到你的桌面。你可以设置屏幕保护密码。另外，屏幕保护程序还有保护显示器的作用。
　　设置：右击桌面空白处并执行“属性”命令，打开“显示属性”设置框，如图2，然后单击“屏幕保护程序”选项卡并从“屏幕保护程序”列表框中选择喜欢的屏幕保护程序，激活系统的屏幕保护功能。接下来我们应复选“密码保护”选项，然后单击“更改”按钮，打开“更改密码”对话框并重复两次输入密码即可达到设置密码保护的目的。

　　四、WPS密码

　　用来防止他人非法打开你的WPS文档。
　　1. WPS for DOS
　　设置：按“ESC”键激活系统菜单，然后执行“文件”菜单中的“设置密码”命令，打开设置密码对话框，最后输入所需的密码即可（注意，输入密码之后必须执行一次存盘操作，否则密码无效。）。
　　注： WPS for DOS有一个万能密码“Ctrl＋QIUBOJUN”（求伯君的拼音），用此密码即可打开加密文档。
　　2. WPS 2000
　　设置：执行WPS 2000“文件”菜单的“密码设置”命令，打开“密码设置”对话框，然后在“设置密码”栏中输入密码，并在其下的“加密类型”栏中选择适当的加密类型（有“普通型加密”和“绝密型加密”两种选择）即可。

　　五、Word密码

　　设置：在保存Word文档时，在“另存为”对话框中单击“选项”按钮，然后在弹出的“保存”对话框（如图3）的“打开权限密码”或“修改权限密码”栏中设置密码（设置“打开权限密码”后，不知道密码的用户将无法打开文档；设置“修改权限密码”后，其他用户仍然可以打开文档进行浏览，但是不能对文档进行修改），最后单击“确定”按钮即可。 　　为了保证自己数据的安全，防止他人的非法破解，我们在设置密码时应注意以下几点：
　　1. 每三个月左右更改一次系统开机密码和Windows 98的启动密码（防止他人在无意中得到了你的密码）。
　　2. 不要将多个不同项目的密码设置得完全一样，否则他人只要知道了一个密码，你就没有秘密可言了。
　　3. 密码千万不要少于六个字符。目前各种密码破解软件大多是共享软件，在没有注册之前最多只能破解1～5位密码，因此我们设置密码一般不要少于六个字符。另一方面，加长密码也可增加计算量，增加破解密码的难度。
　　4. 不要将自己的个人资料作为密码（如姓名、英文名字、生日、电话、身份证号码等)，也不要采用某个现成的英文单词作为密码（某些密码破解软件有首先穷尽所有英文单词的功能）。
　　5. 密码最好采用英文字符、数字及符号的混合，如A2s?9Qb1＋等，以扩大采用穷尽法时的计算量(穷尽法是破解密码的一种方法)。
　　6. 除少数软件在设置密码时不区分大小写外，更多的软件则是将大小写字符视为不同的密码，广大用户一定要搞清楚自己的软件是否对大小写敏感。
　　最后要提醒读者，那就是当你设置密码之后，千万不要忘记了密码，否则……
（湖北 胡锦承）
