看上去很美――Linux发行版巡礼

　　Linux 可以运行在从PC到大型机各种层次的计算机系统上。它发行版种类很多，大约有200多种。而为不同的处理器开发的新发行版正在不断涌现。不同的用户可以选择自己需要的发行版。本文介绍了流行的Linux发行版本。
　　1.Debian Linux
　　Debian Linux(http://www.debian.org/)是完全由网络上的Linux爱好者负责维护开发的，其目的是制作一个可以同商业操作系统相媲美的免费操作系统。目前，有300多人正在为它所集成的软件包工作着。
　　Debian现在的版本为2.0，使用libc6系统库。用户指南捆绑在Debian CD中。这个指南的HTML文件还可以从Web免费得到(http://www.linuxpress.com/)。
　　尽管目前使用Debian Linux 的人不多，但Debian Linux 的确是一个非常优秀的Linux发行版本，在众多发行版中被认为是最正宗的一个。它深深地打上了自由软件的烙印。是软件世界中最为灿烂的一颗星。
　　下载站点：ftp://ftp.debian.org/debian/在此站点上，除了提供所有的软件包外，还提供一个CD―ROM的映像文件，用户可以将它下载，制作成同原发行版完全相同的光盘。也可以从http://cd－image.debian.org上下载，文件名为：debian－i386.raw，以ISO9660格式下载后，在各操作系统下均可以将它刻录出来。
　　2.OpenLinux (Caldera)
　　Caldera OpenLinux Base(http://www.openlinux.org)是面向企业解决方案市场的商业发行版本， 它的特点是有一个集成的桌面环境和广泛的系统管理工具。
　　OpenLinux包含一份240页的启动指南和附录， 两张软件CD，一张安装软盘。
　　Base包中有许多商业产品，包括Metro－X 服务器，一个Novell NetWare客户端程序，CRiSPlite文本编辑器和BRU2000－PE备份系统。还提供了一个有“非商业许可证”的Star Office版本。这是一个高度专业的办公产品套件，可以读写许多Microsoft Office格式的文件，值得一试。
　　Caldera还出售许多附加商品软件包，其中包括Adabas－D SQL数据库服务器，Netscape FastTrack Web服务器，和Wabi等。
　　目前Caldera OpenLinux版本为： 1.2。
　　3.Red Hat Linux
　　RED Hat Linux (http://www.redhat.com)是由Red Hat Software公司发行的，出现较晚的Linux发行版。由于它简单的安装、配置有特点，使得它的市场占有率不断升高。
　　Red Hat Linux目前版本已为5.2稳定内核为2.0.35。
　　Red Hat的特点有：
　　(1)支持多平台
　　Red Hat Linux从4版本开始，同时支持Intel ， Alpha ， SPARC处理器。支持其它处理器的系统正在开发之中。
　　(2)优秀的软件包管理
　　Red Hat Linux的软件包管理系统－RPM（RedHat Package Manager）是一个高效的包管理系统，它提供了安装、 升级、 查询和校验软件的可靠方法，同时帮助确认系统的整体集成性能。Red Hat Linux被用户安装过一遍之后，以后的工作就是不断使用RPM升级系统而不必从头安装整个系统。
　　(3)系统安全性好
　　Red Hat Linux在缺省配置下就有较高的安全性，同时提供PAM，更增强了系统安全性。Red Hat Linux提供快速的系统安全补丁。Red Hat Linux的安全方面的邮件列表是权威的Linux安全问题的消息发布地之一。
　　目前，已有DELL、HP、COMPAQ等硬件厂商准备在他们的服务器上预装Red Hat Linux 系统。可以从ftp.redhat.com站点下载，也可以Sunsit.uni.edu下载，国内用户可以从freesoft.gov.cei.cn/pub处下载。
　　4.SlackWare
　　SlackWare Linux(http://www.slackware.com)是最早的Linux发行版之一，由Patrick Volkerding开发的。SlackWare在Linux发行版本的进化中有着悠久的历史。 它是最早提供许多软件包的发行版本之一， 并在一段时间里是业界领先者。由于发行时间上的原因，SlackWare Linux是目前市场占有率最高的发行版。
　　SlackWare Linux是最早的Linux发行版，同时，也是唯一提供由软盘安装整个操作系统的一个发行版。其软件包按软件的功能分别放在不同的目录中。如：a、ap、d、e、f目录等。它的优势是文档比较完整，用户数量比较多，可以在网络上找到较多的用户，方便相互之间进行讨论学习。
　　目前SlackWare Linux的版本为3.5.0，内核基于2.0.31。用户可以从ftp.cdrom.com处下载，也可以从Sunsite.uni.edu处下载。
　　5.S.u.S.E Linux
　　SuSE Linux(http://www.SuSE.com)也是一个流行的发行版本。S.u.S.E的总部在德国。有很多很好的软件包， 并且对德语有很强的支持。
　　一旦装好，SuSE非常好用，它有许多在线文档和好的软件包。YaST是一个多用工具，它还可以用来做许多系统维护的工作，包括用户账户管理，包管理和一般的配置。
　　SuSE Linux目前的最新版本为6.0，共有5张CD，同时还提供多达450页的手册，新版的SuSE Linux中收集了约900多个基于Linux的软件，其内核为SuSE 认为很稳定的2.0.36，同时也可以使用2.2版本的内核。
　　6.Xteam Linux中文版
　　Xteam Linux中文版(http://www.xteamlinux.com.cn)是北京冲浪平台软件公司推出的全球第一套Linux中文发行版本。它经过“Xteam“小组近一年的努力而开发完成。作为面向广大中文用户的Linux操作系统，Xteam Linux中文版具有明显的技术特色。
　　在系统安装方面，Xteam Linux提供了全球第一套智能化的图形安装方式。它构建了一套独特的控制台图形界面库，在此基础上形成真正的图形安装环境；此外，还提供智能安装助手，可以根据用户资源的使用状况，自动提供经过优化的缺省配置方案。Xteam Linux对这两种汉化方法进行了权衡，最终采用了以内核汉化为主、外挂平台为辅的方式。
　　Xteam Linux成功地引入最新的Linux内核Kernel 2.2，并对系统核心作了更加适合国内用户使用的优化设置。经过对各种窗口管理器上千小时的使用、测试，开发小组认为KDE的表现最为稳定，也比较适合国内用户的使用习惯，因此Xteam Linux采用了最新了KDE1.1作为标准的中文图形用户界面。
