如何更改Windows的默认打开方式


　　一般来说，直接在文件上用右键单击后选择“打开方式”即可改变文件的打开方式。但是我们在使用的时候常常会有一些情况让我们很烦恼。比如，要打开一个文件来看看，但是，又不愿等待大型的编辑或查看软件；又比如，你以前安装了一套软件，现在已经卸载了，在资源管理器里双击某些有关的文件时，Windows又提示不能找到以前的启动文件。这样我们怎么办呢？没问题，用资源管理器就可以“搞定”！举个例子来让我们可以快速地查看并编辑图标文件(＊.ICO)。
　　假设我们有一套图标编辑软件叫ICONEDIT.EXE，存放在C盘的Windows子目录里，则操作如下：
　　1. 打开“资源管理器”，选择菜单“查看”中的“文件夹选项”；
　　2. 选择“文件类型”；
　　3. 在“已注册的文件类型”的列表里找到名字为“图标”的项目，然后单击它；
　　4. 单击“编辑”以打开“编辑文件类型”的窗口；
　　5. 单击“新建”，在“操作”里输入“Open”，在“用于执行操作的应用程序”里输入“C:\Windows”，然后单击“确定”；
　　6. 这时可以看见“操作”的列表里多了一项名为“Open”的项目，单击它，然后单击“设为默认值”；
　　7. 按“确定”回到“文件夹选项”，按“确定”即可。
　　以后你再双击后缀是ICO的文件就会直接用ICONEDIT.EXE来打开它了！
　　这种方式还有一个好处，就是可以改变文件的图标。还是接着上面这个例子：
　　1. 打开“资源管理器”，选择菜单“查看”中的“文件夹选项”；
　　2. 选择“文件类型”；
　　3. 在“已注册的文件类型”的列表里找到名字为“图标”的项目，然后单击它；
　　4. 单击“编辑”以打开“编辑文件类型”的窗口；
　　5. 单击“更改图标”后系统会自动检测ICONEDIT.EXE里带的图标，如果没有系统则打开SHELL32.DLL里的默认图标库;
　　6. 单击喜欢的图标后按“确定”至退出;
　　好了，现在你就会发现所有后缀是ICO的文件就会显示选择的新图标了！该例子在Win'95下实现。(广州　陈琼)
