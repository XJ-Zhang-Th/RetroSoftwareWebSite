我们的新举措

　　电脑报函授学校的学习方式为自学与各类函授辅导相结合。除了专版配合函授学校的教学进度、刊登各类辅导文章外，还在专版上设“答疑信箱”、对学员来信中共通性的问题作统一回答。学习期满，通过邮寄试卷进行考试，成绩合格者，寄发相应的结业证书。
　　电脑报函授学校本着低价收费、高质服务的原则为广大计算机爱好者服务。我们热忱欢迎广大学员把参加函授班学习的心得体会寄给我们，我们将选刊优秀的学员来稿。如果学员或读者还有更好的建议和要求，欢迎给我们提出来，我们将尽可能地吸收合理的意见，以开办更适合广大学员和读者需要的培训班。
　　电脑报函授学校在办学的过程中，先后吸纳了广大读者提出的一些合理建议，采取了以下措施：
　　1. 开设了函授热线 (023－63892147)；
　　2. 通过E－mail向学员答疑(school@cpcw.com)；
　　3. 对于经济比较困难的学员，我们给予力所能及的帮助和优惠；
　　4. 对各个函授班，结业考试成绩前十名将给予重奖，并授予“电脑报函授学校优秀学员”荣誉称号，具体奖励办法参见各函授班的招生简章；
　　5. 对各函授班，按报名的先后顺序，前100名学员将获赠电脑报社最新一期的《电脑报配套光盘》；
　　6. 不定期以各种方式加强和学员的紧密联系。
　　明年，根据计算机技术发展的需要，结合学员和读者的建议，函授学校将在以下几个方面作出新的举措：
　　1. 继续开办初级班、中级班、等级班和水平考试班，并对教材和讲授内容作大幅度的更新和调整。
　　2. 开办Internet 培训班，建立网上学校，进行网上教学，普及网络知识；
　　3. 与著名软件公司联合举办CAD培训班，充分发挥计算机的辅助应用功能；
　　4. 聘请资深专业人士作顾问，开设广告创意短训班、网页制作班等；
　　5. 特别加强编程类的培训班，如VB、VFP、VC等；
　　6. 应广大硬件爱好者的要求，在适当的时候开设硬件维修短训班。
　　在这普天同庆的日子里，我们共同播下希望的种子，共同去努力、去创造，预祝我们的每一位读者都学有所成，脱颖而出，期待我们共同的事业不断发展和壮大！
　　最后，欢迎广大学员和读者来信就如何办好我们共同的函授学校发表您的高见!
　　函授学校地址：重庆市人民路236号电脑报函授学校（400015）。
