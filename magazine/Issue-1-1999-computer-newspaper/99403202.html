　　9. 从其他电子邮件程序导入邮件
　　使用 Outlook Express 导入向导，可以轻松地从其它电子邮件程序（例如从Eudora、Netscape Communicator等，但不能从foxmail）导入邮件。在“文件”菜单上，指向“导入”，然后单击“邮件”。 选择准备导入邮件的电子邮件程序，然后单击“下一步”。 按照屏幕上的提示操作。
　　10.从其他电子邮件程序导入通讯簿
　　您可从其它电子邮件程序的通讯簿或以逗号分割的文本文件(后缀为CSV)中导入通讯簿联系人。单击“文件”菜单，指向“导入”，然后单击 “通讯簿”。 选择要导入的通讯簿或文件类型，然后单击“导入”。如果未列出你所用的电子邮件程序的通讯簿，可以在你所用的电子邮件程序中将通讯薄导出到CSV文件或 LDIF（LDAP 目录交换格式）文件，然后再导入OutlookExpress通讯簿。
　　11.查找感兴趣的新闻组
　　要查找感兴趣的新闻组，可以通过新闻服务器搜索新闻组名称中的特定单词。你可以在文件夹列表中，单击服务器名，然后单击工具栏上的“新闻组”按钮。 在“显示包含以下文字的新闻组”框中键入要搜索的内容。
　　12.预订新闻组
　　预订的好处在于预订后的新闻组将包含在文件夹列表栏中，以方便用户访问。添加新闻服务器时，Outlook Express 会提示你预订该服务器上的新闻组。 查看未预订的新闻组时，可单击“工具”菜单，然后单击“新闻组”。 再单击文件夹列表中的服务器名，点击工具栏上的“新闻组”按钮，选择要预订的新闻组，单击“预订”。
　　13.在邮件和新闻阅读之间切换
　　在文件夹列表中，单击“Outlook Express”，然后单击窗口中的目标链接。或者在文件夹列表中，单击“收件箱”或新闻服务器的图标。
　　在浏览器中可以将 Outlook Express 设置为默认电子邮件程序。这样，单击 Web 页上的电子邮件链接，或者选择浏览器或新闻阅读程序中的邮件命令时，都将打开 Outlook Express。
　　与此类似，也可以将 Outlook Express 设置为默认的新闻阅读程序。在“工具”菜单上，单击“选项”。 在“常规”选项卡上，选中以下复选框：“将 Outlook Express 设置为默认的电子邮件程序”或“将 Outlook Express 设置为默认的新闻阅读程序”。
　　设置Microsoft Outlook Express的一些小技巧
　　1.更改用于通知用户有新邮件到达的声音。可以双击“控制面板”中的“声音”图标，选择“事件”列表中的“新邮件通知”。
　　2.要以纯文本方式发送电子邮件（而不是HTML格式），请单击“格式”菜单上的“纯文本”命令。如果想以纯文本作为新邮件、回复邮件以及转发邮件的默认格式，请单击“工具”菜单上的“选项”命令，单击“发送”选项卡，然后选择“邮件发送格式”下的“纯文本”命令。默认情况下，新闻邮件按纯文本格式发送，同样，其设置也可在“发送”选项卡的“新闻发送格式”下更改。
　　3.可以指定只想查看未阅读的新闻邮件。为此，请单击“工具”菜单上的“新闻组”，然后依次单击“查看”菜单、“当前查看方式”、“未阅读的邮件”。
　　4.要查看向你发送电子邮件的用户的电子邮件地址，请通过双击打开这封邮件，然后双击“发件人”域中的作者名称。另外，还可以按以下方法将此人添加到通讯簿中：用鼠标右键单击作者名称，然后单击“添加到通讯簿”。
　　5.在 Outlook Express中，可以在服务器上保留邮件的副本。请单击“工具”菜单上的“帐号”，选择要修改的邮件帐号，依次单击“属性”命令、“高级”选项卡，然后选中“在服务器上保留邮件的副本”复选框。
　　6.可以将签名添加到所有邮件中，包括回复或转发的邮件。为此，请单击“工具”菜单上的“信纸”，在“邮件”或“新闻”选项卡中，选中“将签名附加到所有待发邮件的末尾"复选框。
　　7.可以针对一封新闻邮件查看所有标题。请双击这封邮件并在单独的窗口中打开邮件，单击“查看”菜单并单击“全部标题”。
　　8.如果想高效地查看新闻组，你可以查找关于特定主题的新闻组或选择文件夹列表中的新闻服务器，然后单击工具栏上的“新闻组”按钮。也可以在“显示包含以下文字的新闻组”框中，键入相关的主题或文字。
　　总之，Microsoft Outlook Express方便地实现了全球范围的联机通信，无论是与朋友交换电子邮件还是加入新闻组进行思想与信息的交流，Microsoft　Outlook Express都是你最得力的助手。说了这么多，快去试试吧！
