群芳争艳――喷墨打印机选购指南

　　打印机市场可谓硝烟四起，群雄逐鹿。面对着众多的喷墨打印机产品，对一般家庭用户和中小型办公商业用户而言，如何选购一款性价比最高的喷打产品呢？在1998年国内喷墨打印机市场上，由于HP、Canon、EPSON三大巨头拥有绝对的市场优势（一份权威调查报告显示，这三家公司的喷墨打印机所占市场份额达到了98％），因此，我们就用这三家公司的喷打产品来向读者推介。

选购须知

　　选购一款称心如意的喷墨打印机之前，主要应注意以下指标和基本技术：
　　1．打印精度指标DPI（Dot Per Inch）：
　　DPI是业界衡量打印质量的一个重要标准，也是一个最为基本的判断打印机分辨率的指标。所谓DPI是指打印机在每英寸可打印的点数。提请读者注意的是，300DPI是人眼能否辨别输出文本图像锯齿边缘的临界点。如果再考虑到在喷墨打印机工作过程中，可能会因为调墨、抖动（Dither）或者淡化（Diffusion）等问题而在纸张上形成所谓“卫星”现象――在打印出的线条周围出现微小锯齿，从而在一定程度上影响输出效果，那么360DPI以上的喷墨打印机才是比较令人满意的选择。
　　2．打印速度指标PPM（Page Per Minute）：
　　PPM是衡量包括喷墨打印机在内的非击打式打印机打印速度的一个重要标准。PPM是指打印机每分钟可打印多少页。例如，5PPM即表示每分钟可打印5页英文字符，中文字符环境下该值则要减半。这里有两点提请读者注意：其一，打印机说明书上所标明的PPM值是指该机所能达到的最高打印速度，实际使用时，这个值会受到多种因素，诸如打印文件的大小、主机的处理速度、耗材、打印头的质量和寿命等的影响而发生变化。其二，说明书上所标明的PPM值是指该机在连续打印时的平均速度。如果只打印一页，即使标明了5PPM，那么也可能需要耗费一分钟。
　　3．打印幅面：
　　一般喷墨打印机的打印幅面有A4和A3两种。普通家庭用户和中小型办公用户普遍使用的是A4幅面的喷墨打印机。A3幅面的喷墨打印机则较适合特殊或者专业用途，如条幅打印、海报打印、数码影像处理等。再大幅面的打印输出需求，如CAD用户、广告制作、艺术设计、印刷出版业等，就需要借助大幅面喷墨打印机了。
　　4．打印驱动程序：
　　打印驱动程序是一个很容易被用户忽视但又与打印输出密切相关的重要保证。买来一台喷墨打印机却打不出中文的现象，就是购买时未注意打印驱动程序内置文字类型的缘故。更为重要的是，众多先进的打印技术都是依靠打印驱动程序来实现的。
市场上的主要喷打产品
　　Canon(佳能)公司
　　SP系列喷墨打印机是Canon公司最为风靡世界的喷打产品。Canon BJC－255SP是该公司在1998年年中推出的面向家庭的一款喷墨打印机。它集中了超经济模式、全新墨盒设计、四重色调控制技术和图像优化技术等多项先进技术，是Canon公司在我国市场首次推出的千元级照片质量彩喷产品，现市场售价约在850元左右。
　　Canon BJC－4310SP是办公型墨滴调整喷墨打印机，可选用360DPI的彩图扫描器，也具备超经济模式、图像优化技术等多项先进技术，输出色彩细致鲜明。Canon BJC－4310SP,在超经济模式下，使用BC－20黑色墨盒，可以打印3600页A4纸，打印分辨率最高达到720DPI，打印速度可高达510cps（10 cpi），可进行条幅打印。Canon BJC－4310SP重量约在3.5公斤， 现市场售价约在1300元。
　　Canon BJC－80是一款便携式喷墨打印机，重量仅有1.4公斤，集成了Canon公司推出的“世界上最小的扫描器”――IS－22，带有与IrDA1.0兼容的无线通讯红外接口，现市场售价约在2350元。
　　Canon BJC－7000也是在1998年年中推出的一款喷墨打印机，引人注目之处在于它采用了Canon公司独创的P－POP技术。另外，在四色墨水的基础上，Canon BJC－7000还采用了三种新型Photo墨水，实现了前所未有的九重色控，打印分辨率高达1200DPI，现市场售价约在2600元。
　　HP(惠普)公司
　　HP DeskJet 200为普通纸经济型喷墨打印机，定位于普通家用，市场售价约870元。该机现已升级到HP DeskJet 200Cci，采用了新型流线型设计，大容量黑色经济墨盒，市场售价约在910元左右。
　　HP DeskJet 670C是一款个人全功能彩色喷墨打印机，它采用黑色和彩色双喷嘴，打印时无需更换喷嘴即可自由选择黑白或者彩色打印。对于黑白打印，最佳方式下打印分辨率达到600×600DPI；对于彩色打印，普通纸打印分辨率为300×300DPI，光面纸或透明胶片打印分辨率可达600×300DPI。HP DeskJet 670C的打印速度，A4幅面黑色文本为4PPM，A4幅面彩色文本为1.5PPM。HP DeskJet 670C适合于家庭和小型办公使用，现市场售价约在1250元。
　　1998年春季，HP公司发布了全新的HP DeskJet 890C，作为HP DeskJet 系列喷墨打印机的旗舰产品。它第一次采用了HP富丽图色阶增强技术和二代智能色彩增强技术，面向中小型办公环境、图像处理人员和部分专业人士，打印速度，黑白方式为9PPM，彩色方式为5PP，是目前HP公司喷墨打印机中打印速度最快的，现市场售价约在3300元。
　　HP公司于去年底还推出了一款升级机型――HP DeskJet 695C，采用了全新的“经济高速”模式设计，定位于小型办公。HP DeskJet 695C装纸容量普通纸为100页， 打印机内存设有32K bytes的输入缓冲区，512K bytes的内置RAM。HP DeskJet 695C的打印速度，黑色文本是5PPM（经济高速模式），彩色文本是1.7PPM（经济高速模式）。HP DeskJet 695C的最高打印分辨率为600×600DPI（黑白）和600×300DPI（彩色），最高打印幅面是A4。HP DeskJet 695C机器重量5.3公斤，现市场售价约在1450元。
　　EPSON(爱普生)公司
　　EPSON Stylus Color 300在1998年5月正式面世，是与Canon BJC－255SP、 HP DeskJet 200处于同一档次的普通家用四色喷墨打印机，采用EPSON公司的完美成像系统，市场售价约在900元左右。
　　EPSON公司在1998年6月发布了两款具有照片级打印效果的彩喷新品――EPSON Stylus Photo 700/EX。EPSON Stylus Photo 700是全球第一款同时具有1440DPI的最高分辨率和六色打印功能的彩色喷墨打印机，集成了EPSON公司的多项先进技术，现市场售价约在2250元。
　　EPSON Stylus Color 400也是一款基于EPSON完美成像系统面向普通家庭用户的彩色喷墨打印机，据EPSON（中国）公司市场销售部门负责人称，EPSON Stylus Color 400是该公司喷打产品中销售最为成功的一款。
　　EPSON Stylus Color 440是EPSON Stylus Color400的升级产品。它一方面继承了 原来400的诸多优点，可以精确控制输出墨滴和色调误差，确保色彩层次的平滑过渡及人体肤色的自然柔美，720×720DPI的高分辨率等等。另一方面，EPSON Stylus Color 440更进一步，全面扩展打印功能，例如，打印驱动程序完全支持Windows 3.1、Windows 95以及最新的Windows 98操作系统，使得安装操作极为简单快捷。驱动程序装有EPSON最新的高级图像处理技术PhotoEnhance 3，处理图像细节更加细致入微，色彩层次更加分明，过渡更加细腻清晰。为了方便用户日常操作，EPSON Stylus Color 440设置了智能化全自动模式，能针对不同类型的数据和打印介质，自动调节色彩和打印质量。
　　EPSON Stylus Color 440设有64个黑色喷嘴、21×3个彩色喷嘴（青色、洋红色、黄色）、10K bytes的输入缓冲区。EPSON Stylus Color 440的最高打印分辨率为720×720DPI，最大打印幅面是A4，机器重量5.2公斤。EPSON Stylus Color 440使用两种墨盒：黑色专用墨盒（S020187）在5％的覆盖率下可打印370页；彩色专用墨盒（S020191）在每种颜色5％（青色、洋红色、黄色三种共15％）的覆盖率下可打印300页。现市场售价约为1350元。
　　EPSON Stylus Color 600定位于小型办公环境，打印精度达到了1440×720DPI，现市场售价约在1850元。
　　EPSON Stylus Color 640定位于小型商务和家庭办公，是EPSON Stylus Color 600的升级产品。EPSON Stylus Color 640实现了1440DPI的高分辨率照片质量打印。EPSON Stylus Color 640还利用EPSON公司最新的高级图像处理技术PhotoEnhance 3，可轻松调节图像的亮度、对比度、与饱和度。EPSON Stylus Color 640机器重量5.2公斤（不含墨盒），现市场售价约在1960元左右。
　　EPSON Stylus Color 800定位于办公和商用环境，打印精度为1440×720DPI，在多平台兼容性和联网上前进了一步，用户可以安装EPSON Stylus RIP软件，将其作为一台低成本的Postscript彩色打印机。EPSON Stylus Color 800现市场售价约在2900元左右，其升级机型EPSON Stylus Color 850也于日前推出。
　　Lexmark(利盟)公司
　　另外，在国内市场上表现较好的还有Lexmark，Lexmark的喷墨打印机虽然名气和市场占有率都不如其它三大公司，但其喷墨打印机产品质量也是有口皆碑。其主流产品有Lexmark 1100、Lexmark 3200、Lexmark 5700、Lexmark 5770等。Lexmark 3200号称最高分辨率达到1200×1200DPI，黑白打印速度为6PPM，彩色打印速度为2.5PPM，机器重约5.7公斤。
