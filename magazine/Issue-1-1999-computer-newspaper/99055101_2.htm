中高档声卡市场两强并立

　　北京市场：不用说，买创通的PCI声卡是比较保险的，它的最低档的PCI产品是audioPCI，也叫做PCI32，不过现在的包装上都贴着64位的字样，让人迷惑不已，原来它跟PCI64用的是同一芯片，只是驱动程序不同，不过PCI32只要280元，PCI64可要460元呢！还有一种PCI128的卡，更少见了，价格600元。其实PCI64已经不错了，还支持4音箱，算是中档声卡中的佼佼者。最昂贵的是创新的新产品SB Live!，价格现在是1780元，还好比去年的1998元便宜了一点，但也够离谱的！它的简化版本SB Live Value 价格800元，这个省掉子卡的品种或许值得发烧人士考虑了。创通的产品价格不菲，而与其同档次的就要算帝盟的产品了，帝盟的声卡也是赫赫有名，目前卖得最火的S90，支持A3D音效，价格已经从几个月前的600元跌到了370元！成为部分人士推崇的经典中档声卡，而准4声道的S70反倒并不看好。帝盟的拳头产品MX300现在已经从2个月前的1480元下跌到1200元了，还有MX200，现在也只卖900元而已。对了，再补充两个支持A3D的声卡，除了PCI64/S90这样的声卡以外，目前支持A3D的声卡还有启亨A3Dpro (价格450元)和A3D Wave(价格 350元)。与S90不同，这两款是4声道的声卡，类似PCI64，不过看目前的样子好像买的人不多似的，怪？
　　武汉市场：较具代表性的中档声卡有Yamaha 724、帝盟 S90、创新 PCI 64、启亨呛红辣椒A3D PRO版等。其中，Yamaha 724声卡价格最为低廉，有多家厂商生产，价格从160～360元不等，最著名的是中凌724。这种声卡配合软波表，效果也不错，不妨选择此款。帝盟质量优异，口碑不错。入门级的S70和最畅销的S90都很受玩家青睐。创新的PCI 32/64/128系列销售商家最多，相信你可轻松购得，它们可以软件模拟得到四声道输出和A3D效果。提醒大家，创新升级驱动程序非常频繁，你可要经常去下载哟。启亨的呛红辣椒A3D PRO版也支持A3D，买的时候可要看看包装盒，有没有“A3D Pro版”字样。呛红辣椒品种太多，一不小心就会搞混。
　　更高档次的声卡包括帝盟 M80、帝盟 MX200、帝盟 MX300、创新 SB Live!、SB Live Value。这些声卡价格在800～2000元之间。相对而言，这些声卡的销售量很小，毕竟价格不是普通人能够承受的，它们的用户群主要是专业人士和一些经济上宽裕的发烧友。
　　上海市场：中档层次最有代表性的是帝盟Sonic Impact S90声卡，由于采用了A3D标准，十分适合一些大游戏，很爽的！而且它的售价相对又较便宜，因此具有极大的竞争力，目前在上海卖得非常的红火，报价为395元，唯一的美中不足就是做工稍欠火候而且只提供了一对音箱接口。此外它的兄弟产品S70（采用ESS Maestro－2芯片）也是款不错的产品，售价350元，在上海卖得也不错。S70/S90这两款声卡的共同特点，用一句话可以概括：对游戏支持都比较好（能很好地支持DirectX），相对于高档声卡价廉物美，是普通游戏玩家的首选产品。
　　高档声卡由于价格都普遍太高，因此买的人比较少，我也就简略地说一下。目前上海主要的高档的声卡是创新的SB Live! 和MIAMOND MX300。SB Live!虽然价格高达1960元，但在上海还是有一定的市场。另外创新公司还推出了它的普及版，减少了数字接口的SB Live Value，价格却跌到955元，销路也颇为不错。至于帝盟 的MX300，作为帝盟的最新力作，它偏重于游戏音质上的表现，（在MP3方面也十分的出色），值得向游戏、Mp3爱好者推荐，上市价为1250元，比SB Live! 便宜些，只是要享受真正AC－3效果，还需要破费购买它的扩展卡，而且淘淘认为它的驱动程序还有待改进。带有MIDI音色库专用记忆体子卡的MX200的报价为1260元。
　　成都市场：一般经济条件不错的游戏发烧友，以及专门搞电脑音乐创作的朋友，多选择千元以上的高档声卡，成都声卡市场的中高档产品基本上是创新与帝盟的两雄相争。中档声卡市场主要有创新 SB64和帝盟的Sonic Impact S90。
　　新年伊始创新 与帝盟的竞争就呈白热化之势，创新开展“迎新春，送大礼”的活动，购买Sound Blaster Live Value 一块，优惠150元购PCWorks 4.1音箱一套。帝盟除不断降低产品价格外，还赞助了有CCTV足球之夜主持人参加的“道洋电脑超市网络足球赛”，为获奖者提供S70声卡等奖品。
　　广州市场：中高档市场可以说是创新公司和帝盟公司的天下。中档市场有创新的SB64和帝盟的Sonic Impact S90，它们价格比较接近，但S90采用PCI结构，且支持A3D功能，在广州的中档声卡市场占有明显的优势。高档声卡有SB Live!（SB Live Value）和MX300。目前SB Live又比MX300更被用户看好。其实它们之间的竞争并不只是两款产品的竞争那么简单，SB Live和MX300分别支持EAX和A3D2.0，一款产品的胜负和一个标准的确立比起来，分量上就差得太多了。
