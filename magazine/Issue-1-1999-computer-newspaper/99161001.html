文件夹自动同步工具Second Copy 97

　　比较并同步硬盘中的两个文件夹，使其中的文件保持完全一致，是数据备份的一种重要操作。常见的各类文件管理工具均无此功能，如果你有这种需求，请试试文件夹自动同步工具Second Copy 97（http://www.centered.com）。最新的Second Copy 97 V531，试用期为30天，试用期过后仍然可以运行，不过，不能添加新的同步任务，只能运行以前设置的任务。
　　Second Copy 97的安装很简单，安装过程中只需设置好安装目录、缺省的同步目标目录即可。程序在开机时自动运行并缩小为系统状态条中的图标，单击该图标即可打开Second Copy 97窗口（图1）。

建立同步任务

　　单击Second Copy 97工具栏中的“New Profile”按钮（或“File”菜单下的“New Profile”命令），进入配置同步任务向导。
　　1．选择配置方式：向导首页有“Express Setup”（快速设置）和“Custom Setup”（自定义设置）两个选项，快速设置没有下面介绍的5、6两步。选择“Custom Setup”后单击“Next”。
　　2．选择同步的源文件夹：程序弹出选择需要同步的源文件夹对话框。单击“Browse”按钮，可选择源文件夹，也可从“Source folder”下拉列表中选择前面选过的文件夹。如果需要连同子文件夹进行同步，须将对话框中的“Include Sub folders”项选中。完成后单击“Next”按钮。
　　3．选择同步的目标文件夹：程序弹出对话框，用来选定同步时的目标文件夹。方法同上。
　　4．选择操作方式：程序弹出选择操作方式对话框（图2）。Second Copy 97的最大特点是具有自动和手动两种文件夹同步操作方式。若选中“Periodically”下的某一项，则可以设定每隔X分钟（X可以自行设定）、1小时、12小时、X天等的某一时刻自动同步文件夹。若选中“Manual Control”下的“I will start the copy”项，则以手动方式进行同步。
　　5．文件过滤：点击“Next”进入设置文件过滤窗口，若需要对文件夹中的所有文件进行同步，可选中“All files”项，若需要包含或排除某些类型的文件，可选中“Only Selected files”项，程序列出文件过滤选择框（图3）。
　　在该对话框中可以设定同步时必须包含某一类型的文件，方法是在“Include File Speclfications”框中输入文件的扩展名，如TXT 、DOC等，单击“Add”按钮加入包含列表。若须排除某一类型的文件可在“Exclude File Speclfications”下输入文件的扩展名。
　　6．选择同步方式：在接下来的选择同步方式窗口中(图4)，可以从六种同步方式中选择。这六种同步方式依次是：1将文件从源文件夹复制到目标文件夹；2将文件从源文件夹复制到目标文件夹，并删除目标文件夹中的同名文件（可建立一备份文件夹存放被删除的文件）；3将文件从源文件夹移动到目标文件夹；4将源文件夹中的文件压缩到目标文件夹的ZIP包中；5将源文件夹中的文件压缩到目标文件夹的ZIP包中，并删除目标文件夹中的同名文件（也可建立备份）；6源和目标文件夹同步，相互将最新的文件复制到对方（推荐）。单击“Next”。
　　7．为配置命名：如果你在上一步没有建立备份文件夹，则在单击“Next”按钮后会再次询问你，选择“No”为不建立，之后，在对话框内输入刚才建立的同步任务的名称，单击“Finish”按钮，则该同步任务的名称就会出现在Second Copy 97窗口中。

同步操作方法

　　1．自动同步和手动同步：如果你为某个同步任务选定了自动方式，则程序会在规定的时间开始后台同步工作，同步过程中系统状态条中的图标会有动画显示。如果选择了手动方式，则单击系统状态条中的程序图标打开窗口。选中窗口中建立的某个任务，单击工具栏中的“Copy Now”按钮，程序就开始文件夹的同步复制并以进度条显示同步进度。同步过程中单击进度栏右边的“Stop Copy”按钮可中止同步。
　　2．修改与删除：选中Second Copy 97窗口中的某个同步任务，单击工具栏中的“Properties”按钮可对它进行修改。单击工具栏中的“Delete”按钮则可将它删除。
　　3．查看同步的文件列表：选中程序窗口中的某个同步任务，单击工具栏中的“Show Log”按钮打开“Second Copy 97 Activty Log”对话框，可以查看最近一次同步的文件列表。单击其中的“Save As”按钮可保存该列表，单击“Print”按钮可打印该列表，单击“Clear”按钮可删除该列表。
　　4．快捷菜单：用鼠标右击系统托盘中的“Second Copy 97”图标弹出程序的快捷菜单。单击“Profiles”命令，可打开“Second Copy 97”窗口；单击“Log”命令可打开“Second Copy 97 Activty Log”对话框；单击“Suspend”命令可停止Second Copy 97的后台运行。单击“Exit”命令可退出Second Copy 97。
