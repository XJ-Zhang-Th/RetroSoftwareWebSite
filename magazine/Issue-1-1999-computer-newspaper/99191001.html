　　NOT Word 我们用WPS

　　提起WPS，这个国内文字处理领域的“老牌劲旅”经历了由DOS向Windows转换时的“阵痛”，差点被Word打倒，但现在已经重新崛起了。1997年金山推出的WPS 97已经实现了支持所有的中文Windows平台、文本编辑与所见即所得方式并存、支持多种修饰效果、自动存盘和停电保护、灵活易用的表格、完善的链接、内嵌的中文校对系统、兼容多种文件格式等优秀功能。此时的WPS 97已具备了与Word一决雌雄的实力。今年4月又推出了跨世纪的WPS 2000，从而在中文文字处理软件领域建立了自己的优势。例如WPS 2000所提供的操作向导功能可根据用户选定的不同对象而自动显现与之相关的命令集合，这就比我们在使用Word时一层一层的从下拉菜单中费力查找所需功能要方便得多。再如WPS 2000首创了图文符号库功能，我们可利用它任意在文档中插入各种图文符号（如数学、化学、文字等），这就比Word的公式编辑器要方便。同时，针对目前系统中存在多种不同汉字编码的具体情况，WPS 2000又特意提供了多内码支持功能，它支持GB2312、BIG5、GBK等三种主要汉字编码，我们除可直接编辑采用这些内码的WPS文件外，还可输出GB及BIG5内码的RTF、TXT、HTM等类型的文件，这可是Word所望尘莫及的。在多媒体方面，WPS 2000具有支持语音输入和多媒体文件播放功能，它允许用户直接使用语音命令完成WPS 2000的各种编辑工作，并可直接播放AVI、MIDI及WAV等多媒体格式的文件，可满足广大用户进行多媒体处理的需要。另外，WPS 2000还改变了WPS 97在Internet方面的不足，增强了Internet/Intranet的网络功能，允许用户直接在WPS内部发送电子邮件、访问WPS 2000及金山公司主页，实现WPS 2000的在线升级等。WPS 2000的网址为http://www.wps2000.com。

　　

　　NOT Excel 我们用Cell表

　　用过Excel的人都知道Excel有很强大的功能，目前还没有哪个国产表格软件在功能强大方面能超过它，但是如果不使用Excel，有国产软件能够替代它吗?下面就来看看国产表格软件Cell表。

　　

　　1.制表：制表是电子表格软件的基本功能，Excel有很强的制表功能，单元还具有多种风格可以选择，如字体、颜色、对齐方式、数值显示方式等。这些功能Cell表也都具有，而且Cell表对国人的表格提供了全面支持。我国的表格一般比较整洁，需要表格线，还有单元内可有多重斜线，可在每个斜线分割的区域里输入内容。
　　2.数据计算：电子表格软件第二个重点就是数据计算，Excel通过公式来实现，Cell表也支持相对单元公式，可以自动重算，输入公式时能够检查是否循环引用，如有语法错误可以指出在哪儿出错，便于用户修改。Excel有近300个函数，Cell表相对较少，提供了近百个系统函数，但Cell表是一个开放的电子表格系统，如果觉得Cell表函数不够用，你可以增加自定义函数。
　　3.对象功能：Excel支持插入多种对象，例如各种控制、图片、图表等等，Excel都是用对象功能来实现的。也可以把Excel表插入到其他程序如Word中去。Cell表在此功能上较弱，不支持插入对象功能，但Cell表也支持一些常见的对象，Cell表有一些特殊的单元格，如下拉框，滚卷钮，都是很好用的功能。在Cell表中也可以插入图片和图表。
　　4.图表：Excel提供了强大的图表功能，总共有17大类50多种图表类型。Cell表也提供了图表功能，但种类较少，只有柱型图、折线图、散点图、饼图等几种常见类型，Cell表能根据表格数据自动生成图表，表格数据改变之后，图表可以自动刷新。Cell表的特点就是傻瓜式制图。
　　5.输入输出：Excel可以从各种外部数据源中取数，包括Access、Foxpro、dBase、Paradox、Lotus 1－2－3以及大型数据库等。这些功能Cell表中也都能实现，除此之外，Cell表还能读入DOS下带有制表符的表格文件，如CCED，WPS表格文件，还可以引入Excel文件，包括Excel文件中的字体，公式等信息多可以引入到Cell表中来。Cell表可以保存为Excel文件、Text文件以及HTML文件，所保存出来的HTML文件与Cell表文件在外观上相差无几。Cell保存出来的文件结构紧凑，在相同的信息量下，Cell表的文件大小是Excel文件的1/3～1/8，如果你想把你的表格通过Internet发送出去，Cell表的优势就显露出来了。
　　从上面我们可以看到，Cell表在功能强大方面虽不如Excel，但作为一个表格处理软件还是绰绰有余的，实际上Cell表也有一些Excel所不具备的功能，例如读入CCED文件、全表查询、区域查询、舍位平衡等功能，另外Cell还提供了各类办公用表模板233种。
　　Cell表普及版是一款共享软件，与Excel相比，注册费用仅需29元。访问网址为http://xcell.yeah.net。

　　NOT XINGPLAY 我们用超级解霸

　　三年前，当有人问到“你采用什么软件播放VCD”时，相信90％用户的答案都是“XING”，而三年后的今天，答案就会变成一个响亮的名字“超级解霸”。

　　

　　到底是什么原因导致这么多用户“倒戈”到国产软件呢？关键就在于超级解霸提供了许多与众不同的独特功能，如它独创了Direct CD－ROM“防读死”技术，超强纠错，能对付任何“烂”碟，即使Windows不能识别的无文件“怪”碟也不例外，这就使得国内一大批质量不高的VCD碟得以“起死回生”；超级解霸提供的图像捕捉和录像功能可截取VCD中的精彩片段，从而帮助用户独自制作VCD，满足了用户的特殊要求；而它的MP3压缩功能更是紧跟了时代潮流，使得用户可以在听VCD的同时压缩制作自己的MP3音乐。至于超级解霸的强劲震撼效果的HiFi环绕回响立体声功能，先进的预测读盘技术也都是XING之类的VCD播放软件无法与之匹敌的。另外，超级解霸在VCD的播放效果上更有它独特的一面，它特意提供了对3DNow、MMX等技术的优化，从而使得电脑播放VCD更加顺畅。正是因为有了上述法宝，超级解霸才得以令XING之类的其他同类软件汗颜，实现了执VCD播放软件之牛耳的目的。超级解霸由北京豪杰计算机技术有限公司开发，网址为http://www.herosoft.com。

　　NOT MACFEE＆NAV 我们用KILL和Pccillin

　　作为一个出色的杀毒软件，它首先必须具有全方位、多平台、高效率等特点，这样才能充分杀灭病毒，保护用户数据的安全。以往，这些要求只有美国的产品（如MCAFEE VirusScan、Norton AntiVirus等）才能满足，而现在，我们中国人也有类似的杀毒软件，如北京冠群金辰公司的KILL98（《电脑报》第十七期对KILL有过详细的介绍）和宝岛台湾的Pccillin。Pccillin是一个Windows下的杀毒软件，它具有对内存及硬盘中的病毒进行自动扫描、直接对压缩文件和电子邮件附件进行扫描、通过Active Channel直接获取最新病毒代码信息、提供Web Trap功能以阻止Internet Java Applet及ActiveX病毒、提供“手动扫描”和“即时扫描”等多种不同的病毒扫描方式。对检测到的病毒也提供了多种不同的处理方式，可在屏幕保护状态下从事后台病毒扫描以节省时间。它可任意指定扫描的文件类型，清除带毒文件时自动制作备份，允许对指定类型的文件或指定文件夹中的文件不予扫描等众多优秀功能。可以说只要外国杀毒软件能实现的功能，Pccillin都实现了，甚至在某些方面还超过了它们，如通过Active Channel直接获取最新病毒代码信息、在屏幕保护状态下从事后台病毒扫描以及用于阻止Internet Java Applet及ActiveX病毒的Web Trap功能是MCAFEE VirusScan、NAV等软件都不具备的。Pccillin的网址为http://trendmicro.com.cn。

　　

　　NOT AutoCAD 我们用华正电子图板

　　AutoCAD是美国AutoDesk公司的产品，长期以来国内有不少工程师在使用。如果需要购买电脑辅助设计软件，同样有很多国产优秀CAD软件是可以代替的。如华正电子图板，完全按照我国的制图和国家标准设计，具有良好的售后服务，其性能价格比也是AutoCAD所无法比拟的。华正电子图板采用纯32位设计，运行于Win95/98/NT系统，支持DWG等数据接口，是一个相当优秀的国产辅助设计软件。另外还有正直CAD7.0、清华高华CAD、目CAD、大凯凯达BCAD等都是国产CAD软件精品。华正网址为http://www.caxa.com。

　　

　　NOT PageMaker 我们用维思

　　维思(Wits)是北大方正开发的中文排版系统(另外还有面向专业排版领域的方正飞腾FIT排版软件)，它是基于Windows的交互式集成组版系统。在整个方正电子出版系统中负责图文组版部分的工作。除了其组版系统外，还有表格排版、数学公式排版、化学公式排版、三棋一牌(围棋、中国象棋、国际象棋和桥牌)、条形码、电子表格、简谱、五线谱等专用子系统，可方便地进行中、英、日、俄以及多种少数民族文字的彩色图文编排。维思的排版方式非常适合中国人使用。它支持TIF、JPG、BMP等多种图形格式，排版前还可进行图片和文字的预览。对已编排文件中的图形及文字可以随意放大、缩小和旋转。它还有几十种图形库，利用自带的底纹和花边可以对图形库中的图形进行方便地修饰。对编排的文字，维思可以进行颜色填充和立体、勾边等多种特效字的制作。它还充分利用了快捷键这一功能，将键盘上的大多数字母键定义了自己的功能，可以让用户更快捷地编排出自己脑海中设计已久的各种复杂的版面。维思强大的组版功能，绝不逊色于国外的各种组版软件，如PageMaker等，是国人自己开发的一个优秀组版软件之一。查询网址为http://www.founderpku.com.cn。

　　

　　NOT网络吸血鬼 我们用网络蚂蚁

　　使用网络蚂蚁下载文件，除了具有网络吸血鬼所具有的断点续传功能之外，还进一步扩展了断点续传的功能，可对同一文件进行多点同时传输（最多5点），也就是将同一个文件分成五块同时下载。试想一下：用五个连接同时下载一个文件，其速度是不是会比采用单个连接下载要快。据称网络蚂蚁采用五点同时下载方式时最多时可比网络吸血鬼快2～3倍！就冲着这一点，我们也该“换旗易帜”了。况且网络蚂蚁在其他方面上也提供了不少出色的功能，如自动拨号上网、自动下载文件、下载完成自动关机、断线自动重拨；提供在下载完成之后自动调用病毒检测程序对下载的文件进行扫描；支持HTTP、FTP代理服务器；提供剪贴板监测程序；连入Internet后，程序会自动开始所有未完成的下载任务；支持IE、Netscape浏览器的单击直接下载；提供了创建下载文件的说明文件功能，方便用户进行下载软件的安装和协助用户管理下载文件等，所有这些都极大的方便了用户的使用。看看这个优秀的国产软件，我们为什么不使用它呢？网络蚂蚁的网址为http://netants.yeah.net。

　　

　　NOT ICQ 我们用PCICQ

　　网上寻呼是目前一个比较热门的话题，资深网迷都拥有一个网络寻呼号，网络寻呼工具也就不可避免走进了我们在冲浪时的生活了。最早的网络寻呼机就是ICQ，它是一个英文软件，在处理中文信息方面并不完善，而且连接速度也比较慢，经常莫名其妙地死机，但我们也只有硬着头皮使用，因为它是我们唯一的选择。目前的情况就大不一样了，我们拥有了自己的ICQ软件――PCICQ，这是一个国人自己开发的网络寻呼机软件。它是一个全中文的软件，我们可以利用它方便地发送中文信息，也可以尽情的和好友在网上用中文聊天，效果可是不同一般。另外，PCICQ还具有同时支持163和169用户，可自选号码(3～12位)等优点。这些非常符合国内广大网迷的需要。PCICQ的网址为http://www.pcicq.com。

　　

　　NOT Outlook 我们用FoxMail

　　随着网络的流行，采用E－mail电子邮件取代传统的信函也成为一个新的时尚，如何快速高效对E－mail进行处理就显得尤为重要了。对此，很多用户采用的是IE4.0中附带的Outlook，不过国产软件FoxMail的功能要更强大一些。

　　

　　FoxMail一向以收发信件迅速、支持GB\BIG5\HZ等汉字编码、支持多个邮箱账户、可同时从多个服务器下载邮件等优点而著称。而最新的FoxMail 3.0更实现了多用户、多账户、多POP3支持的目标。它允许每个用户设置属于自己的邮箱目录结构和设置，并可设置不同的口令保护，从而实现了多个用户共用一台机器的FoxMail进行收发邮件的目的。FoxMail还提供了宏定义功能，它允许用户采用“你好，％TONAME”之类的宏代替每封邮件前的“你好，XXX”，从而节省了用户书写邮件的时间。该功能就连发明“宏”功能的“祖宗”――微软公司自己推出的Outlook也没有提供。至于FoxMail 3.0的“过滤器”功能则更令Outlook望尘莫及了。正因为这些优点，因此我们可以毫不夸张的说――FoxMail 3.0将是目前最合适中国用户使用的E－mail服务软件。FoxMail的网址为http://www.aerofox.com。
