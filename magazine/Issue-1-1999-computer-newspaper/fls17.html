系统维护工具Norton Utilities 4.0使用技巧
●赵巍　张英

　　一、Norton Utilities简介

　　流行的系统维护软件举不胜举，但无论哪一种，和Norton Utilities相比，只能是小巫见大巫。Norton Utilities是大名鼎鼎的Symantec公司开发的软件包，其注册用户已超过一千五百万，可见其影响之大，其新产品是Norton Utilities 4.0。
　　在中国，Norton Utilities在资深玩家乃至入门新手中都有口皆碑，但多数人安装后，觉得使用不方便又删除了。我们可通过许多人的评论来了解Norton Utilities的特点：
　　(1)体积太大，占用内存太多，降低了系统的运行速度。Norton Utilities集合许多系统维护功能，形成了一个庞大的软件包。而想要发挥它的各种功能，在启动电脑时就必须装载它，因此电脑启动会需要长一点的时间。
　　(2)查毒过敏。系统本来没病毒也老是提示有毒，这种情况比较少见。笔者遇到过一例，经KV300， VRV， Rising等等检查均报告无毒，独有Norton Utilities顽固不化地一直报告有毒。后来偶然想起在添加新硬盘时安装不得法，重装该硬盘，结果Norton从此不再提示有毒。此可见，Norton Utilities提示有毒时，系统很可能存在问题。
　　(3)界面太多，不好掌握。Norton Utilities可以部分地或全部地代替如WinRescue、 Winsafe、急救专家等系统急救软件，Windows 9x和Cacheman等系统优化工具，Regclean这类注册表清理软件、Clean System这类清除垃圾软件，以及Recover NT这类恢复删除文件的软件等等。它所能代替的这些软件和程序（它们对于维护电脑正常运行都大有裨益）如果全部安装会包括多少界面？Norton Utilities把诸多程序集于一身，可以实现操作的简便，更易于掌握。
　　如果不想因为死机、运行速度变慢、系统崩溃等等麻烦事愁白了头，如果想使自己的电脑永远在良好的状态下运行，最好装上Norton Utilities。你会发现它的确功能全面、强大、可靠，不负你的苦心。如果你是普通电脑使用者而不是高手，它对模可能更为宝贵。比如它能在没发生故障之前发现并提示潜在的系统问题，诸如某磁盘碎片过多，某应用程序有错可能造成无效页面等，所谓防患于未然。这是纠错软件极少具有的功能。
　　同时，掌握它绝不是难于上青天。它的界面清楚，操作程序统一，只要下点功夫，一两天足可应用自如。如果没有时间集中学习，不妨边用边学。在界面设计上，它已考虑到这点：点击任何界面的Help按钮，就可以立即得到关于该界面的说明。

　　二、安装

　　Norton Utilities下载地址是http://www.symantec.com，这是国外网址，下载速度慢。国内不少站点都有镜像下载，速度很快。笔者最近在网络世界的（www.benbencomputer.yeah.net）下载了Norton Utilities的一个庞大独立组件AntiVirus的最新版本，用网络蚂蚁几十分钟就下载完毕。安装时有精灵引导，基本上一路点“Yes”即可，与一般软件的安装没有什么区别。不过出现“Programs to Start Automatically with Windows”这个提示时，是要求你对启动Windows时是否同时启动Norton Disk Doctor/Norton System Doctor（磁盘/系统诊断程序）加以选择。以后每当启动电脑时，Norton就根据你的选择自动检测硬盘（包括软盘）或检测系统。如果选择了启动Windows即检测系统，检测完毕后，屏幕右下角的任务栏会出现一个象交通灯一样的图标。这种检测相当于Windows的磁盘扫描，其实除非“多事之秋”，必要性不大。笔者通常都关闭它。
　　安装结束时，安装程序会要求你创建急救软盘（Rescue Disk）。在软驱中插入空白软盘，Norton便会自动为你创建。如果不是空白盘，它会进行格式化。电脑系统发生改变时，比如添加了新程序，应重建急救盘。顾名思义，急救盘是系统万一瘫痪时在Dos下恢复系统的，因此非常重要。
　　安装后，桌面出现一个小图标，点击可调出下图1的界面。同时任务栏出现另外两个小图标：小旗形状的是启动加速器；袖章形状的是解冻结器，它可以解决运行某程序时出现的死机故障。

　　同许多软件一样，使用者可以随时通过Internet进行更新（点击主界面的“Live Update”即可进入更新站点）。Norton的更新有点与众不同，它不仅可以更新自己和独立组件AntiVirus，还可以更新安装在电脑上的许多其它软件程序，比如Windows的程序。更新时，只要下载了相关文件，无需手工干预更新就会自动完成。

　　三、主界面功能说明

　　1．综合应用 (Norton Utilities Integrator)
　　图1左侧一栏共四项，自上而下分别是故障查找与排除（Find and Fix Problems）、改善性能(Improve Performance）、预防性维护（Preventative Maintenance）和疑难解决 （Troubleshooting）。
　　图1右侧是上述四项的子项。此图是点击“Find and Fix Problems”后出现的子项。点击其中的一项，比如第一项，则会执行系统的检测，一旦发现问题可以手动或让Norton自动解决。如果选中左侧的Improve Performance，右侧就会改成与改善性能相关的子项。
　　Norton Utilities的各项功能都集中在这个界面里。换句话说，维护系统或解决系统问题都可以通过这个界面来实现。具体的应用在后面有详细的解释。
　　2．系统诊断（Norton System Doctor）
　　在自动检测的界面，它提供了硬盘分区的各种相关资料。总共可显示十六项信息，自左向右自上向下十六个格的具体功能如下：
　　1～4：显示磁盘的资源（可用空间）；5：CPU的负载情况；
　　6～9：硬盘的碎片比例，6中没有标明比例，说明磁盘碎片比例过高应该整理了；
　　10：磁盘坏区提示；
　　11：病毒特征库需要更新的提示；
　　12：病毒显示（此图显示有一个病毒）；
　　13：解冻，即检测并纠正造成死机的程序；
　　14：Norton系统需要更新的提示；
　　15：Windows系统需要更新的提示；
　　16：时间。
　　对于普通电脑使用者，不必管工具栏，按默认设置即可。

　　四、使用方法

　　下面结合主界面的各项功能，详细讲解Norton Utilities 4.0的使用方法。
　　1．故障检测与排除
　　首先点击桌面图标调出综合应用界面。如果打算查找系统错误，点击右栏的第一项“Norton WinDoctor”，查错范围包括Windows系统、注册表以及其它应用程序。首先出现的是欢迎界面，这里有关于系统检测的简介。请注意，选用其他任何功能也都出现类似的欢迎界面，熟悉后不妨把它关闭。关闭的方法很简单，只需要在它出现时选中其中的空白小方格。点击欢迎界面的“Continue”，出现另一界面。这里有三项选择：第一项(默认)是全面检测；第二个选择是部分检测；第三个选择是察看系统修复史，用于修复发生失误时进行改正。使用者可根据自己的判断确定检测的范围，初学者其实只需点击“下一步”采用默认选择。大约一两分钟后，检测结果列出。如果有错，点击界面的Repair All，错误就会立即得到改正。Norton检测错误的功能很可靠，而Regclean之类软件只能查注册表错误，对于其它错误无能为力。而Windows的磁盘扫描费时不少，查错的效率不敢恭维。
　　有时Norton列出三个错误，其中一个你拿不准是否应该修改，这时应选定其他两个，然后点击Repair All。用户最好在系统看起来运行完全正常时也定期进行检查，以排除隐患。在添加和删除某些程序之后，运行即使良好，也有可能埋下隐患，更应检查系统。
　　当计算机系统不稳定、打开窗口过多，或者某个应用程序过大，都会造成死机。一旦死机，往往工作前功尽弃，有时令人痛不欲生。为了防止这种情况发生，你可以打开主界面右侧的第二项（Norton CrashGuard），在出现的界面上选AntiFreeze（解冻），这时出现一个列表。如果表中某一项之后有“no responded”字样，运行该程序时就可能造成死机。选中它点击“下一步”就可以避免死机。解冻界面也可以通过点击任务栏的图标进入。
　　如果某程序不是潜在冻结而是即将冻结时，Norton也会弹出警告。这时我们可以在警告牌上操作。警告牌对死机恢复可能性的大小有说明，如果可以，点击“yes”多数情况可以恢复运行。不过，据笔者经验，这项功能开发是Norton最不成功之处，即使它提示恢复性是乐观的，你也不要乐观。出现死机情况时，我们通常会在Windows下按Ctr+Alt+Del去关闭冻结的程序。请注意，如果你安装了Norton，对话框中就会多出一项“Norton AntiFreeze”，读者不妨试一试不关闭应用程序，而用Norton去解冻。
　　Norton还可以检测和修复磁盘，这一点是众所周知的。它检测和修复的范围包括CMOS、Partition、Boot以及压缩盘等等。点击图1右侧第三项“Norton Disk Doctor（著名的NDD)”，然后在出现的对话框里选中要检测的磁盘即可，操作方法很简单。
　　删除一个文件之后发现删错了，这种情况是人人都会碰到的。现在有很多软件可以恢复删除，不过多数都要求你除了提供确切的文件名称之外，还要提供正确的路径等等，同时可恢复的范围有限。因此，一旦删除，恢复的可能性就很渺茫。现在不妨试试Norton的身手：选中右侧第四项“Unerase Wizard”，将出现一个对话框，上面列有三个选项，分别是：查找最近删除的文件；查找Norton保护的删除文件（通常是二十五个）；查找特定的删除文件。由于前两项操作比较简单，这里仅介绍第三项，点击选中第三项，在出现的对话框里填汛文件名称，点“下一步”。新界面有个列表，要求你选定文件的类型。如果你不记得已经删除的文件类型，不要慌，点“下一步”。这时出现的新界面要求选择文件原来所在磁盘。如果连在哪个磁盘也忘了，就继续点“下一步”。现在Norton开始查找，找了几分钟，结果可能令你失望：对话框空空如也。但Norton还留有一手。再点“下一步”试试看。这时Norton会在“unused”区域再次寻找。只要这个文件不是彻底地无影无踪，就会被找出来。
　　安装了Norton，回收站的图标会贴上“N”这个标记，说明回收箱已在Norton保护之下，其中的文件可以完整地恢复。这时的回收站不能用通常方法清空了，但清除也很容易：选中要彻底清除的一个或多个文件，这些文件的文件名会变蓝，把鼠标移到蓝色区，右键单击，在下拉菜单里点击“Delete”，就万事大吉。
　　图1右栏的最后一项是文件比较。但Norton只能对有限种类的文件进行比较，远不如“Duplicate Finder”这类软件功能可靠强大，这里不再赘述。
　　2．性能改善
　　选中图1左侧第二项“Improve Performance”，右侧的对话框变成下述三个子项：磁盘提速（Speed Disk）、优化向导（Norton Optimization Wizard）、清空向导（Space Wizard）。
　　(1)电脑运行速度变慢，很重要的一个因素是磁盘碎片过多。同时磁盘碎片过多也容易造成程序出错。因此，经常整理磁盘碎片是维护系统的重要手段。然而，Windows的磁盘碎片整理慢，尤其是在大容量且采用FAT32分区的硬盘上。Norton的磁盘提速就是磁盘整理，急性子的人对它的速度可能还会不满，但比Windows的要快很多。在开始整理前，Norton首先提供碎片占用空间的比例，并据此提出三种整理办法：全面整理、仅整理碎片、仅整理自由空间。初学者最好使用Norton建议的那种。如果是仅整理碎片或自由空间，整个过程瞬间就能完成。
　　(2)优化向导的范围包括交换文件（Swap File）、应用文件装载、注册表等的选项，初学者只需一路点击“下一步”。优化完成后，重新启动就能生效。
　　(3)清空向导的目的是清除没用的以及各种临时文件。其中有两个选项：简化或快速（Express）、全面（Comprehensive）。简化清理包括回收箱和各种临时文件，和Windows的磁盘清理程序无异。全面的清理同时还包括无用文件的清理。选中此项后，Norton会显示无用文件列表。但我们万不可以掉以轻心，不分青红皂白地把这些文件一概清除。其中有一些可能是最近一段时间内一直没使用但并非真正没用的文件。一旦删除，相关程序可能无法运行。
　　3.预防性维护
　　选中图1左侧第三项，右侧的子项是系统监测（Norton System Doctor）、急救盘（Rescue Disk）、镜像（Image）和注册表跟踪（Norton Registry Tracker）。
　　“系统检测”功能前面已加以介绍。“急救盘”包括急救软盘的创建，使用已建急救盘以挽救崩溃的系统；“镜像”与Windows 98的华生医生基本相同；“使用注册表跟踪”不仅可以察看注册表的改变，还可以改正注册表里关键文件的错误。
　　4.疑难解决
　　图1左侧最后一项相应的子项包括系统信息（System Information）、注册表编辑（Norton Registry Editor）和网上服务支持（Norton Web Services）。这里仅介绍一下系统信息。
　　一台电脑的硬件和软件再加上外围设备可能非常复杂，相关数据成千上万，某些数据连查询高手也感到棘手。然而这些数据对于安装某些新软硬件、排除电脑的某些故障可能至关重要。Norton对于电脑各种软硬件包括外围设备都能提供详尽的信息，打开“系统信息”界面，菜单栏共有九项：系统（System）、 显示（Display）、打印机（Printer）、驱动器（Drive）、输入设备（Input）、多媒体（Multimedia）、内存（Memory）、网络（Network）和互联网（Internet），这九项覆盖了与电脑相关的所有方面。仅以“系统”为例，Norton能给我们提供些什么信息：界面上方是有关硬件的信息，下面是操作系统信息。有关CPU、BIOS、BUS Type（总线）、Ports（端口）、Memory（内存）、Floppy/Hard disk、sound（声卡）、CD-ROM、Video（视频）、Windows、DOS等等的数据应有尽有。总之，用Norton Utilities查询电脑的资料非常方便。
