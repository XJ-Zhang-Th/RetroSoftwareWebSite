如何制作 Dreamweaver 的 Objects 类型插件

　　Dreamweaver 2.0 以其强大的功能，典雅的界面和良好的可扩展性而深受广大网友的欢迎。但是 Dreamweaver 自带的插件非常有限，你有没有想过也为 Dreamweaver 制作几个插件，让你在制作网页的时候更加轻松呢？
　　其实制作 Dreamweaver 插件非常的简单。Dremweaver 的插件格式与其它具有可扩展性的软件有所不同，它使用的是 HTML 格式的文件。打开 Dreamweaver 目录下的 Configuration 目录，在这个目录下我们可以发现有这样几个文件夹：Behaviors、Commands 和 Object。在这几个文件夹下保存的就是 Dreamweaver 的三种不同类型的插件。Object 类插件在这几种插件中是最容易制作的，你甚至不需要任何的编程经验就可以制作出相当实用的插件。此类插件实际上是由 HTML 文档中 部分的一个基于 Java 脚本的 ObjectTag() 函数，返回一个 HTML 字串到你当前编写的网页文件中。
　　下面就以如何制作一个插入版权标志的 Objects 类型的插件为例介绍一下如何制作此类插件：
　　首先使用任意的网页制作软件，建立一个标题（Title）为“插入版权标志”的空白 HTML 文档，在这个文档中不要输入任何文字。然后打开文档的 HTML 的源代码编辑方式，在文档的 与 之间插入下列代码：

　　将这个文件命名为“版权标志.htm”并保存在 Objects 目录的任意一个子目录下，然后启动 Dremweaver，在 Insert 菜单下你就可以发现多了一个名为“版权标志”的菜单项了。点击一下这个菜单项试一试……。同时你也会发现在 Objects工具条中多了一个带有 ？号的图标，将鼠标移动到这个图标上就会显示一个“插入版权标志”的工具提示。如果你希望在 Objects工具条中显示一个图标，你可以建立一个与这个插件同名的 18 ＊ 18 的 .gif 格式的图标。这样在 Dreamweaver 的 Objects 工具条中你就可以找到这个插件的图标了。(Weber)
