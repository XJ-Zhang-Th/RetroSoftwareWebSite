用Word97进行数学运算

　　1.用“运算符”计算
　　Word7中的“表格”菜单下内置有“公式”命令，调用该命令就可以应用“运算符”（加“＋”、减“ －”、乘 “ ＊”、除“ ／”、乘方“^”、百分比“％”、大于“>”、大于等于“>=”、小于“<”、小于等于 “<=”、不等于“<>”）进行计算,对于简单的数学计算题可以直接应用它编辑公式求出结果。例如：求=?，可在等号后,调用“表格／公式”命令，在它的“公式”栏内输入“=1.5^3／16^2＋3.2^3／25^2"后单击“确定”即可求出=0.07（如图1）。这里编辑公式的运算顺序与一般笔算顺序是一样的，可以基本按我们习惯的用笔运算的形式编写计算公式，十分方便。

　　2.用“粘贴函数”进行计算
　　在上述“公式”命令中，除“运算符”外，还为嫌编辑公式麻烦的朋友另外内置了一些简单的“函数”， 如你的计算问题恰好是其中一个函数或几个函数组合的求值问题，就可以不编辑公式，直接调用“函数”，在函数的自变量框内输入数值从而求出结果。但美中不足是，由于微软公司在设计中考虑不周，使得应用Word97中的“粘贴函数”计算时，对你在公式的选择栏窗口中选中的函数,没有它的功能说明,要你先记住它的功能才能用。为方便大家应用，这里将内置的函数名称与功能列表如下：
　　函数 功能
ABS(x) 数字或算式的绝对值（无论该值时正还是负，均取正值）。
AND( x,y) 如果逻辑表达式 x 和 y 的值均为true，那么取值为 1；
　　　　　　　　如果这两个表达式中有 一个的值 false，
　　　　　　　　那么取值为 0。
AVERAGE( ) 一组值的平均值。
COUNT( ) 一组数中的个数。
DEFINED(x) 如果表达式 x 是合法的，那么取值 1(true）；
　　　　　　　　如果该表达式不能计算，那么取值 0 （false）。
FALSE 0 （零）。
IF(x,y,z) 如果条件表达式 x 的值为 true，那么结果取y；
　　　　　　　　　如果该条件表达式的值为 false， 那么结果取 z 。
　　　　　　　　　注意 y 和 z （通常为 1 或 0零））可以是任意数值或单词
　　　　　　　　　“True”和“False”。
INT(x) 对值或算式 x 取整。
MIN( ) 取一组数中的最小值。
MAX( ) 　取一组数中的最大值。
MOD(x,y) 　值 x 被值 y 整除后的余数。
NOT(x) 如果逻辑表达式 x 的值为 true，那么取值 0（零）（false）；
　　　　　　　　如果该逻辑表达式的值为false，那么取值 1 （true）。
OR(x,y) 如果逻辑表达式 x和 y 中有一个的值为true，那么取值 1（true）；
　　　　　　　　如果两者的值都为false，那么取值 0 （零）（false）。
PRODUCT( ) 一组值的乘积。
ROUND(x,y) 对数值 x 舍入到由 y 指定的小数位数。x可以是数字或算式的结果。
SIGN(x) 如果 x 是正数，那么取值 1；如果 x 是负数，那么取值 －1。
SUM( ) 一组数或算式的总和。
TRUE 1。
　　下面，我们对应用粘贴函数进行计算举例说明：
　　例1:求高中代数中排列数=？
　　可调用“公式”命令后选择粘贴函数：PRODUCT( ) 后再在它的自变量的框内输入（16，15，14，13，12），单击“确定”，就可求出P= 524160。
　　例2：求－12＋22－32＋42－52＋62＋.....－492＋502=?
　　应用数学变形为3×25＋25×24×4／2后,调 用“表格／公式”命令，选用粘贴函数组合：SUN(PRODUCT(3,25),PRODUCT(25,24,4)／2),求得 －12＋22－32＋42－52＋62＋.....－492＋502=1275 （如图2）。
　　例3:比较233 与322的大小。
　　调用“表格／公式”命令中的粘贴函数：IF( )，在它的自变量括号中输入：（3^22>2^33,1,0),单击“确定”,就可得到该函数计算的逻辑判断值为:1,故322>233。
　　3、应用“插入Microsoft Excel工作表”计算
　　对文稿中涉及到的比较复杂的计算问题，以上两种方法可能无能为力。这时我们可以应用Word97中的“插入Microsoft Excel工作表”按钮，在Word文稿中插入一个Excel工作表，应用Excel工作表中丰富的函数功能进行计算。例如求：
|log|=？，可点击工具栏上的“ 插入Excel工作表 ”按钮，调用Word97的“插入Microsoft Excel工作表”功能；通常计算结果仅为一个数值，故对插入的Excel表格仅选一格即可。在计算机将Word97切换为Excel97后，单击其工具栏上的“插入fx图标”，再在出现函数列表后,逐一选用所需函数,依次插入到Excel97的单元格,在公式编辑栏内编辑出上述题目的计算机函数形式：
　　“=ABS(LOG(POWER(32,1／3)＊POWER(423,1／12)＊POWER(8,－1／30),5)－SIN(155＊PI()／180))”后，单击公式编辑栏上的“√”，就可得出0.565229如图3、图4）。
　　同理，对求行列式= ?,单击工具栏上的“fx"图标,在出现的“粘贴函数"列表中选用“数学和三角函数"类中的矩阵行列式函数：“MDETERM”，然后再在出现的该函数参数输入对话框中输入行列式的计算机表达式：“=MDETERM({12，23，17；21，8，43；－34，0，78})”单击公式编辑栏上的“√”确定，即可求出=－59188 。
