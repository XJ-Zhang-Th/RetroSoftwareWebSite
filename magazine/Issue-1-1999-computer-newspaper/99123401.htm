在你电脑中检测和修复2000年问题

　　随着2000年问题的日益迫近，我们越来越担心我们的电脑会因为2000年问题而在世纪之初罢工。一般来说，个人电脑的2000年问题主要存在于主板的BIOS中，而软件的2000年问题一般较少，流行的软件大多已经解决了这个问题。因此个人电脑的主要任务是检测主板的BIOS是否存在2000年问题。下面介绍几个免费检测和修复PC2000年问题的软件和方法。

　　一、Norton 2000 BIOS Test Retail Edition 1.0

　　文件大小：1.27 MB
　　文件名称： btestfix.exe
　　类型 ：免费
　　下载地址： http://www.symantec.com/
　　这是一个检测主板的BIOS是否能渡过2000年问题的软件，如果存在问题它还能修复。这个软件在《电脑报配套光盘99春节特刊》中可以找到（path\y2k\btestfix.exe)。运行btestfix.exe，将出现一个安装画面，依照提示完成安装。打开“程序”→“Norton 2000”→“Make BIOS Test Disk”，出现制作检测软盘的界面（如图1)。将一个刚格式化的软盘放入A驱或B驱（必须要重新格式化），选择相应的“Make A:”或“Make B：”的按钮， 软件将制作一个奇怪的启动盘（该盘只有一个biostest.sys文件），该软件将随后提醒你需要用这个软盘重新启动，并按屏幕提示完成操作。随后用这个启动盘启动，系统将重新启动两次，第一次检测BIOS时钟从1999年12月31日23：59：59到2000年有没有问题。第二次检测2000年到2015年的闰年是否存在问题(系统提示大约花费80秒钟）。在检测过程中对系统的问题全部选择Yes。测试完成后，如果你的系统出现问题。该软件将在config.sys中添加DEVICEHIGH=C:\PROGRA～1\NORTON～1\BIOSFIX.SYS ＋R ＋T,这样在以后的每次启动中你可以看到BIOS FIX的字样，表明你的主板的BIOS 2000年问题被软件修复了。

　　二、ACT－Y2000

　　文件大小：107KB
　　文件名称：acty2000.zip
　　类型：免费
　　下载地址：http://www.year2000.gov.hk/document/acty2000.zip
　　这是一款功能比较完善的2000年问题测试软件，它不仅能对BIOS进行测试，还能对应用软件进行测试。难能可贵的是它还有一个详细的硬件和软件对2000年问题解决方法的清单（安装目录下的y2k_hw.doc、y2k_sw.doc）。也许是以上原因，它被香港资讯科技署推荐为选用软件。这是一个必须在DOS下使用的软件。下载该软件后用Zip解压后，执行其中的install.exe。程序自动安装在c:\acty2000目录下。用命令行方式启动Windows98,执行acty2000.exe，出现界面（如图2)，根据界面上的几个菜单，这个软件可以完成BIOS和软件的2000年问题检测。在菜单中选择相应的字母选项可以激活相应选项。第一个选项（[A]－ About Automatic Compliance Tester for Year 2000）是介绍ACT－Y2000这个软件的。 第二个选项（[B]－Test PC BIOS ＆ Scan PC software in local machine）是马上全面检测本地硬盘中的所有软件和主板 BIOS的 2000年问题。 第三个选项（[C]－ Test PC BIOS for Y2K compliance ）是只对硬件进行2000年问题测试，值得注意的是，在这个测试中有可能系统要重新启动。当出现“Result: The test is PASSED.The PC BIOS under test is found to be Year 2000 compliant.Leap Year Test is PASSED!（结果：测试通过，BIOS2000年测试通过。闰年测试通过!）”时，表明系统硬件没有2000年问题(如图3)。如果发现任何问题，它将生成一个acty2000.log的记录文件。 第四个选项（ [O]－ Scan for non－compliant PC Software ）可以选择特定的目录里的软件进行2000年问题检测，如果发现任何一个软件有2000年问题，程序将在屏幕中显示，并记录到acty2000.log记录文件中去。第五个选项（[L]－ View the log file）是当你测试完毕后，可以在这里看完整的测试报告acty2000.log(如图4)，并打印。 第六个选项（[V]－ View the user guide）是当你在使用时有任何疑问可以在这里看它的帮助文件（不过全是英文的）， 第七个选项（[X]－ Exit the utility）是退出。

　　三、手动检测

　　如果你没有任何软件，用下面的方法也可简易地查到你的主板BIOS有没有2000年问题。
　　1.开机进入BIOS设置日期时间项，检查日期项中是否用4位数字表示“年”。如果不是，则该电脑存在2000年问题。
　　2.将日期设置为1999年12月31日，时间设置为：23：59：01，观察BIOS的时钟在走到59秒后能否顺利变到2000年1月1日0：0：01。如果不能进到2000年，则该电脑存在2000年问题。
　　3.将日期设置为2000年2月28日，时间设置为：23：59：01，观察BIOS的时钟在走到59秒后能否顺利变到2000年2月29日0：0：01。如果不能变到该日期，则该电脑存在2000年问题。
　　4.检查 02/29/2000到03/01/2000、 02/28/2001 到03/01/2001、02/28/2004到02/29/2004、 02/29/2004到03/01/2004等日期的过渡情况，若不能正常过渡，则该电脑存在2000年问题。
