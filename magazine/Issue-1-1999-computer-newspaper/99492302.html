清除光盘刻录失败故障

　　故障现象：一台电脑，配置为：奔腾133 CPU 、64M EDO内存、mediaCamp5多媒体显卡、4.3G硬盘 、Delta 6速读2速写的CD－R刻录机（IDE接口）。装有Pwin98SE、刻录软件Adaptec Easy CD Creator 4 Deluxe和 Nero 4.0.3.5汉化版。光盘刻录机购置一年有余，刻录绿盘50余张，坏盘率仅为4％，而且均为掉电、死机等所致。Delta刻录机价格低廉，刻录效果又如此之好，自然欣喜万分。然而不幸的是，停用半年后，近期重新又开始刻录，结果40多张绿盘竟然没有一张刻录成功。根据Easy CD Creator显示出来的刻录状态表明，总是在365或371处受阻，而Nero则显示出错信息为“SCSI错误，无法完成刻录”。情急之下，赶紧拨通163，上网找了个相关的新闻组向高手们求个“药方”。有的认为，这属于IDE接口刻录机的劣根性，无法救治，只有将其更换成SCSI接口的刻录机才能根本解决问题，而另外的则认为是刻录头坏了，没救！面对高高一叠的报废绿盘，心疼银子的同时，后悔当初不该为节省SCSI卡的银两而买了这令人如此懊恼的IDE接口刻录机。
　　故障解决：懊恼归懊恼，问题还得想方设法来解决，总不能将2000多元的Delta扔掉吧。首先，运用“排除分析法”，找出症结：1.会不会是盘片质量所致？不会，因为不可能连续40多张盘片都会有质量问题，更何况使用的不是零卖散片，而是带有出厂包装的正规绿盘。2.会不会是操作系统的Bug所致？将系统重新安装，Pwin98SE改成Pwin98，再改成一年前的Pwin95B，问题仍然存在，因而可以断定，与操作系统无关。3.会不会是刻录软件的Bug所致？Easy CD Creator 4 在安装过程中，的确有出错信息，而Nero 4.0.3.5又是个汉化版，或许汉化过程对该软件某些部分造成损毁，或许这两个刻录软件相互冲突。为此，卸掉Nero和Easy CD 4，依次改装为Easy CD 3.5或3.0或2.5，甚至一年前使用的Easy CD 2.0，结果问题仍未排除，看来同刻录软件也没有关系。4.会不会是刻录机IDE接口设置不当与硬盘冲突所致？硬盘接在IDE1上，而刻录机接在IDE2上，始终如此，过去使用正常，现在也应该不会有问题。有可能导致刻录机刻盘失败的上述诸因素都被一一排除了，可问题并没能解决。
　　那么，症结到底在哪里呢？难道真是IDE接口刻录机固有的劣根性所致不成？内心很不情愿接受这样的结论，便掂量起那40多张被刻录机淘汰下来的盘片。突然发现，其中真正被激光头刻录过而报废的盘片只有几张，大多数则没有一丝一毫被刻录的痕迹。这种状况表明，计算机屏幕显示出刻录过程，其实并不一定真有激光射到盘片上。如此断定，问题与计算机主机软硬件及数据接口无关，而与刻录机内部有关。深入推敲，不会是激光头坏了的缘故，否则便无法读取光盘数据了；也不会是激光头老化的缘故，否则一定会有刻录痕迹留在每张盘片上。唯一的可能性，便是有东西部分遮挡住了激光头发出的激光束，令其衰减到能读而不能写的程度！得出这样的分析结果，当然是可喜的，因为果真如此，便不必把IDE接口的刻录机扔到垃圾箱里去了。
　　其次，运用“物理诊断法”排除故障：1.打开机箱，卸下刻录机。2.将刻录机底面朝上，用十字改锥卸掉4个固定螺丝，拿掉底盖。3.将刻录机底面朝下，小心卸下刻录机的金属外壳，露出刻录机的机芯，注意刻录机塑料面板左右两端的固定销，不要弄断了。4.取几张镜头纸，将其卷成小纸棒，注意不要将纸纤维毛边露在外面。5.用纸棒的一端，从里向外轻轻擦拭刻录机机芯上的小透镜。哇！没想到不足绿豆大的小透镜，竟从表面上拂下一层黑灰，而且似乎还有些油腻好像是油烟所致。擦拭小透镜时，千万不可用力太大，否则小透镜位置偏移，刻录机便报废了。6.用镜头纸将刻录机里其他部位的灰尘一并擦尽。7.将刻录机金属外壳原样装好，然后把刻录机重新装到计算机上。
　　盖上机箱后，启动计算机，进入Pwin98SE，在刻录机里放入一张空白盘片，任务条右端Systray 里的Creator 4自动调出了Adaptec Easy CD Creator 4 Deluxe，一年前那种随心所欲刻光盘的感觉又被找了回来……万万没想到，油烟竟然是刻录失败的真凶。
　　(安徽　王旭东)
