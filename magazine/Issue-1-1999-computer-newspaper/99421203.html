硬件配制文件“轻装上阵”

　　Windows98的“即插即用”功能让我们在安装新硬件时倍感亲切，但可惜的是“她”不具备硬件自动卸载能力。这样，对于经常更换外设的用户来说就很不方便，因为他们的硬件配置文件将越来越臃肿，极大浪费系统资源。笔者的电脑前一阵子就处于这种状态，通过实践成功地实现了“减肥”。
　　安装Windows98时，Windows98自动检测硬件情况并将其保存在注册表中，同时生成一份初始的硬件配置文件（Original Configuration），利用这个文件来启动计算机。当外设发生变化时，Windows98还使用原来的硬件配置文件启动系统，由此造成系统资源的浪费。那么如何生成符合当前硬件情况的硬件配置文件呢？可按如下步骤进行：
　　一、选择“开始\设置\控制面板\系统”，单击“硬件配置文件”标签，出现图1对话框，列表中的“Original Configuration"即目前电脑所使用的硬件配置文件。
　　单击 “复制（C）…”按钮，在弹出的“到（T）…”对话框中输入一个新文件名，如“Original Configuration old1”，单击确定按钮，出现如图2的对话框，得到一个新的硬件配置文件。这样做是为保证计算机系统中存在不同的硬件配置文件，这种情况下Windows98才会重新检测硬件并生成一份新的、符合当前硬件情况的硬件配置文件。
　　二、选中“Original Configuration"，单击“重命名（R）”，在弹出的对话框中输入一个新文件名（不同于“Original Configuration”），如“Original Configuration old2"，单击“确定”，再单击“系统属性”对话框的“确定”按钮。
　　三、以上操作完成后，重新启动计算机，蓝天白云出现后，系统显示硬件配置文件菜单：
　　Select one of the following configuration:
　　1. Original Configuration old1
　　2. Original Configuration old2
　　3. None of the above
　　Enter your choice:
　　键入“3”回车，这样Windows98就会重新检测计算机硬件系统并生成一份新的、符合当前硬件配置情况的硬件配置文件，名为“Original Configuration”，并用它来启动系统，此过程大约需要2～3分钟（具体时间由您的计算机配置情况决定），同时屏幕上会出现提示信息。当Windows98启动成功后，我们的目的就达到了。
　　四、当然，请别忘了再次进入“硬件配置文件”标签，将“Original Configuration old1”和“Original Configuration old2"两个文件删除。否则您每次开机都要面对硬件配置文件菜单了。
　　（江苏　李馥娟）
