选个软件打IP电话


　　IP电话软件现在越出越多，真是“乱花渐欲迷人眼”，有心尝试IP电话的朋友，挑选起来不免有力不从心的感觉，所以在此我们专门推荐了几种IP电话软件，并对它们进行比较，给以详细的说明，希望可以对朋友们有些帮助。

　　一、NetMeeting

　　类型：免费，微软公司在IE中附带。
　　下载地址：http://tucows.hnpta.net.cn/files/Nm21.exe
　　最新版本：2.1 Build 2203
软件功能：NetMeeting除了为你提供声音的交流外，在相应硬件的支持下，还提供视频图像的传送，让交流者可以彼此看到对方。同时提供白板、共享应用程序等功能。
　　评价：对于不熟悉英文的使用者来说，中文NetMeeting是他们的最好选择(注意，NetMeeting本身分中文和英文两种版本)。而且不用费事地到处去寻找，IE 3以上的中文完全版本直接内置了NetMeeting,中文Win98更是直接内置了NetMeeting的安装版本，只要你安装的时候选择该项即可。
　　NetMeeting中文版的亲和力是这几种IP电话软件中最好的，但是连接速度却是最慢的，这也是它最致命的弱点。NetMeeting登录主机的过程非常漫长，登录后刷新使用者资料以及在通话者之间建立通话连接等，都是慢腾腾的，最恼火的是等待半天后，显示你呼叫的对方不存在或者网络无法连接，让囊中羞涩的中国网民不禁暴跳如雷。有时候长时间的连接等待，让人搞不清楚到底是机器死机了或者是网络出了问题。
　　所以，NetMeeting最适合对英文不熟悉，但有充足的上网时间和相对较富裕的上网费用的朋友使用。

　　二、IPhone

　　类型：ShareWare,不付费可以正常试用14天，到期后会提示注册，如果不注册将无法继续正常使用。
　　下载地址：http://tucows.hnpta.net.cn/files2/iphone5.exe
　　最新版本：5.01
　　软件功能：IPhone可以让你用声音和对方说话，还提供视频图像的传送支持，同时还具有文字交谈、白板、声音邮件（声音邮件的播放需要用vmplay.exe)、文件传输等功能。在已经付费注册并购买了通话时间后，可以用IPhone拨打国际长途，费用比用电话打国际长途便宜得多。
　　评价：IPhone是全英文的界面，对于英语不熟悉的朋友可能会觉得使用比较困难，它的使用和设置是所有IP电话软件里最为复杂的，当然功能也是一流的，它是唯一同时提供PC与PC通话（同时支持视频），以及PC与普通电话通话功能的软件，是Internet上最富盛名的IP电话软件。
　　最新的5.01版本使用了浏览器来作为聊天室基本界面，又大量使用Java技术，所以登录服务器并显示使用者资料的过程反而比前期版本4.5.5慢了许多。如果你仅仅是用于网络通话，那么笔者还是建议你使用4.5.5版更好些。另外，IPhone的注册使用期限，也大大限制了它的使用，你只有付费注册，才能长时间地使用该软件。
　　建议真正需要和国外亲友电话联系的朋友使用它，最好请比较熟悉IP电话的朋友为你安装并调试。

　　三、Net2Phone

　　类型：FreeWare,但是使用的时候除非拨打800免费电话号码，否则必须付费。
　　下载地址：http://tucows.hnpta.net.cn/files3/n2p9054.exe
　　最新版本：9.057
　　软件功能：专业提供通过网络打到普通电话的功能。
　　评价：Net2Phone在Internet和电话系统之间建立了联系，这样就可以用你的电脑拨打电话，它的费用低廉，但由于音频质量的问题，你的朋友也许听不出那是你的声音(另外，目前IPhone也可以提供类似的服务了）。
　　现在Net2Phone在世界各地纷纷建立了代理服务器，目前国内也已经有了代理商。各位可以使用人民币购买使用时间了，通话费率可以参看其主页。
　　使用界面直观简单，安装也并不太复杂，但是需要对英文有些熟悉。在网络速度快的时候，声音听起来尚可。当然，具体的语音质量和当时的网络连接速度以及你欲拨打的地区有关。
　　非常适合有亲友在国外，而亲友本身又不上网的用户使用。(注意：经过本人使用，且经过查看Net2Phone帮助文件求证，发现Net2Phone并不适用于那些不直接和Internet相连的用户，所以169的用户将无法正常使用Net2Phone，请169网的用户注意了。）

　　四、MediaRing Talk

　　类型：目前为免费，正式版本推出后，将需要在使用时交通话服务费用4.95美金/月
　　下载地址：http://chinesewoman.com/98.9/mrtalk.EXE
　　最新版本：MediaRing Talk 99
　　软件功能：提供网络两用户之间的实时通话，并采用了特殊的呼叫技术，可以让通话双方不必事先约定时间。
　　评价：MediaRing Talk采用了一种面向台式计算机用户的新式 Internet 电话方案，它是一个具有全新概念的IP电话软件，提供PC与PC的通话。使用它的双方不需同时在网上，它提供了语音留言功能。由于采用先进压缩算法，在网络拥挤时，仍能保持让人比较满意的通话质量（有着独一无二的漂亮界面！是我所见过的IP电话软件中最为漂亮的一个，如图）。
　　MedIaRing Talk采用特殊的一响即断的震铃方式呼叫对方的计算机。所以，并不要求通话双方事先约定时间，也不需要事先与 Internet 连接。当对方的电脑正巧开着时，调制解调器即可激活他机器中正在后台运行的MediaRing Talk软件，让他的计算机自动拨号上网并准备开始用MediaRing Talk和你通话。
　　软件正式版出来以前,MediaRing Talk完全是免费的，连4.95美金/月也不用交。所以各位朋友不妨下载一个MediaRing Talk99来试试。另外，MediaRing Talk的中文帮助系统非常详尽，强烈建议你多多阅读！MediaRing Talk音质不错，是种较有前途的IP电话软件，建议喜好新奇的朋友安装来看看，效果不错喔。(注：经过本人使用发现,MediaRing Talk同Net2Phone类似，169的用户将无法保证正常使用。）

　　五、VoxPhone

　　类型：ShareWare,不付费可以正常试用14天，到期后会提示注册，如果不注册将关闭一些高级功能，但是最有用的通话等功能将继续保留。
　　下载地址：http://www.voxphone.com/pub/toolvox.exe
　　软件功能：VoxPhone提供真正的多人交谈，话质优良，语音圆润，VoxPhone还提供按E－mail地址呼叫对方，直接输入IP呼叫对方，地址簿功能，通话的同时传输文件功能，以及基本的chat功能。
　　评价：我使用过多种IP电话软件，但是在我国的网络传输速度下，语音的效果能达到可以和我们普通电话效果相媲美的，却以VoxPhone为首选。
　　VoxPhone不像其他的IP电话软件，要求对方必须使用相同的软件才可以通话。VoxPhone本身提供的会议功能，更是令其冠绝天下，它建立可以多人实时会话，实现真正的多人交谈。另外，一般IP电话软件都是试用版本，有试用期限，VoxPhone虽然也是个试用软件，免费试用期也只有短短15天，到期不注册会立刻关闭它的“多人会议(Conference)”“地址簿(Address Book)”等高级功能，但是它保留了最为重要的通话功能。所以如果你仅仅是用它来和朋友一对一的讲话联系(这不正是我们要使用IP电话的目的么？)，那么你可以永久的使用它。
　　VoxPhone另外还提供有独一无二的“变声”功能，你可以令自己的声音听起来像太空人，像小孩……
　　VoxPhone确实设计合理，如果你在通话的同时给对方传送一个文件，你会惊异地发现，通话质量丝毫不因文件的传输而受到影响，因为VoxPhone小心地控制了文件传输。另外，VoxPhone的语音处理技术十分优秀，在较慢的传输速度下，也能获得很好的通话效果。
　　综合上述，对于广大网虫来说， VoxPhone是他们首选的IP电话软件！ (VoxPhone可以顺利地运行在163网或者169网上，但是163用户呼叫169用户时，将发生无法呼通的现象。169用户呼叫163用户无问题，但是169用户呼叫169用户时，也将发生无法呼通的现象。解决上述问题的唯一办法是获得对方的IP,让VoxPhone用“IP呼叫功能”进行呼叫。)

　　六、Cool Talk(Conference 4)

　　类型：免费，Netscape公司在Netscape Navigator完全版本中附带,免费赠送
　　下载地址：ftp://ftp.netscape.com/pub/cooltalk/windows/ctalk32.exe
　　或者： http://www2.nease.net/files2/tucows_n32e408.exe (Navigator完全版本）
软件功能：提供两个网络用户的即时通话，以及白板，文件传输，网页电话簿，地址簿等功能。
　　评价：作为和IE竞争的Netscape,所推出的这个软件的功能也是相当不错的，早在Netscape 3.0的时代，就已经推出了CoolTalk，现在随Netscape 4.04一起发行的是Conference 4，不管怎么说，Netscape提供了一个替代NetMeeting的选择。但是一般用户使用此软件的极少，可能很多人也并不知道Netscape提供了这么一个东东。一般随同Netscape Navigator一同安装，设置相对很简单，而功能和NetMeeting比，除了无法提供视频传输外，速度却快得多。
　　建议酷爱使用Netscape的网络爱好者使用，音质一般，且可以自行调节音质编码等，可以较好地适应国内的网络速度，不过速度很慢时，必须以牺牲部分声音质量为代价。(文/江苏油田设计院 谢勇)
