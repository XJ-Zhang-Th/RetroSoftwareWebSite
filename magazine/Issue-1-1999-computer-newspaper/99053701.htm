FOXBASE自测题


　　一、填空题
　　1．数据管理经过了三个发展阶段，分别是______、________和________。
　　2．数据处理是指_________，数据库处理的数据范围是_________。
　　3．数据库应用系统指____________。
　　4．数据库的数据可抽象为_______、_________和_________三种数据模型。
　　5．FoxBase中的数据可分为五大类型，它们分别是______、________、________、________和________。
　　6．FoxBase的文件类型有______、________、________、______、________、________、________、______、________。
　　二、选择题(多项选择题)
　　1．FoxBase的主要功能有( )
　　A．数据的输入与更新　B．文字处理
　　C．开发应用程序　 D．建立数据库
　　2．下列正确的日期表示有( )
　　A．07／08／98　B．15／32／98
　　C．12／30／98　D．40／12／98
　　3．下列数据中属于字符型常量的有( )
　　A．75.75　B．“75.75”　C．[学生] D．'75.75'
　　4．下列属于逻辑型常量的数据有( )
　　A． “.T.” B． .F. C． .y. D． .k.
　　5． 符合FoxBase运行的硬件环境有( )
　　A．640KB内存
　　B．单色显示器(CRT)
　　C.软盘驱动器与硬盘
　　D．IBM PC兼容机
　　6. 把内存中内存变量保存于.MEM文件中的命令是( )。
　　A．RELEASE B．SAVE
　　C．RESTORE D．LIST
　　参考答案：
　　一、 填空题
　　1． 人工管理阶段、文件管理阶段、数据库系统阶段
　　2． 对各种类型数据结合加工过程的总称、数值与非数值的各种信息
　　3． 在数据管理系统的基础上为某一特定的用户或特定的用途而编制的应用系统
　　4． 网状模型、层次模型、关系模型
　　5． 数值型、字符型、逻辑型、日期型、备注型
　　6． 数据库文件(.DBF)、数据库备注文件(.DBT)、内存变量文件(.MEM)、文本文件(.TXT)、索引文件(.IDX)、命令文件(.PRG)、屏幕格式文件(.FMT)、报表格式文件(.FRM)、标签文件(.LBL)
　　二、 选择题
　　1. A C D 2. A C 3. B C D 4. B C 5. A B C D 6. B
