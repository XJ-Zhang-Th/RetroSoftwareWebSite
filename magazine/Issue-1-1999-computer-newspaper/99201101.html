文件索引工具――DiscoMania

　　如今的硬盘容量越来越大，装载的数据文件也越来越多，很少有人能说清装了哪些文件。这里就介绍几个索引、注释工具，或许能帮助你摸清自己的“家底”，为你查找文件提供方便。
　　在笔者用过的几个文件索引、注释工具中，DiscoMania 2.04（国内下载：http://home.jxdcb.net.cn/~newhua/file/discoma.zip）使用简单（直接运行，无须安装），功能全面，而且体积较小（878K），成为笔者的首选。下面对它的使用作一简单介绍。

　　一、建立文件索引

　　DiscoMania 2.04可以给某一逻辑盘或文件夹建立索引，并提供多种检索方式。
　　1.选择索引对象：在Win95/98下运行DiscoMania，单击程序的界面窗口（如图）“Catalogs”菜单下的“Create”命令，打开索引设置对话框。单击对话框中的“Drive or Directory Name”下的按钮，选择需要建立索引的逻辑盘或文件夹。然后在“Options Lnclude”下的六个选项中，设定索引包含的文件和文件夹类型，这六个选项依次为：隐含文件、只读文件、系统文件、子文件夹、隐含文件夹、ZIP文件夹。若将对话框中的“Confim Each ZIP Extract”项选中，则建立的文件索引还包括ZIP压缩包中的文件。
　　2.开始索引：选择好索引对象后，单击索引设置对话框中的“Index”按钮，DiscoMania即开始建立文件索引，结果将显示在界面右面的索引列表窗口中。
　　3.设置索引名：索引建立完成后，在索引设置对话框中的“Catalog Name”框内输入索引名（可用中文），单击“Save”按钮。
　　4.索引管理：
　　查看全部索引：选中界面左窗口已建立的某一索引，单击“Catalogs”菜单下的“View All Catalogs”命令，则该索引中的所有信息即显示在右窗口中，包括文件名、文件大小、文件建立时间、路径等，你可以很方便快捷地查找文件。
　　删除索引：选定DiscoMania左窗口中的索引名，选择“Catalogs”菜单下的“Delete Selected”命令，即可将该索引删除。
　　更新索引：若建立的索引对象由于增删文件发生了变化，可进行索引更新。选择菜单命令“Catalogs/Update Selected”，单击弹出对话框中的“Update”按钮完成更新。
　　索引输出：DiscoMania可以以文件或打印两种方式输出已建立的索引。如果要将索引数据输出为文件，先选择一个索引，打开“Visual Table/Save Contents As”菜单。选中“Text File”、“HTML File”或“Csv File”三种文件格式之一，在对话框中选择存盘路径、输入文件名，单击“保存”即可输出该索引文件。

　　二、注释索引文件

　　DiscoMania可以给文件或文件夹建立注释（备注），便于用户了解文件的相关信息。
　　1.文件注释：选中一个索引，选择“File Comments”菜单下的“Add/Edit”命令，打开文件注释对话框，即可在对话框下面的空白格中输入注释文字（上空白格为注释历史）。建议将重要文件和有特殊意义的文件加以注释。注释的文字可在索引列表窗口中的“Comments”项查看。
　　2.目录注释：选择“Catalogs”菜单下的“Vice/Add/Edit Selected Comment”命令，然后输入注释文字即可。

　　三、其他索引工具

　　1.豪豪档案柜（http://www.tensing.com.cn/newhua/down/fm_setup.zip），这是一个国产的文件注释工具（Win95下面需要VB5运行库）。可以像DiscoMania那样为文件或文件夹做注释，同时可以输出生成的注释文件。
　　2.Advanced Disk Catalog 1.20（http://home.jxdcb.net.cn/~newhua/file/adc120.zip）是一个与DiscoMania功能近似的目录管理工具（只是体积大了点），可以为文件和目录注释，自动产生报告以及搜寻重复文件等。
　　3.Explorer Notes 2.1（http://www.zdnet.com/pcmag/pctech/content/17/15/ut1715.001.html）一个文件和目录注释工具，简单好用。功能单一，没有文件索引功能。
