让Win98改头换面的E-Icons 98


　　虽然Win98带给了我们多方面的好处，但它自带的桌面主题显得不尽人意。虽然它可以用真彩色显示图标，改变背景图案、颜色、字体名称和样式等，但想要更改开始菜单中的图标、文件夹、各种文件的图标却不是那么容易。如果有了E－Icons 98(以下简称为Ei98)后，做到以上几点就轻而易举了。
　　1.Ei98的安装非常容易，把大约2M的zip文件解开后安装，指定路径，待安装完成后自动在程序菜单中建成一项E－Icons 98。
　　2.启动Ei98后出现了简单的界面，主要由两个下拉菜单、11个快捷按钮、3个标签组成，单击“E－Icons 98”按钮弹出菜单第一项“Change Icons”，用于更改各种图标，功能相当于第一个标签。“System”是第二个标签，用于更改回收站的名称，刷新图标，确定是否使用所有颜色显示图标，是否使用大图标，改变后是否返回警告信息等。“Apply Change and Exit”则用于实现所有改变并退出。“Apply Change”与前者差不多，只是实现改变后不退出。“Rebuild Icon Cache”用于刷新图标。而“Open Icon Theme”则可打开图标主题(Ei98也自带了两个相当不错的图标主题)。“Save Icon Theme”可将你自己的图标主题保存下来。“Change Any Folder Icon”用于改变某一个文件夹的图标。“Author”可显示作者信息。点击“Exit”则退出。而“Help”菜单中则给出了帮助主题，在“Internet”中给出了作者Email和连接Ei98主页的选项。最后一个“About E－Icons 98”相当于第三个标签，是关于Ei98的信息。
　　Ei98是免费软件，它个不大，但麻雀虽小五脏俱全。喜欢的朋友可以到这里去下载：http://zfiles.zj.cninfo.net　(重庆　缪　星)
