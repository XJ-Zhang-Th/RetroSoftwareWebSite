找回失踪的猫

　　1996年我购置了一台兼容机，配置为：P133 CPU、16M内存、昆腾大脚1.2G硬盘、华硕T2P4主板、14英寸彩显，这在当时是高配置，但计算机的飞速发展却使我的“宝贝”迅速落伍。后来为了上网又添置了64M EDO内存及一台贺氏33.6K MODEM，倒也能勉强应付，上网速度也可以忍受。由于现在的软件越来越大，动辄数以百兆计，光WIN 98就将近300M，我那当初还是“海量”的硬盘已不堪重负。今年三月，我又添置了一块昆腾火球七代6.4G硬盘，可是这块花了我1440元的硬盘却给我带来了烦恼，我的“猫”失踪了，费了好大劲才找回失踪的“猫”，其中的喜忧不少人也许曾经历过。
　　由于新硬盘容量大、速度快，因此我准备将它作为第一主盘，原有的硬盘作为第二主盘，这样就省了将原有的硬盘拆下设置跳线为从盘的麻烦(硬盘出厂时缺省设置为主盘)。打开机箱，将原来插在IDE1口上的老硬盘及光驱连线转插在IDE2口上，将新硬盘连线插在IDE1口上。检查连线无误后，接通电源，开机，进入BIOS设置，将Chipset Feature Setup项下的Onboard PCI Enable项设为“Both”，IDE 0 MASTER／SLAVE Mode及IDE 1 MASTER／SLAVE Mode均设为“Auto”，让BIOS去自动侦测硬盘。存盘后重启，再进入BIOS设置，选IDE HDD Auto Detection ,BIOS开始自动侦测硬盘，结束后存盘，用WIN98启动盘重启，用FDISK分区后格式化，一切OK。接下来用DOS6.22启动盘重启，安装DOS6.22后，再用WIN98启动盘重启，安装WIN98，同样一切顺利，重启后顺利进入WIN98。
　　此后安装应用程序均未出现问题，但在安装MODEM时，却发现无论如何也检测不到COM2口上的MODEM，当时觉得非常奇怪，打开“我的电脑／控制面板／系统”，发现各项设备均正常，COM2口也正常，也没有设备冲突现象。怀疑操作系统有问题，于是重装WIN98，问题仍未解决，难道是MODEM坏了?在未装新硬盘前是没有任何问题啊!再说贺氏虽然倒闭了，可产品是有口皆碑的，用了半年不到就坏了也太没道理吧?但为了解决问题，仍将鼠标接到COM2，将MODEM接到COM1，开机系统提示未安装串口鼠标，接下去又提示发现了33.6K MODEM，并自动安装，这说明MODEM的连线及本身都没有问题。排除了MODEM出问题的可能性后，怀疑COM1口有接触不良的问题，虽说T2P4主板早已过时，但我对这块当时的“主板王”还是非常看重的，现在出现这个问题，也只好再想办法。
　　由于T2P4主板支持PS／2鼠标，曾想换成PS／2鼠标，把COM1口腾给MODEM用，但这样一来，我的两个串口鼠标就废了，总是心有不甘，于是一遍遍重装MODEM的驱动程序，同时按照WIN98的帮助文件把不能检测到MODEM的种种方法都试了一遍，没有任何作用。但我仔细观察在检测COM2口上的MODEM时，MODEM上的TR指示灯有反应，看来还有解决的可能性，会不会是安装了双硬盘的原因?仔细看了一遍T2P4主板的说明书，没发现有任何线索，试着将插在IDE2口上的老硬盘及光驱拔下，重启，设置BIOS后，进入系统，还是检测不到COM2口上的MODEM，看来不是双硬盘的问题。
　　能想到的办法都试了，回想安装了双硬盘后只调整了有关IDE设备的选项，不曾动过别的设置，而且在这之前一直工作正常，因此一直都没有动BIOS设置的念头，但现在看来有必要仔细检查一下BIOS设置。拿定主意后，重启，再进入BIOS设置，仔细看了一遍各项设置，终于在Chipset Feature Setup项下的UARTS2 Use Infrared处发现此处设为“Enable”，而此项设置好像跟COM2口有关，于是仔细看了一下主板的说明书，发现该项设置只要设为“Enable”，则系统就将COM2口的资源分配给红外线通讯设备，串口设备就不再能使用COM2口，也就是说COM2口上的MODEM就不能再激活。看罢说明书，赶快将UARTS2 Use Infrared项设为“Disable”，存盘后重启，安装MODEM，果然检测到了我失踪多时的“猫”!
　　看着找回来的“猫”欢快地眨着眼连上了因特网，真是别有一番滋味在心头。回想整个过程，始终不能明白，为什么原来只有一个硬盘时，UARTS2 Use Infrared项设为“Enable”，也能正常检测到COM2口上的MODEM，而且还上了好一阵子网，安装了双硬盘后就不行了，虽然自己敢肯定在安装双硬盘的过程中不曾改过此处的设置，但又觉得以人家华硕的技术水准也不可能有这种BUG，再想想像我这样坚守T2P4的人恐怕还有一大批，因此将我的寻“猫”历程记录下，但愿大家在添加新硬盘时不要动扔掉主板或串口鼠标的念头，毕竟我们口袋里的“银子”太少了!
(河北　陈军)
