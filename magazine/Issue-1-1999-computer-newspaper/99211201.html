突突网上选软件

　　很久了，一直忙忙碌碌的，觉得自己做了很多。不知道是哪一天，是为了什么，也许是看见小孩子明亮的双眸，也许是编辑们心中的童趣。其实也许只是因为这个六月，让我想起了曾有过的那些快乐而无忧无虑的日子。让我们想起该为可爱的孩子们做点什么。于是我们开始关注这个领域，于是我们发现，其实在电脑里，孩子们的世界也很精彩。

　　松鼠博士来加密

　　

　　Decoder 是一个让孩子用来为邮件加密解密的软件。当Internet 进入了家庭，不光是家长可以上网收发电子邮件，小孩照样可以使用电子邮件。小孩一样有自己的隐私权，所以Decoder 为小孩发送的电子邮件进行加密以及接收的邮件进行解密。
　　Decoder 提供了两种方法：Encoder（邮件加密）和Decoder（邮件解密）。小孩子在发送邮件之前可以用Encoder 对所需发送的邮件进行加密，将发送者的姓名及邮件内容进行加密。将加密后的代码复制到剪贴簿中，通过第三方的邮件发送软件将剪贴簿中的代码发送给自己的朋友们，如果对方未安装Decoder软件，就会对这封加密过的邮件束手无策，可以达到小孩子之间恶作剧的目的。
　　Decoder不论是在加密还是在解密过程中，都有一个“松鼠博士”在一台机器旁繁忙地进行操作，看似正在进行加密或解密，然后逐渐显示加密或解密后的信息。并且在操作过程中和显示解密后的信息时还伴随着一种合谐的声音，给人一种生动的感觉。
　　遗憾的是，Decoder不能使用中文，只能输入英文，但这样可以锻炼孩子从小使用英文，以便为将来能更好地掌握计算机作准备。
　　下载网址：http://www.hypno.net/pub/download/decoder.zip

　　大头伯伯压缩机

　　

　　Zip up the web Pro是一种压缩网页进行离线浏览的软件。它通过将网页下载到本地硬盘后，再进行阅览，有着漂亮的界面，所有操作在界面上一览无遗。这种软件通过离线方式浏览网页，既可以节省上网费用和时间，又能保证网页的健康性（针对小孩）。
　　Zip up the web Pro的主要功能是将用户需要的网页下载并经过压缩保存到硬盘中（后缀名为.exe文件），随时可以进行阅览。它具有下载容易、快速等特点，并且可以设置将主页的链接1～5层压缩到硬盘中，甚至将整个网页的内容全部下载。
　　当在用Zip up the web Pro进行压缩下载时，可以选择保存文件的类型，保存以及将保存的网页取出浏览都非常容易（通过双击保存文件就可以进行浏览）。用户可以在压缩时，自动地让被压缩网页。通过缺省的电子邮件系统发送给其它用户（这种方法适用于孩子离父母较远的用户家庭），并且，用户还可以指定是否在压缩的网页中加入水印。为了随时知道给孩子看的网页是否为最新内容，可以在压缩的网页后面加入日期和时间，并在其中加入密码保护。
　　因此，Zip up the web Pro 适用于那些有孩子上网的家庭，特别适用于孩子离父母较远或孩子经常不在身边的家庭，用以管理孩子上网。
　　Zip up the web Pro只能提供注册下载（4MB）：http://www.zipuptheweb.com/
　　注册费用：＄29.95。
　　另外，还可以免费下载Zip up the web Lite版本：http://www.zipuptheweb.com/download/ZipUpTheWeb.exe

　　阿猫阿狗浏览器

　　Monja Kids作为一种浏览器和电子邮件系统，在其中加入了卡通类的动画和图片，属于卡通式样浏览器和电子邮件系统，专门针对儿童制作，真是太好了！并且在Monja Kids中，无论是浏览器还是电子邮件都是采用卡通画作为图形特征，并且背景也采用多种颜色。

　　

　　对小孩子来说，Monja Kids软件的操作非常简单，但又不会失去它作为浏览器和电子邮件系统这两大功能的特征。在浏览器中提供了“向前”、“向后”、“刷新”和“停止”等功能，这些都属于浏览器的基本功能。伴随着打开网页的过程，会出现卡通人物的动画，儿童可在进行网页浏览的同时欣赏卡通动画，使得小孩子在上网时不感到枯燥，而觉得有趣味。并且在浏览窗口下方会出现动态的文字（从右至左移动，也可设定为停止），更显得动感十足。
　　在电子邮件系统部分中，撰写邮件和回复邮件的窗口会出现5个可爱的卡通画，它们分别有5种共25种预设的卡通动画，小朋友可以将这些卡通动画附在邮件上发送给自己的好朋友，这样就可以相互之间收到带有卡通画的电子邮件了，真是帅呆了！并且， 电子邮件的功能相对来说也比较简单，虽只有“收取”、“撰写”、“回复”和“删除”等功能，但又显得不缺什么功能。

　　

　　操作系统中必须安装有IE4.X浏览器，该软件才能使用。下载网址：http://www.monja.com/kids/

　　安全小卫士

　　

　　KidSafe Explorer 是由Arlington公司针对儿童制作的网页浏览器。它有着漂亮的界面，根据孩子的爱好而设计，满足孩子的新奇感。
　　KidSafe Explore的特点是家长为孩子选择网页。在第一次进入系统时，由家长设定一个密码，指定小孩用KidSafe Explore可以访问的网页。用一个通俗易懂的别名来替代那些复杂的网页地址，并且还可用卡通图片建一个网站书签列表，让小孩子通过卡通图片就可以知道该网页的内容。
　　孩子就只能访问你所指定的网页，不让他接触那些有害的内容，为孩子提供一个健康的网络空间，有利于孩子的健康成长。这样，既能增长孩子的电脑知识，又能防止一些不健康的内容对孩子的成长造成不利的影响。家长可以随时根据孩子所需而设定网页，由于设定了密码，小孩不能自行进入设置，起到真正的防范作用。
　　KidSafe Explore 还可以实现几台计算机共享一个设定的网页列表，这样就可以由一个家长或一个老师来管理多个孩子或学生上网，在学校特别适用。
　　因此，KidSafe Explore适于家长和老师对孩子上网进行一定范围内的控制。
　　KidSafe Explore 的系统需求：操作系统中必须安装有IE4.X浏览器，该软件才能使用。
　　家长朋友可以到以下网址下载试用30天的软件，下载地址：http://www.arlington.com.au/kse/setupkse.exe
