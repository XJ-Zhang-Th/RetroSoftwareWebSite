游戏修改大师―Game Master操作技巧
●周振兴

　　GM（Game Master）是来自台湾的又一款游戏修改工具，由精讯资讯有限公司（原FPE5.0的代理商）代理发行。从功能上，GM并不输于FPE，实际使用也证明了其游戏修改能力之强大。

　　一、GM的特点

　　GM 6.X（包括GM、GM Plus、SFE三个工具）是一个通过修改游戏中电脑内存的数值来帮助你渡过游戏难关的工具，有了GM你将在游戏中所向无敌。《Game Master》以及《Game Master Plus》是针对目前的Windows 9.X操作系统开发出来的游戏修改软件。对于许多不善于修改或不曾修改过游戏的使用者而言，起始的第一步总是找不到头绪，现在有了GM 之后，就算你完全不会修改，也可以由 GM 所提供的地址表格中的读取功能将高手修改的成果直接取得并分享，或者利用GM所提供的制作无敌档功能，这样你不用执行GM，也可以到达无敌的境界。以下是GM的一些功能：
　　1.自制无敌档（EXE文件）：即使朋友家中有GM也可玩到无敌版。
　　2.超级GM Plus功能：针对某些游戏在启动修改软件时会造成无法再回到游戏的无奈情形，特增加了游戏进行中直接下达修改指令的功能。
　　3.中英文切换功能：在使用GM时，你可以随时切换成中文版或英文版，方便了不同语言习惯的使用者。
　　4.还原搜寻功能：以往的修改程序在进行搜寻时，常会因为一时的失误而破坏正在进行的资料分析过程，GM特别提供还原的功能，让你不再追悔莫及。
　　5.游戏存盘文件扫描功能：这是一个先进的修改功能，你可以直接对比和修改游戏的存盘文件，一般的修改程序只能根据经验或利用编辑工具对存盘文件进行修改，而GM却能提供类似内存扫描的功能，并能在地址表格中随时修改。
　　6.多重搜寻功能：可以同时搜寻五十个目标，每个目标都可自己命名。
　　7.Interrupt Game（中断游戏）功能：可以让你强制性暂停游戏的进行，令搜寻更加准确。
　　8.浮点数扫描功能：可以扫描浮点数的整数部份，从而修改从前无法修改的游戏。
　　9.重新编写的低阶扫描程序：扫描效率大增，而且可以扫描4GB的内存范围。
　　10.地址表格储存／读取功能：你可将你的修改结果直接储存，方便下次读取直接使用，不需重新进行搜寻修改。
　　11.可以随时上网站上传或下载地址表格档，取得或提供游戏修改结果（游戏金手指功能）。
　　12.地址表格打印功能：可以方便你整理修改结果，将搜寻结果打印出来。
　　13.将地址表格存成HTML格式：方便你随时加入网页中，与网友分享修改结果。
　　14.编辑内存界面及功能：方便你随心所欲修改软件资料。
　　15.抓图功能：你随时可以抓取心爱的影像作留念。
　　16.图像浏览程序：同时支持BMP、JPG、GIF、EMF、ICO等多种格式。
　　17.系统编辑程序：你可以有更大的自主权，决定某个软件可用的系统资源。
　　18.招式记录功能：可以将游戏招式记录下来，让你在玩格斗游戏时更加轻松地使用必杀技。

　　二、呼叫GM

　　GM虽为32位的Windows应用程序，但也能够修改在Windows 98/95 DOS窗口以DOS模式执行的游戏。且GM有相大强大的呼叫能力，能在任何游戏中呼出，并自动找到正确的程序，你只要在游戏中按下“热键”就可以了（“热键” Windows 9.X下的默认值为“F12”，DOS下为“Ctrl＋F12”）。你也可以自己设置一个热键，只要用鼠标点一下“系统设定”栏,然后选“热键设定"即可，但键名必须在F9～F12之间。
　　还有一种呼叫GM的方法：在“资料搜索”菜单内，按一下“刷新”按键，GM会列出所有运行中的窗口程序名称，你可以用鼠标点选一个即可，以后就以“Alt＋Tab”来切换游戏和GM。

　　三、操作说明

　　GM是以分析内存变动情况来查找目标的，所以你必须在目标的数值有所变动时以GM来分析它。举个例子：例如我们玩红色警报，刚开始有5000金钱，现在我们开始进行修改：

　　步骤一：通过热键（如“Ctrl＋F12”）呼出GM（如图1），在“资料搜寻”栏的“搜寻目标”框中输入目前的金钱数“5000”（你还可以输入十六进制数、文字、数字列、范围列、低阶搜寻符号），然后按下“搜寻”（或直接回车），GM就会开始分析，并把可能的内存地址数告诉你。然后我们按“回到游戏”按键返回游戏。
　　步骤二：回到游戏后花点钱，如买飞机花掉638单位金钱，此时金钱数应为4362，再以热键呼叫GM，仍然在“搜寻目标”框中输入目前的金钱数“4362”，按下“搜寻”，此时GM告诉你的内存地址个数应该会少很多了。如果还是太多，请使游戏中的金钱再次变动，并重复步骤二直到只有3个以下的可能内存地址为止。
　　步骤三：经由多次搜寻，内存地址位置会逐渐明确，只要最后结果数目为“1”时，即表示所需的地址找到了。这个时候，你只要在找到的地址上双击鼠标左键，就会出现“Setting”窗口，依照你的需求输入数据即可。此时你还可以使用“自动锁定”锁定数值。（当然，你也可以单击鼠标右键，这时会出现一选择菜单，你可以选“加入一个地址”，也可以选“加入多个地址”，加入到“地址表格”窗口进行编辑，这就看你的具体情况了）。
　　特别需要注意的是“用热键改变”这个选项，它是在众多游戏修改工具中独一无二的功能，如果你调用了这个功能，你只要按键盘的“+”键目标数值就会增加1，按“-”键数值就减1，按“*”键就会自动锁定当前数值。因此只要我们找准金钱所在内存地址并加以锁定，那就会有花不完的金钱了。

　　四、高级使用

　　GM Plus这个功能是针对某些不能用“ALT-Tab”或热键转移到到其它程序,或转移后不可再返回的游戏（如《风云》）制作的，有了它你可以在游戏中输入一些指令来指挥GM在游戏后台进行自动搜寻,而不需要切换到GM窗口（GM Plus要和GM同时执行才能生效），而搜寻结果你可在显示器的左上方看到。
　　指令解释:
　　SCAN??：
　　例如: Scan‘abc'表示命令GM搜寻‘abc’字串；Scan11.5表示命令GM搜寻11.5这数值。
　　Clear：表示命令GM清除搜寻结果。
　　Add??：
　　例如: Add99表示命令GM把所有搜寻结果全部放到地址表并锁定数值为99，?? 的部份可以是字串或十六进制数等。
　　List：表示命令GM列出头8个搜寻结果和结果数量。
　　Value??：
　　例如: Value123表示命令GM把地址表的所有数值锁定为123，?? 的部份可以是字串或十六进制数等。
　　Lock：
　　表示命令GM把地址表中所有数值锁定。
　　unLock：
　　表示命令GM解除地址表中所有数值的锁定。
　　Pause：
　　表示命令GM将强制性暂停或是恢复执行游戏（但一定要用Option中的,Interrupt 功能来执行游戏才能生效)。

　　五、补充说明

　　1.游戏记录修改
　　SFE是“Save File Editor”的简写，是一个简单易用的游戏记录修改程序。你并不需要懂得十六进制或程序语言，你只需要一个游戏记录和你想修改的数值，便可以靠SFE来修改游戏记录（图2）。
　　先按下“Load File”的按扭，然后找出你想修改的游戏记录，你可以在“Input a number ”中输入把你想修改的数值，例如在RPG游戏中主角的体力值是100，你可以输入100。再后按下“Scan File”的按扭来搜寻，搜寻后结果可在右边看到。通常在第一次搜寻时结果会很多，这时你可以按下“Seek with last result”的按扭，然后返回游戏，令主角的体力值减少一些，存盘，然后重做上一步。如此类推，直至搜寻结果只剩下一个就可以修改了。便用鼠标按一按右边的结果直至变成蓝色，再到“Please input a number for Modify（请输入一个更改后的数值）”中输入新的数值,然后按下 “Modify”修改。最后按下“Save”， 把修改好的资料储存到原本的档案。
　　2.金手指功能
　　金手指起源于日本电视游戏机，指的是游戏中的一些特殊地址，如生命、魔法等重要的数据，它的使用方法如下：开机后执行GM及GM Plus，选择地址表格，读取游戏的金手指挡案，以“风云”为例：进入GM的 “系统设定”，选择用中断（Interrupt）功能来执行游戏，在指令栏位中输入游戏路径（如C:＼Program Files＼Enigmas＼Winc＼Winc.exe），或直接使用档案浏览功能选择。
