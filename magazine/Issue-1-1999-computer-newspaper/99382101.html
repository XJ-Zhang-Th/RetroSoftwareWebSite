RIVA TNT终极优化

　　目前第四代3D旋风刮得正凶，难道我的第三代显卡真的无用武之地？升级的路不通，我只好在超频（OVER CLOCK）和优化之间找到一条自己的路。
　　如果说nVIDIA年初推出的“DETONATOR” 雷管驱动程序大大提高了TNT性能的话，那么能够兼容TNT的TNT2驱动程序的推出无疑是玩家的福音（据说第一版的TNT2驱动程序就可以把TNT的性能提高13％）。从nVIDIA的“DETONATOR”开始，我对每个驱动程序都做了测试（测试机配置为：ASUS P2B主板、创新TNT、P2－350 超频到466MHz、64MB PC100 SDRAM），最后我还是选择了创新原配的基于nVIDIA 2.04版的驱动程序（从稳定性和性能两方面考虑）。
　　由于驱动程序是Beta测试版本，所以创新没有提供SETUP安装文件，而是需解压缩的INF文件。第一次装驱动程序可以选“更改显示适配器”，然后指定位置找到该驱动程序所在的文件夹，升级驱动程序也基本相同。安装完成后在系统设备里可看到驱动程序更新日期为1999年7月1日，驱动程序的版本为4.11.01.2103.03.0204。
　　第一步安装结束，在桌面空白处单击右键打开“属性”，选“设置”→“高级”→“BlasterControl”，你现在看到的是显示适配器信息：显示芯片为 nVIDIA TNT NV4（TNT的代号，最新要推出的TNT3即GeForce 256代号是NV10）。在上面的工具栏框中选TWEAK，最上面的是“Direct3D/Performance/Compatibility（D3D性能和兼容性）”，第一项“Enable fog table emulation (打开雾化表仿真功能）”建议打勾选上，虽然对速度会有一点影响，但为了游戏的画面还是值得的。下面是“Use DirectX 5 Compatibility mode（启用DirectX 5兼容模式）”，建议不要选中，虽然只支持DirectX 5的游戏还有不少，如《NFS Ⅲ》、《FIFA 99》等，但随着DirectX 6的普及和DirectX 7的即将问世，这一选项变得没有多少意义，打开的好处是跑DirectX 5的游戏会更快。再往下是“Direct3D Mipmapping（D3D 贴图）”，第一项“Automatically Generate（自动贴图级别）”有的文章介绍说设的值越小越好（比如0），但是我发现打开Powerstrip后自动贴图是关闭的，这就说明“0”是不可取的，那到底多少更好呢？想当初我刚买回卡时用的是创新去年8月24日的驱动程序，这一项提供的“MIPMAP LEVEL”最多是6，到了后来是8，现在是12，由此可以看出12是我们的最佳选择。下面的是“Auto－mipmap method（自动贴图方式）”，建议选“Trilinear(三线性贴图过滤）”，这会使你在游戏中看到更好看更真实的天空和光线。接着来是“Mipmap detail level（贴图细节级别）”，建议选“Best image quality（最佳图像质量）”，这样在游戏中的贴图会更好看，其实即使你选了“Best performance（最佳性能表现）”，游戏中的“performance” 也不会有多少改变的，而贴图质量下降却很明显。下面还有一项别忽视了：“Allow mipmap dithering（允许贴图抖动）”，建议打开，因为DirectX的这项功能会在游戏中通过适当的抖动达到以假乱真的效果。如果你只装完了驱动程序而没有设置以上这些选项，性能会受到限制的。
　　好了，喘口气，我们接着来优化。看到了“Advanced...”了吗？点击打开，你首先看到的是“More D3D”选项，首先选中“Anti－Aliasing（边缘抗锯齿，也叫边缘柔化），并把滑动条滑到“Better image quality”处，旧版本的驱动程序是以“Performance”和“Image quality”相对倍数的关系来表示的，看了让人莫名其妙（以前好像有篇文章提到设为4×4最佳，我经过对比和测试证实4×4即是“Better image quality”）。这一项的功能是让你的FIFA99的球场边沿线看上去不再像是锯齿，让你的劳拉不再深一脚浅一脚地踩进墙壁里。下一项是“Texture Alignment（贴图纹理排列）”，这是为数不多的需要选默认值的地方。
　　下一项是“Texture Cache Size：Use up to__ MB of system memory for textures(贴图缓存大小，即你准备分出多少系统内存用作贴图显存）”AGP接口的TNT这里没有必要设置，因为是系统自动分配的（规则是不超过系统内存的一半），PCI接口的显卡可按规则自己设置（值设的太大会造成系统使用大量的虚拟内存）。下面的是关于显示器刷新率的设置：
　　“Wait for VSYNC（等待显示器刷新同步）”建议去掉，你也可通过“Powerstrip”调节相应的选项，选“Disable”即可。
　　“Render no more than __frames ahead（提前渲染不超过多少帧）”，笔者通过查询资料并多次实验证实设为2比较好。
　　紧接着是第二栏，“Open GL”、“Buffer Region Extension（缓存区域扩展）”下面的两项都选中。首先“Enable Buffer Region Extension”，然后“Allow dual planes extension to use video memory（允许二维平面扩展使用显存）、最后选中“Texture optimization（贴图优化）”。这里就看你自己的爱好了，如果像我这样追求Open GL画面的精美，那就选“Best Image Quality”；如果是狂热的Quake族就选“Best Performance”吧，那样速度会更快。“Other Options”中基本没什么可调的啦（除了“Memory clock speed”显存速度可调以外）。不过建议超频还是用Powerstrip好。
　　OK，TWEAK里的优化完成了。要想获得更高的性能，不超频怎么行？ 想超频不解决散热问题怎么行？我的显卡在芯片散热片上和背面正对芯片的位置各装了一个风扇，超到115/130MHz不成问题。这时你可以在1024×768 16位色下毫无顾忌地玩《NFS Ⅳ》了（我把全部效果打开之后也丝毫不觉得画面停滞）。
　　好了，我想经过这些优化的TNT显卡至少在GeForce 256问世前是不会让你在游戏中捶胸顿足了！怎么样，试试看？
（显卡之家 tomrun)
