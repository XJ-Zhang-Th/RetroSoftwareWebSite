把MP3变成CD

　　你想把你的经典MP3转换烧录成CD后加以珍藏吗？MP3 CD Maker可以帮助你完成这个心愿。
　　MP3 CD Maker是一个很Cool的烧制CD盘片软件，界面做得很漂亮（如图）。它用MP3音乐作为原料，通过编码转换，制作出来的CD盘片可以直接在CD机上播放；它同时也是一个MP3解码器，可以将MP3文件转换成WAV文件，只要单击鼠标右键，选取“Export wav”；它甚至可以直接生成WinAmp的Play list。
　　MP3 CD Maker使用非常简单。选择一个MP3文件，然后按“Record”按钮就可以直接烧录。在真正烧录之前可以“Test”，烧录速度可以根据刻录机情况选择，软件自己推荐使用1倍速，大概这样的成功率会高一些。双击MP3文件可以播放该文件，也可单击窗口左边的“小喇叭”按钮进行试听。
　　MP3 CD Maker自动识别刻录机，它支持的刻录机型号很多，比如雅马哈的Yamaha series，惠普的HP CD－Writer＋ 7100、7200、7500、8100、8200，飞利浦的Phillips CDD3610，理光的Ricoh MP6200S，MP7040A，松下的Panasonic 7502、7582等等。它在直接烧录CD的时候，不生成临时WAV文件，所以速度很快。
　　顺便说一句：用MP3 CD Maker“Make”出来的CD文件，大小与普通CD文件差不多。
　　MP3 CD Maker文件很小，下载文件不到500K，下载网址http://www.zy2000.com。它是一个shareware，注册费29.95美元，未注册版本限在一张CD上烧录4首歌。
