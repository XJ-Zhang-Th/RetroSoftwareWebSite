Internet术语释义


　　公用网关接口：公用网关接口CGI（Common Gateway Interface）是一个可以产生相同结果或随用户输入而变化的程序。CGI程序一般完成Web网页中表单数据的处理、数据库查询和实现与传统应用系统的集成等工作。
　　数字数据网：数字数据网DDN（Digital Data Network）是利用数字信道传输数据信号的数据传输网。它可以向用户提供高质量的端到端的数字型信道传输业务，提供话音、数据、图像信号的半永久性连接电路的传输网络。
　　动态宿主配置协议：动态宿主配置协议DHCP（Dynamic Host Configuration Protocol）是管理IP地址的服务器。在网络中使用DHCP，网络管理器便可不必预分配IP地址，而是将一组有效地址分配给DHCP服务器，由DHCP服务器来管理那些地址。
　　综合布线网络系统：综合布线网络系统IDNS（Integrated Distribution Network System）是建筑物与建筑群的传输网络。它不仅使语音和数据通信设备、交换设备和其它信息管理系统彼此相连，也使这些设备与外部通信网络相连接。
　　综合业务数字网：综合业务数字网ISDN（Integrated Services Digital Network）能够支持多种信息媒体的传送，为用户终端提供端到端的数字链接。它的特点是能够通过一对用户线提供多种类型的信息通信业务。
　　局域网仿真：局域网仿真LANE（Local Area Network Emulation）是一个ATM Forum标准，它使得ATM可以兼容所有的网络协议。
