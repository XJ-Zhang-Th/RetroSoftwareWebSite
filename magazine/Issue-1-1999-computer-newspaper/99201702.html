TNT家族的四兄弟

　　大家对即将大规模上市的TNT2显卡可能早就垂涎已久，但是你知道TNT2还有兄弟姐妹吗?它们各有什么本领?下面请听笔者为你一一道来。

老大――TNT2 Ultra

　　它是TNT2大家庭中当之无愧的“大哥大”，据笔者了解的情况，nVidia给TNT2 Ultra制订的核心频率为150MHz，而它的显存频率高达183MHz。想当年TNT2芯片上市时，nVidia制订的标准MCLK(显存频率)仅为110MHz，Ultra是它的167倍，你可以想像到由此所带来的性能提高幅度。当然如此之高的频率会使成本大大提高，只要是使用TNT2 Ultra的显卡一般都会配置32兆显存，而且为了配合183MHz的高频，必须使用5ns左右的SDRAM，此种SDRAM生产技术仅被IBM、三星等公司掌握，价格十分昂贵。据一位显卡大厂的工程师说，一块Ultra TNT2显示卡的出厂成本在150美元左右，发烧友可推算出经过层层盘肃后的零售价了。另外，Ultra芯片虽然使用025微米生产工艺，仍然必须安装风扇，否则……。这完全可以理解，因为其晶体管数量太多，耗电量高达95W，封装引脚数高达496针，复杂程度已超过了Pentium Ⅲ，所以nVidia特别推荐使用AAVID(一家著名的风扇、制冷器生产商)的风扇来进行散热。笔者得到Ultra的初步测试结果是3D Winmark99分值为2700分，确实够Ultra(超级)。只不过大家面对它更Ultra的价格不知还能不能开心，TNT2 Ultra也是一款敢于向专业级3D加速芯片提出挑战的产品，ELSA为它开发了全新的OpenGL ICD，它把3D Labs的VX1、Permedia 3作为竞争对手，“Ultra”可真是一个多面手啊!

二当家――TNT2

　　TNT2不只比TNT2 Ultra少了一个“Ultra”，其核心频率和显存频率都有所下降，分别是125MHz和150MHz，不过nVidia推荐使用的显存是167MHz的SDRAM，为将来的发展留有余地。TNT2芯片仍然采用496针PBGA封装，核心电压比TNT2 Ultra的285V低025V，发热量有所降低，但加一个风扇仍是必不可少的降温措施。TNT2的耗电量比TNT2 Ultra降低了3W，对AGP插槽的电源功率要求大为减轻，对于TNT2 Ultra来说，你一定要买一张高档主板来与之配合，而TNT2只要有普通的主板就够了。生产TNT2显卡的厂商很多，因为6ns(可上至167MHz)显存的来源较广，价格也相对便宜，因此TNT2在整个家族中可能是最“得宠”的。笔者有一个未经证实的猜测：那就是既然TNT2与TNT2 Ultra的生产规格如此相似，nVidia很可能会将Ultra芯片中的不良品当作TNT2来卖，因此发烧友拿到的一些TNT2可以超频当TNT2 Ultra用。

三少爷――TNT2 Model 64

　　nVidia为了降低生产成本，提高设备通用性，生产了与前一代芯片TNT针脚完全兼容的320针TNT2 Model 64，虽然其内核仍然沿用了一些TNT2的技术，例如其核心频率和显存频率仍保持在125MHz和150MHz，不过它的显存通道只有64位，性能必定会有所降低，nVidia肯定将其定位在中低档层次产品上，因为它推荐使用7ns(143MHz)和6ns(167MHz)两种显存，不同的显卡厂家可以根据自身需要选择不同价格的显存以满足市场的要求。大家可能注意到nVidia原来将TNT2分为TNT2 Ultra，TNT2 Plus和TNT2 Pro三种型号，Model 64就充当了原来Pro的角色，现在大批生产TNT显卡的厂家可以在第一时间内转产Model 64，“三少爷”可能会成为家族中产量最多的产品。

小弟弟――Vanta

　　Vanta实际上是nVidia进军品牌机主板的第一颗棋子，除了其封装工艺与TNT2 Model 64差不多外，整个内核与TNT2没有太多的相似之处，其核心频率已降至100MHz，而显存频率也低至125MHz，nVidia的推荐配置是16M或8M 125MHz SDRAM，实际上许多显卡厂商都只采用8M 8ns显存。Vanta芯片的核心电压是25V，耗电量只有48W，可以算TNT2家族的小不点了，它上面只需安装散热片就已足够了，省去了AAVID风扇的几个美金成本，它更容易做到主板上去，不需要对主板的走线以及电源部分做太多的改动。因为Vanta和TNT的管脚完全兼容，所以其理所当然成为TNT的廉价替代者，然而它的RAMDAC只有250MHz、8M SDRAM的显存数量显然限制了它在QuakeⅢ、Unreal2、Photoshop、Autocad等各种高端软件中的使用性能。
