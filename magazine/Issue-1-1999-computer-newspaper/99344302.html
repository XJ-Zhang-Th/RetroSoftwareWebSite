极品飞车系列回顾

　　在大城市中有这样一群人，他们有的腰缠万贯，有的一贫如洗。不管是成年人或是孩子，他们都有一个共同的爱好――喜欢各式各样的汽车，这些人就是城市中的爱车一族。他们关心各大汽车制造商每一年的新款车，几乎每次车展都要去大饱眼福，家中名车杂志本本俱全，对汽车的型号更是像球迷熟背球星姓名一样如数家珍。这样描述他们虽有些管中窥豹，但这种热情确是真实的，是常人难以企及的。
　　不过，我们今天并不是要来分析这一颇具特色的社会群体，而是要说说能让所有车迷梦想成真的一部游戏，在这里你能开上在梦中都见不到的梦幻赛车，这便是我们游戏世界中的名车大全――《极品飞车》系列游戏。
　　《极品飞车》系列游戏到目前为止总共发行了六部游戏，下面我们就分别来看一看这六部游戏，了解一下它们各自的特点。
　　极品飞车　　The Need For Speed
　　当时这部游戏是著名的《ROAD＆TRACK》杂志与电子艺界联合制作。游戏中可供玩家选择六个赛道，以及八辆当年非常火爆的跑车。另外，在游戏中还有这些赛车的详实资料，像跑车的性能、历史、机械参数，甚至真实赛车在赛道上的电影都一应俱全。当玩家驾驶赛车比赛时，赛道上会不时地有警察出现，当他发现你超速时，便会将你拦下来。如果超速的次数达到两次时，便会被取消比赛资格。
　　由于当时软硬件的限制，《极品飞车》还只是DOS下的版本，486DX/66，8MB内存就可以顺畅地运行。画面虽不够真实，但是它的拟真度和音效还是挺不错的，在驾驶时，玩家已经能够进行四个镜头的切换，比赛完后还可以看看六个不同镜头拍摄的精彩回放，这些足以弥补在图像方面的不足了。难能可贵的是游戏已经开始支持网络比赛了，不过，可惜的是，它只支持通过调制解调器的二人面对面比赛。
　　总的来说，这部游戏作为一个经典系列的先驱还是很成功的，这为它后来占据赛车类游戏半壁江山打下了良好的基础。
　　极品飞车特别版　　The Need For Speed：Special Edition
　　在原有游戏的基础上增加了一些新的东西而已，本质还是没有太多的改变。它比原来的那个版本增加了两条赛道，而且在音乐方面有了14首CD音轨。不过，这个版本已经开始支持Win95，在Win95下玩家还可以进行多达8人的联网游戏。
　　极品飞车Ⅱ　　Need For SpeedⅡ
　　到了1997年的4月，从这一代开始，游戏的名称变为“Need For Speed”，同时游戏本身也有了很大的变化，其中最大的一点就是玩家在游戏中可以精细地调节自己的赛车性能，包括扰流器设置、齿轮比、车身颜色和其他的参数，这一点让很多爱车一族兴奋不已。
　　游戏在图像和音效方面也有了不小的进步。
　　极品飞车ⅡSE　　Need For SpeedⅡSE
　　这个二代的“特别版”可以与《极品飞车》的历史意义相提并论。因为从这一代开始起，《极品飞车》系列游戏开始支持3D加速卡了，从此游戏的真实度上了一个新的台阶。
　　游戏中为玩家又提供了两条新的赛道，和四辆新的赛车。另外，游戏还支持赛道的镜像模式，这无形中将赛道的数量增至十六条。
　　极品飞车Ⅲ：闪电追踪　　Need For SpeedⅢ：Hot Pursuit
　　在《极品飞车Ⅲ》中，有八条不同的赛道以及一条隐藏赛道供玩家选择，除了十三款最新的世界级名贵跑车外，还有像El Nino这样的超级跑车。而更叫人兴奋的，就是新加入的“下载新车”功能，随着游戏的推出，各位玩家还可以到《极品飞车》游戏的网站，下载各类新车。
　　《极品飞车Ⅲ》中所加入的“闪电追踪”模式，应该是这一代产品的最大特色。玩家可以选择扮演警察追捕那些超速的车辆，或者扮演逃避警察追捕的超速车辆的驾驶员。除此之外，《极品飞车Ⅲ》之闪电追踪，为了提供比前一代更真实的驾驶感受，诸如飘雪、落叶、下雨等，各种气候的细节变化也真实地呈现在游戏当中，加上配合逼真的音效和支持力回馈摇杆的设计，绝对能让大家耳目一新。
　　极品飞车：孤注一掷　　Need For Speed：High Stake
　　它的辈份问题一直被人们争论不休，有的人认为它是三代的“特别版”，有的人则认为它就是第四代。现在事情终于有了一个了断，《极品飞车：孤注一掷》就是《极品飞车Ⅳ》。
　　这款游戏的最重要的特色是增加了一种新的称为“孤注一掷”的追逐模式。这种模式所增添的游戏风险性使玩家更感紧张刺激。玩家在游戏中进行比赛，失败者将失去自己的汽车，而胜利者可以由此而拿到高额的奖金。这种模式所增加的风险性的确让每一位玩家都兴奋不已。这也使它成为今年最受欢迎的赛车游戏。
