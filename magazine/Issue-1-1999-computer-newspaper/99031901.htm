HPC来了


　　一、HPC(手持电脑)发展概况
　　PDA(Personal Digital Assistant，个人数字助理)是美国Apple公司于1992年1月提出的一个产品概念。Apple公司在1993年正式发布了第一个PDA产品――Newton Message Pad。
　　美国《BYTE》杂志曾为PDA下过这样的定义:PDA是一个面向广大非专业用户的计算和通信设备,它不仅为用户提供可使用的各种通用软件,而且可以使用笔对这些软件进行操作和控制,并且便于携带,适合在各种场合使用。
　　自Apple公司的Newton PDA问世以来,国际上许多厂商相继加入到了PDA产品的开发行列当中,PDA新产品相继不断涌现。
　　自从微软发布Windows CE操作系统后，采用Windows CE 操作系统的PDA已经成为该行业的时尚，同时又被赋予了新的产品概念“手持电脑”(Handheld PC简称HPC)。
　　手持电脑作为一个移动计算环境的工具从1995年在国外开始出现，手持电脑可以看作是电子计事本、PDA的发展和延伸。自从以微软为代表的一些软件开发厂商给手持电脑赋予生命、灵魂（操作系统)之后，手持电脑以一个崭新的面貌出现在世人面前。
　　1996年微软的适用于手持电脑的第一代操作系统Windows CE 1.0版本正式被作为HPC的动力引擎，来帮助使用者更方便地利用和使用HPC。
　　目前基于Windows CE操作系统的HPC厂商主要有：HP、Casio、LG、NEC、Sharp、Compaq、Philips、Hitachi。
　　世界各国的公司虽然推出了众多的HPC产品，但对于某一公司来说，一般只有根据自己原有的技术基础和发展战略开发自己特色的产品。因此导致了到目前为止，各种HPC系统各自有其自身的特征。这样使HPC产生了分化，形成了几类各具特色的分支。从目前的实际情况来看，主要有以下两类不同的HPC产品：
　　1.掌上电脑(Palm PC简称PPC)
　　它的最主要特点是不带键盘，采用笔输入方式，比手持电脑更小巧，携带很方便。典型的掌上电脑有Casio公司的Zoomer和 Apple公司的Newton Message Pad(100、110、120、2000)等。
　　2.手持电脑（Handheld PC简称HPC ）
　　手持电脑的开发商多为传统大型、小型或微型计算机系统的生产者。HPC在一定程度上是便携式计算机的缩微产物。手持电脑的特点是注重它和微型机的兼容性，带有键盘，输入方式有键盘输入和笔输入两种。典型的手持电脑有HP公司的100LX、200LX、320LX、360LX、620LX，IBM公司的ThinkPad710等。
　　二、HPC的特点及应用领域
　　1.HPC的特点及功能：
　　HPC的特点有：体积小、便于携带,集通信功能、个人信息处理和计算功能于一体。计算处理能力在60到360MIPS之间、有8M到32M的内部RAM、屏幕显示分辨率为640×240(或320×240）、支持压控式触摸屏和笔、PCMCIA卡，重量一般都小于500克，这些特点决定了它极其适合于移动使用。从功能上看，HPC集成的应用程序有：微软Pocket Word、Pocket Excel、Pocket PowerPoint以及Internet Explorer和一些个人信息管理的应用程序，并且目前HPC具备红外操作的能力，可用来与其他设备进行信息的交流和处理工作。
　　2.HPC的应用领域：
　　(1)公司管理人员的出差辅助办公。通过内置的FAX/MODEM可以连接上网，进行网上浏览、收发电子邮件。
　　(2)一般业务人员的办公伴侣和业务助理。可以将公司的产品情况，通过HPC和投影仪或普通VGA显示器连接，进行展示和介绍。
　　(3)野外作业人员的信息收集和分析。如：地质、勘探等。可随时通过HPC的无线通讯功能和各处信息台进行联系，并进行信息处理。
　　(4)学校日常教学助理。HPC本身的价格比一般台式机、笔记本电脑要便宜得多，学校可以通过HPC的内建网卡构成一个简单的教学系统。老师可以用来进行作业的安排及评判工作，可极大地提高教、学效率和质量。
　　(5)金融、医疗、保险机构中工作人员、客户之间信息联系。
　　三、HPC的发展趋势
　　由于人们工作场所的不固定以及对信息需求的不断增加，基于Windows CE的HPC的出现，给移动计算和移动通讯提供了非常好的解决措施。
　　随着HPC外设从功能上、集成度、速度、容量上的日益提高，HPC将会发生巨大的变化：处理速度将会接近台式机奔腾速度；存储容量也会越来越大；显示效果会更加柔和、色彩更加丰富；体积和重量会更加适合人们的要求。另外，随着语音识别和手写识别等技术的日益成熟和发展也为HPC的输入技术提供了非常好的解决途径。
　　预计，HPC未来将向以下两方面发展：
　　1.集计算机、通信设备和家用电器于一身
　　就现在的情况来看，HPC基本上是计算机与通信设备相结合的产物。随着技术的进步和价格的降低，HPC将会成为一种新兴的家用电器，并像电视机、录像机和音响那样在家庭中得到普及。
　　2. 逐步走向标准化
　　目前HPC领域正不断地通过竞争和合作，朝着标准化的方向迈进,这种标准化趋势的具体表现是若干HPC产品开发集团的产生。这些集团内部的公司在技术上相互补充，整个集团对他们开发的HPC系统有着比较明确的产品规范和发展策略。现在世界上主要有六家大的HPC开发集团：Apple；NEC、AT＆T；CASIO；Microsoft、Toshiba；Compaq、Intel、IBM；台湾PDA联盟。值得我们注意的是台湾PDA联盟已经开始着手制定有关HPC中文操作系统应用程序接口（API）的标准。
