巧用Flash 32特色功能四则

　　Flash 32是Infonie公司最近推出的一款屏幕抓图软件，除了常规的抓图功能外，它还具有对图像进行特效处理、直接抓取连续图像生成动画等特殊功能。下面我们就来看看如何利用这些功能来完成一些特殊的工作。

　　一、快速制作按钮

　　 制作网页、演示文档时我们都要用到各式按钮。利用Flash 32，可以在制作过程中随时“就地取材”制作按钮，特别方便。
　　例如在PowerPoint中要用到一个“播放”按钮，在页面中找个空白地方输入“播放”两个字，调整好字体、字号，运行Flash 32。软件自动缩成图标出现在任务栏系统托盘区。左键单击Flash 32图标，依次选择“Captures／Region／Rectangle”（即抓取图片／区域／矩形区域），这时鼠标处于选取状态。按住左键选取包括“播放”两字的区域，松开鼠标后，选取区域就出现在弹出的Flash 32主窗口中。先执行“Colours”菜单中的“Coloriation”命令。在“Blue”栏中键入90，将按钮变为淡蓝色。再执行“Effects”菜单中的“Border／3d border”命令，输入台阶高度，按钮就出现了，在PowerPoint中插入即可。

　　二、从文件中“抠”取图标

　　利用Flash 32从文件中抓取位图、图标或光标的功能，我们可以很简单地从应用程序中“抠”取图标。Flash 32可以提取图形的文件主要有EXE、DLL、VBX、OCX等格式 。单击Flash 32图标，然后单击“Captures／Capture an image in a file”，选择欲抠取图标的文件，就出现图2示的对话框。在下面的图形类型中有位图、图标和光标三种选项。选择“图标”（Icons），在“Name”栏中就列出选定文件中图标的数目，右侧是预览区，选定图标后单击“Capture”按钮，选中的图标就被成功“抠”取了。要选择其他文件，单击“Other file”即可同样进行。

　　三、制作演示教程

　　Flash 32可以用来制作 AVI、GIF、FLC、FLI和MPEG格式的动画文件。单击任务条上的Flash 32的图标，选择“Create an animation／AVI”，在出现的对话框中的“Animation target folder”栏指定动画存放文件夹。在“Animation file name”栏中输入文件名，“Capture mode”栏用于指定全屏幕抓取还是抓取特定窗口。最下面的“Repeat capture every_seconds”设定抓图时的时间间隔（单位为秒）。单击“Options”按钮可以设定动画循环播放、保存中间图像等。确认后一个带有红色禁用标志的小摄像机图标出现在任务栏上。单击该图标则红色禁用标志消失开始抓取屏幕图形；再次单击则结束抓图。

　　四、生成带“播放器”的图片

　　要观看图形文件，一般得使用专门的看图软件如ACDSee32等。Flash 32抓取的图形却可以自带一个“播放器”，或者说它本身就是“播放器”――换句话说就是存为一个不需要任何看图工具就能直接运行显示图形的可执行文件。
　　生成以上文件的方法很简单，在保存图形文件时，从“Format”栏中选择“EXE－self－extract”，在窗口标题栏中键入图形文件标题如“自动播放图片”，如图3所示，选择好颜色深度，保存。以后在资源管理器中双击该文件就可以看到图形了。
