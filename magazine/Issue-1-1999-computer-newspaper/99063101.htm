建个内布“因 特 网”


　　目前，网民越来越多,上网已不是什么新鲜事。但由于种种原因，许多中小单位员工大多利用业余时间在家中搞网络活动，单位集体上网的情况并不多。本文向大家介绍一种简单建立“因特网”的方法。这里所提到的“因特网"是指利用各单位已有的局域网，在不追加任何投资的情况下，实现用IE4.0等浏览器相互访问和通信。用平常惯用的Word97或其它主页制作软件做个属于自己的个人主页，其中的操作和在真正的Internet网络上几乎一样,有异曲同工之妙。这样做不仅充分发挥了局域网的效率，进一步实现内部资源共享，也让那些没有上Internet网的同事过过“上网”的瘾。实现步骤很简单，以Windows NT4.0网络系统为例，各用户环境为中文Windows95，只要完成以下几步即可。
　　1.分别在服务器和用户电脑上添加、配置TCP／IP协议
　　首先，在用户电脑的Windows95“控制面板”里，双击“网络”图标以后，添加微软公司的TCP／IP协议，并进行适当的设置。以笔者建立的“因特网”为例，具体操作步骤如下：
　　a.启动Windows95―>双击“我的电脑”―>双击“控制面板”―>双击“网络”图标―>单击“添加”项―>单击“协议”网络组件类型―>再单击“添加”项―>在厂商栏内单击“Microsoft”―>在网络协议栏内单击“TCP／IP”协议―>选择“从磁盘安装”按钮―>插入装有TCP／IP网络协议的软盘或光盘―>在输入条内输入正确的路径后，单击“确定”按钮开始安装。
　　b.TCP／IP协议安装完成以后，便开始设置。单击“TCP／IP协议” ―> “属性”―>单击“IP地址”项，选择“指定IP地址”并设定各用户电脑的IP地址（如：分别为200.200.200.xxx，这里“xxx”为1～255的任意整数值。每位用户的IP地址不能一样，可以提前为各用户分配好各自的地址）。再输入子网掩码，各用户的子网掩码必须统一，例如统一填入“255.255.255.0 ” 即可。单击“DNS设置”项，选择“启用DNS”分别填入你的“主机”（如qlsl）、“域”(如qlsl.lgh.net)、“DNS服务器搜索顺序”（如200.200.200.1）等内容。
　　其次，在Windows NT4.0服务器上添加并配置TCP／IP协议。其步骤同Windows95没有多大区别，在设定服务器的IP地址时最好设为200.200.200.1,这样既便于记忆又体现了服务器在网络中的“老大”地位。
　　2. 安装“服务”网络组件
　　在各用户电脑的Windows95环境下，添加“Personal Web Server "服务,具体操作步骤如下：
　　启动Windows95―>双击“我的电脑”―>双击“控制面板”―>双击“网络”图标―>单击“添加”项―>单击“服务”网络组件类型―>再单击“添加”项―>在厂商栏内单击“Microsoft”―>在网络服务栏内单击“Personal Web Server ”―>选择“从磁盘安装”按钮―>插入有微软网络服务程序的Windows95安装光盘―>输入正确的路径后，单击“确定”按钮，便开始安装Personal Web Server网络服务程序。安装完成后，重新启动Windows95，这时屏幕右下角显示出“个人Web服务器”的图标,“控制面板”中也有它。
　　3.上网互访
　　以上步骤完成以后，连入网络的每位用户就可以像上Internet网那样通过启用IE4.0浏览器或启用网景浏览器，在URL输入域中输入IP地址（例如200.200.200.5）来调用相应微机用户制作好的主页，实现相互访问。
　　此文所提Windows95均为Windows95（USB）正式版，关于升级版的Windows95或Windows98环境用户是否能实现该功能，首先要按上述第二步检查一下你的系统是否具有“Personal Web Server”网络服务项，若没有此项服务，则只能望“网”兴叹的份了。关于此问题笔者希望和不同版本的Windows95用户及广大Windows98用户继续探索。
　　不花钱也能办成的事情，广大Windows NT 用户不妨试一把。欢迎对NT感兴趣的朋友与我进一步探讨，笔者的电子邮箱是 QSDILU@263.NET。(山东　逯广辉)
