专家坐堂

Diamond S90声卡为何不发声？

　　最近有多位读者问：我购买了Diamond Sonic Impact S90 PCI声卡，在WIN98系统中顺利安装了驱动程序，一切看起来都很正常，但是这块声卡就是不能发声。我试过将声卡插在装有WIN95的机器中，能够发声。检查音量控制，没有任何一项设置为静音。检查设备管理器，发现显示所有设备都工作正常。重新安装WIN98系统也不能解决问题。请问这是什么原因？应该怎样解决？

　　答：接到了多位读者提出的基本相同的问题后，笔者认为这不是偶然现象，也不应是软硬件故障造成的。经过多方搜寻资料和仔细分析之后，笔者认为造成这种现象的最有可能的原因是S90声卡在某些情况下存在着资源配置冲突问题，或称兼容性问题，并在WIN98系统下表现出来。在发生问题的这些机器中，多数都使用了AGP显卡，S90声卡被插在紧靠AGP插槽的第一个PCI插槽中，或者第四个PCI插槽中，这两个PCI插槽分别与AGP显卡和USB控制器共享一个中断，这便是最有可能导致产生此类问题的原因。事实上，除了S90外，还有其它少数型号的PCI声卡也可能会出现类似的现象，S90声卡在某些系统配置下还可能频繁引起系统死机，或者出现中断配置老是自动改变等现象。为了解决此问题，朋友们可以采取以下的方法：把S90声卡插入到从AGP插槽一方数起的第三个或者第二个PCI插槽中，然后在BIOS中把“PNP OS Installed”一项设置为“No”，把“Reset Configuration Data”设置为“Yes”，重新启动系统，最后重新安装一遍S90的驱动程序。

为何火球八代硬盘不能工作在Ultra DMA方式下？

　　读者zhang问：我看了第19期电脑报硬件周刊上《你的硬盘支持Ultra DMA吗？》这篇文章，于是我就想测试一下我的硬盘，可是问题就来了，我的硬盘是昆腾火球八代6.4G，主板是微星6163，开机自检时给出的硬盘工作模式是UDMA 4（Ultra DMA/66），可是在Windows 98中的系统属性设备管理器中硬盘属性中的DMA这一项怎么也选不上。我看《电脑报》上说Ultra DMA/66的硬盘可能需要厂商提供的工具软件来设置其工作模式，可我在昆腾的网站上并没有找到有关软件，希望能告诉我在何处能找到此软件或有何办法能解决此问题？

　　答：不少朋友遇到了与这位读者相同的现象，即无法让火球八代工作在Ultra DMA模式下。这是火球八代的问题吗？其实不是，因为IBM和希捷的Ultra DMA/66硬盘也存在着类似的问题。究其原因，发生该问题的主板具有以下特点：芯片组最高只支持UDMA Mode 2（即Ultra DMA/33），其BIOS为98年10月28日以前的Award BIOS。这种主板在使用UDMA Mode 4（即Ultra DMA/66）的硬盘时，Windows 95/98或NT可能无法让硬盘工作在Ultra DMA模式（更确切地说是Ultra DMA/33，因为芯片组最高只能支持这种模式）下。了解了这种情况后，我们马上可以想到的一种解决问题的方法就是升级主板的BIOS。对于那些不能升级BIOS或者没有较新BIOS的主板来说，硬盘厂商为我们提供了一个小工具来切换硬盘的工作模式，可以让原本支持Ultra DMA/66的硬盘只以Ultra DMA/33模式来工作，不过在使用此类工具时应严格按照其说明来操作，否则可能会导致硬盘出现故障。这些小工具的下载地址为：
　　昆腾： http://service.quantum.com/menus/soft_menu.htm http://service.quantum.com/software/fbcrdma.exe
　　IBM： http://www.storage.ibm.com/techsup/hddtech/welcome.htm http://www.storage.ibm.com/techsup/hddtech/ibmata66.zip
　　希捷： http://www.seagate.com/support/disc/drivers/discfile.shtml ftp://ftp.seagate.com/techsuppt/seagate_utils/uata66-d.exe

微星6163的跳线SW2有何作用？

　　读者zuowx问：我使用的主板是微星MS－6163（带声卡），其上有一跳线SW2，据说明书讲：将SW2断开，就可自由设置CPU的外频。但我将SW2断开，加电后CPU的风扇在转，系统却不能自举，显示器没有任何显示，硬盘没有反应，CD－ROM可以弹出和关闭，但是连开机后的“滴”一声都没有。请问这是为何？主板是否可能是“水货”？

　　答：需要说明的是，MS－6163的SW2跳线断开的作用并非是可以自由设置CPU的外频，而是把外频设置成100MHz，这样就相当于已经对你的CPU进行了超频，此时系统无法自举，说明你的CPU或者内存在100MHz下不能正常工作，你可以参照介绍超频方法的文章，使用提高CPU的核心电压等方法来看看能否超频成功，如果不能成功，就只能重新把SW2短接，降低CPU的外频。

NT4与PWIN95的机器为何联网不成功？

　　上海读者盛康弟问：我现在有二台电脑，新的一台装有PWIN98与NT4.0中文版，旧的一台装有PWIN95。我用NE2000兼容网卡及RJ－45接头的线把二台电脑连了起来，PWIN95与PWIN98利用网上邻居通信很成功。但把NT4.0安装为主域控制器，PWIN95设为客户机登录时，输入口令后总是提示：“没有合适的域服务器来证实您的口令。这样，您可能得不到访问某些网络资源的权限”。用网上邻居通信不成功，我也不知道用其他什么方法可以共享资源。我想问：用这种方法是否能组成最简单的数量最少的局域网？如果行的话，具体该如何操作？是否还有其它少花钱又容易的连接方法？

　　答：两台计算机即可组成一个小的局域网，每台机器配置一块网卡，使用专门为两台机器对连（三台及以上机器就要使用集线器了）而制作的双绞线把两台机器连接起来，然后在操作系统中安装好网卡驱动程序，配置好网络协议，便可实现两台机器之间的通信。你在使用PWIN95和PWIN98进行通信时很成功，表明你的硬件安装正确无误。PWIN95与NT4无法连通，可能有以下的一些原因：NT4下网卡驱动程序没有安装正确；双方没有安装相同的网络协议；协议配置不正确；NT4中没有为PWIN95客户机上的用户创建用户名。为此，笔者建议你首先检查NT4下网卡是否工作正常，然后为PWIN95和NT4都安装TCP/IP协议，并配置处于同一网段的IP地址，例如NT4的机器可配置IP地址为“128.0.0.1”，掩码为“255.255.255.0”，PWIN95的机器可配置IP地址为“128.0.0.2”，掩码为“255.255.255.0”，设置共享文件夹及打印机，最后，在NT4的用户管理器中创建新用户，设置适当的权限，并让PWIN95以这个用户名登录到域中（假设都使用缺省的域名Domain），如果PWIN95能够登录成功，就表明配置基本上都正确。关于NT4更详细的使用说明可以参考相关书籍。其他少花钱又容易的连接方法还有通过串口或并口连接，但是这种方式只适合于Windows 9x之间的机器，并且功能有限，传输速度慢，因此笔者并不推荐这种连接方式。

机器频繁死机是何原因？

　　读者魏成问：我的电脑从3月份起就频繁地死机。症状为：屏幕无反应，鼠标也不能移动，键盘无响应，只能按RESET键重新启动。当初我以为是系统问题，或是病毒引起，就重新分区，格式化硬盘，但是问题依旧。到了4月27日凌晨再一次死机后，我按RESET键重新启动，可电脑通过自检后便停下了，不过光标还在闪烁，硬盘和软盘都不能自举。冷启动也一样。后来我就拿到电脑房去修，在排除了病毒感染的前提下，他们给换了一个风扇，故障未排除，把硬盘连接主板的IDE线拔起后，再插入旁边的IDE接口，机器正常启动了。装入PWIN98使用正常。回家使用了几天后，又开始出现死机，不过按RESET键还能启动。请帮帮我。

　　答：从你叙述的情况来看，引起频繁死机的原因可能是电源的风扇坏了，或者是电源老化之后工作不稳定，在这种情况下，当开机时间略微长一点，电源输出电压出现的波动就可能超过主板或其他设备所能接受的范围，从而导致系统彻底死锁。你可以更换一个好的电源。至于你在4月27日遇到的问题，更有可能是因为C盘的启动文件出现故障，例如机器上安装有PWIN98或PWIN97，如果启动机器到MS－DOS 6.22，此后便无法从硬盘启动了。如果你没有调整CMOS中的启动顺序，也就有可能无法从软盘启动。更换IDE接口后，C盘不存在了，所以能从软盘启动。笔者建议你今后在使用PWIN98时不要启动到老的DOS。

喷墨打印机走纸为何总向一边歪？

　　读者黄晓鹏问：我买了一台爱普生440喷墨打印机，用的打印纸是普通的A4复印纸，当打印第一张时复印纸总是向边一头卷，而不是均匀地向前，这时，我立刻把纸拔出，重新再放好，这才正常，请问这是什么原因呢？

　　答：首先你应清楚很多种喷墨打印机进纸时都以右边作为起始位置，即放纸时应靠着送纸匣的右边，并把左边可滑动的挡板移到纸的左边并轻轻靠拢，这与很多针式打印机和激光打印机正好相反。如果把纸靠在了左边，进纸时就会出现向一边歪的情况，此时最好不要强行拔纸，而应中止打印，以避免对打印机造成损害。如果你的放纸方法正确的话，那么表明这台爱普生440喷墨打印机的进纸机构存在着问题，你可以找销售商进行维修或调换。
