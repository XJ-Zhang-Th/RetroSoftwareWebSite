超越你的创意空间――Photo Tools 3.0新增功能介绍


　　大概玩过PhotoShop的朋友，没有不知道Photo Tools大名的吧。Photo Tools是众多的PhotoShop Plugins中功能最强、效果最好的一个，其中专门制作文字的Photo Text部件，至今还没有一个类似的插件能与之抗衡。
　　随着PhotoShop5.0的出现，EXTENSIS公司也及时推出了全新的Photo Tools 3.0，Photo Tools 3.0的功能大大超越了2.0。除了保留原有的Shadows、Bevels、Glow等功能外，Photo ToolS3.0到底还增强了些什么功能呢，让我一一道来。
    一、PhotoAnimator
　　PhotoTools3.0中最具革命性的工具当属PhotoAnimator，一看名字，大家就知道这是个动画GIF制作工具(见图1)。它提供了各种大量的动画特效库，摆脱了动画GIF必须逐帧制作的苦恼，你只要选择一种动画特效和转变的方式，其余的一切都由PhotoAnimator去处理，特别值得一提的是PhotoAnimator使用了一种特殊的压缩技术。能够最大程度的减少动画GIF文件的容量，特别适合网页制作。

　　
　　　　　　　　　　　　　　　图1

    二、PhotoTexture
　　PhotoTexture让你的想象力能发挥到极限，它不同于其他的材质生成器，能通过对其施加各种特效，随心所欲地产生各种变化万千的自然界和人工合成的贴图，并且这些贴图能够进行无缝的拼贴，这些贴图特别适合作为网页的背景(见图2)。

　　
　　　　　图2

    三、PhotoButton
　　PhotoButton是目前最棒的按钮制作工具，除了原有的功能外，按钮的导角类型、按钮的形状、按钮表面的贴图、贴图类型等(见图3)，用户都可以按照自己的意愿进行定义，真是太方便了。PhotoButton还允许让用户设定按钮的个数、按钮的排列方式、按钮之间的间隔距离，能够一口气对按钮进行批量制作，简直是一个“按钮机器”。

　　

　　　　图3

    四、PhotoGroove
　　PhotoGroove是Photo Tools新增的部件之一，它通过Bevel Shape控制曲线，能够对任意形状、任意大小的图像和物体制作导角和凹槽特效，特别适合制作图像边框、导角文字、按钮(见图4)。甚至还能改变图像的表面，产生各种材质特效(见图5)，如橡胶材质、塑料材质、金属材质和铬合金材质等，可以和PhotoButton配合使用。

　　　　　　　　　
　　　　图4　　　　　　　　　　　　　　　　　　　　　　　　　图5

    五、PhotoCastShadow
　　PhotoCastShadow除了能产生常见的drop和perspective倒影外，还能够产生各种类型的倒影。甚至还可以通过设定光源的个数，使物体在不同方向上产生多个不同类型的倒影(见图6)。这样，制作足球场上的运动员倒影就不是什么难事了！

　　
　　　　　　　　　　图6

    六、PhotoGow
　　想用PhotoShop制作霓红灯效果吗？用PhotoGlow绝对没错。PhotoGlow能够为选定的图像施加光芒特效(见图7)，用户可以自由设定光芒的颜色、大小和形状，也可以从含有各种效果的样本库中选择一种光芒特效。

　　　　
　　　　　　　　　　　图7

    七、PhotoBevel
　　PhotoBevel是PhotoTools中的一个共享组件,它能在图像或文字的的任何一个部分上产生Inner或Outer导角效果。你可以随意调整导角的宽度、尖锐度、高亮/阴影对比度、高亮/阴影平衡度和光源的照射方向等参数。
    八、PhotoEmboss
　　PhotoEmboss与PhotoShop自带的Emboss特效相比，具有后者不可比拟的优越性。PhotoEmboss提供了一个良好的用户界面，可以所见即所得地产生具有浮雕效果的文字和图像，还能够调整高光处和阴影处的颜色，和PhotoBevel一样也可以设定Emboss的强度、尖锐度、高亮/阴影对比度、高亮/阴影平衡度和光源的照射方向等参数，用它制作的浮雕效果(见图8)好得让你不相信自己的眼睛。

　　
　　　　　　　　　　　　　　图8

    九、PhotoBar
　　PhotoTools增加了一个用户自定义工具条PhotoBar(见图9)，利用它，你只需要单击鼠标就能够快速地执行许多隐藏在PhotoShop下拉菜单中的命令，这对于PhotoShop的初学者是一个福音，可以不必为了寻找一个命令而频繁用鼠标在菜单中搜索。PhotoBar还提供了几十个色彩丰富的按钮，可以让PhotoShop一成不变的黑白界面焕然一新。

　　
　　　　图9

    十、PhotoText
　　PhotoText是PhotoTools中最棒的文字输入和处理工具，不过这次它并没有加入PhotoTools的行列，而是成了一个免费的软件，大家可以自由地从网上进行下载。告诉大家一个小秘密，在PhotoText的字体选择框中可以正确地显示出中文字体的名字，这样，大家就不必为在PhotoShop5.0中无法输入汉字而苦恼了。需要PhotoText的朋友，可以到我的站点http://3donline.yeah.net/phototext.zip上下载。
