绚丽屏保任我做


屏幕保护程序

　　显示器就像人一样，如果长久地保持一个姿态，即总是显示一个相同的画面，就有可能因为某些屏幕区域长久地高亮度聚焦，造成该区域屏幕的荧光衰减，从而影响整体显示效果。而在Windows的系统中就提供了屏幕保护程序（简称屏保），当一定时间不使用电脑时，就可以设置Windows启动屏幕保护程序，自动地连续变换屏幕画面，使荧光屏幕能均匀地聚焦，从而保护屏幕。
　　目前，屏幕保护程序除了Windows系统提供的外，还有大量的屏幕保护程序，这些屏幕保护程序充斥在互连网上，显示的画面展示了各种各样的题材，屏保也就成了一个宣传的手段，如《电脑报》第15期15版介绍的金鱼饲养屏幕保护程序。而现在使用一些屏幕保护制作软件，初学者也可以很简单地制作出自己喜爱的屏幕保护程序。
　　制作屏幕保护程序的软件很多，有些非常专业，可以制作动画屏保，当然操作上也比较复杂（如Anmated Screen 3.10），下载地址为http://www.sunv.com/newhua/down/ascrstp.zip。而有些软件就非常简单易用，只需选择几个图片就可以了（如《电脑报》22期15版介绍的Screen Saver Toolbox 2.1)，下载地址为 http://www.sunv.com/newhua/down。这里，给大家介绍一个使用非常简单，但又可以制作出专业级效果的屏幕保护制作工具――Stardust Screen Saver Toolkit TE（简称SST）。SST可以支持多种格式的图像文件，支持AVI动画格式，可以添加背景音乐、标注文字、制作用于发表的屏幕保护安装程序等。用SST制作屏保是简单的向导式操作，非常方便。

安装与启动

　　1.下载软件：SST是一个共享软件，可以到开发该软件的Stardust公司主页http//www.sdust.com/support下载，目前最新版本为2.1版，下载文件大小为2.9MB。也可以到http://www.sunv.com/newhua/down/sst21te.exe下载。
　　2.运行安装文件：从网上下载的SST一般是一个可执行的安装文件，名称为sst21te.exe，在Windows系统中双击安装程序图标（图1），运行安装程序开始安装。

　　
　　　图1

　　3.设置安装路径：SST的安装很简单，安装程序先进入欢迎窗口画面，用鼠标点击右下角的“Next”按钮，在弹出的下一个授权协议说明窗口中同样点击“Next”按钮，然后在接下来窗口中设置程序的安装路径（图2），你可以将程序安装到非默认路径的指定目录中。

　　
　　　　　　　　　　　　　图2

　　4.完成安装：接下来的几步，就是拷贝文件、设置程序启动的快捷方式，都可以点击“Next”按钮继续，不用特别设置。在最后的窗口中点击“Finish”按钮结束安装。

准备工作

　　使用SST制作屏幕保护程序，必须要准备好一些“生产原材料”，如图片、AVI动画、音乐等。SST在安装时，还在安装分区的根目录下增添“My content”和“My Screen Savers”两个文件夹，用于存放图片、影像和声音文件以及“半成品”文件的。
　　1.主题思想：在准备材料的时候，要注意制作什么样的屏保主题，例如动植物、风景欣赏、收藏鉴赏、流行潮、或者是自己的摄影美术作品等，就看你怎样发挥想象力了。注意还要设计好相关的WAV或MIDE音乐。如果是制作商业宣传的屏保，就更需要精心策划。本文就示例一个长江源头风景的环保主题屏保。
　　2.收集材料：确定了主题之后，收集材料的工作就非常重要。我们要收集的材料主要是图片和音乐。现在收集材料大致有五条途径：一是从因特网上下载。二是目前出版了许多光盘图片素材，有些经过授权也是可以使用的。三是将图片资料通过电脑扫描仪扫描获得。四是自己用照相机、画笔创作。五就是用电脑绘图软件剪贴制作。音乐也可以用类似的方法获得。
　　3.整理材料：材料收集到了，应作适当的整理，如修饰图片（如用PhotoShop，初学者可以用画笔、金山画王等），转换图片格式（可以用SST附带的“Image Converter”工具，建议用ACDSee32），标记图片（对图片的内容作好记录）。需要注意的是，图片的幅面应大一些（建议600×400像素以上）、质量高一些，最好都转换成高质量的JPEG图像。

发表屏保程序

　上面的操作步骤，只是将屏保初步设计好了，程序回到图5所示的界面，可以选择该界面的第二个选项修改刚才设计保存的屏保。但如果要实际使用该屏保，还要生成屏保文件或屏保安装文件。
　　点击界面的第三项“Build an Existing Screen Saver project”，打开保存的半成品屏保文件，进入图14所示的界面。在下面的文本窗口中设置屏保程序保存的路径，点击按钮[Start]，即可生成Windows可以使用的屏保文件，将该文件直接拷贝到Windows所在目录的System子目录下，然后在显示属性中设置成当前的屏保即可。

　　
　　　　　　　　　　　　　图14

　　勾选图14所示界面上部的“Internet”选框，可以生成一个单一的屏保安装程序、运行安装程序，即可以自动将你设计的屏保安装到Windows系统中，并激活使用。这就非常便于将你的屏保程序通过因特网提供给他人使用。选择其他项目，也可以创建用软盘、光盘安装的屏保程序。
