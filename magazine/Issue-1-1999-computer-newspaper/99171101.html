装扮Windows

　　大家对微软的Win95不说是了如指掌，也算是混了个脸熟，无非是蓝天、白云、窗口、菜单等。而用过苹果机的朋友一定忘不了那华丽的操作界面、与众不同的操作风格，然而MAC苹果机难求，下面我们就给PC机的Win95动动手术，将之打扮得像苹果的MAC电脑。
　　第一步，换掉开机时的蓝天白云、桌面墙纸、启动时的声音。
　　换开、关机画面非常简单，以前《电脑报》也有介绍，改变桌面墙纸及启动声音也都是Win95的常规操作，这里就不多说了，有兴趣的朋友可以到网站http:∥workinweb.yeah.net论坛上讨论。以下所提到的软件也可以从该网站下载。
　　第二步，更换“开始”按钮。
　　首先下载并安装Tclock Ver 2.1.2软件，它可以改变Win9X的“开始”按钮及时间显示，运行Tclock，任务栏右侧的时间变得既有年月日也有星期显示，在时间上右击鼠标，在弹出的菜单中选择“Tclock Properties”项，然后选择“Taskbar”标签，确定将“Customize Start Button”及“Customize Start Menu”两个复选框选中，然后在“Icon”栏中输入“开始”按钮图片的地址，也可以单击带有三个点的按钮浏览（图标可以是任意的BMP、ICO图片，也可以到网站http:∥workinwebyeahnet上下载MAC的开始按钮标志）。在“Caption”栏中填入希望显示在“开始”按钮上的文字，默认是“Start”，也可以将之置空。如果选中“Hide Start Button”复选框，则不显示开始按钮，但应将“Start Menu From Clock”复选框选中，这样就能通过单击任务栏上的时间来显示开始菜单，最后单击确定。这样就将“开始”按钮偷梁换柱成了MAC似的“开始”。Tclock还有许多选项，如定制“开始”按钮的文字字体、大小、粗细及显示时间的格式等，这些就不一一讲解了。
　　第三步，改变开始菜单外观。
　　先下载并安装NextSTART 1.0软件，运行NextSTART后，会在系统状态条中显示一个显示器样的小图标，右击该图标，选择“Preferences”项弹出“NextSTART”属性设置窗口（也可双击图标弹出窗口），该窗口共有六个选项标签，自左至右分别是菜单标题设置、按钮设置、菜单设置、热点区域设置、通用设置及关于NextSTART六项。标题选项标签可以设置菜单标题栏高度、文字对齐、标题栏字体、标题栏颜色及标题栏是否呈现3D效果等；按钮选项标签则可以定义按钮的宽度及高度、按钮颜色、文字对齐等；菜单选项标签则调整菜单的总体外观及属性；热点区域选项标签可以设定在哪一区域单击鼠标会弹出菜单项；通用选项标签可以选择装载不同的主题，即像Win98的桌面主题一样可以随时更换。
　　第四步，为窗口的边框、标题栏、最大化、最小化、关闭按钮做美容。
　　下载并安装WindowsBlinds 0.40软件，运行程序同样会在系统状态条中出现一个小图标，双击该图标弹出系统设置对话框，第一个“Personality”选项标签中列出了四种窗口效果，单击最后一种“MacPC”效果(也可以下载效果包，单击“a new personallity from a zip”按钮直接从压缩包中安装，安装完毕按“Refresh List”钮刷新)；“Settings”选项标签可以设置一些窗口的外观，包括窗口的底纹、文字对齐等；下一个“Backgrounds”选项标签可以自定义对话框的底纹、浏览器的底纹等；有必要提一下“Exclusions”选项标签，它允许用户指定可改变窗口外观的应用程序，例如将PhotoShop排除在外，不改变它的窗口外观。最后单击“Apply Now”按钮使改动生效，然后打开“我的电脑”看看发生了什么。
　　第五步，改变浏览器的容貌。
　　这一步就是将IE改头换面。下载并安装Neoplanet20软件，该工具虽使用IE的内核，但外观却“美丽动人”。她的操作与IE基本一样，用惯IE的人完全可以轻松上手。最体贴的是“她”可以从网上下载组件而改变“容貌”，真是帅呆了。
(浙江　Hotsky)
