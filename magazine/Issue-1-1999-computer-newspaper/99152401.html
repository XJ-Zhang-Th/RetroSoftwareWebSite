重新抬头的K6-2

　　1998年底到1999年初，Intel低价销售赛扬300A的策略，曾使得K6－2一度在市场上销声匿迹。但赛扬300A虽然很便宜，可对旧机升级的朋友来说，采用K6－2/266由266MHz超至350MHz仍是一个性价比极高的选择方案。
　　MVP3主板＋K6－2/266和一块名牌BX主板的价格相似，而省下买赛扬的钱便可再买条64M SDRAM，机箱也不用换，省下了银子而且性能也并不比赛扬300A差多少，以后还可升级到K6－Ⅲ。现在的赛扬300A越来越难超，相反我发现越新的K6－2越好超。最近从一些网友处得知他们的K6－2/266不加电压或只加到24V便可轻松超到400MHz，还有一位网友在超到350MHz后忘了加散热片，但工作好长时间仍无异样，可见AMD的生产工艺的确有了极大的提高！再加上一些老主板（如T2P4）在广大DIYer的研究下也能支持K6－2了，极高的升级性价比使K6－2重新抬头。笔者根据平时收集的资料加上自己的心得，整理出下面的文章，希望对你能有些帮助。
　　1.现在由于制作工艺的不断改善，K6－2变得更加好超频了。K6－2/350绝大部分都能超到400MHz～450MHz，但一般来说，决不会高于450MHz这个界线，到了450MHz后，无论是加电压，提高散热性，换内存条都无济于事。因为目前K6－2的超频极限就在450MHz左右，如同Intel的极限在600MHz左右一样，这可能是现在025微米制作工艺的极限，600MHz以上可能就要用018微米的工艺了。1999年生产的350MHz的K6－2，在不加电压的情况下，超到400MHz和433MHz一点都不成问题！
　　2.由于AMD并不像Intel那样采取将CPU锁频的做法，因此频率的选择相当有弹性，倍频从×2到×55，而新版本的K6－2(CXT版本)更支援了×6倍频，它的做法是将×2倍频的线路重新定义成×6倍频，就像过去将×15倍频当×35倍频用一样的道理。现在市面上的K6－2几乎都是新的CXT版本，因此都有×6倍频，即使是66MHz外频的K6－2也是一样的。
　　3.最近有小道消息称，现在市场上编号为26351的K6－2用的是K6－Ⅲ的核心，其实就是指CXT版本的K6－2。在德国一个3DNow!网站（www.3dnow.net)，有一个K6－2的优化程序SetK6，它的用途在于控制CPU内部的Write Allocation和Write Combining(只针对K6－2的CXT版本)这两项功能的打开或关闭，可将K6－2的性能提高5％。一般情况下，Write Allocation是关闭的，如果你主板的BIOS能够正常辨识K6－2，那么Write Allocation会自动开启。如果你的主板无法正常显示K6－2（如改造后的T2P4），你便需要通过启动Write Allocation来提高性能。而Write Combining则是新版的K6－2/CXT才有的功能，通过该程序开启它后，可获得额外的性能提高。详细的介绍及使用方法可参考它的说明文件。
　　4.有些K6－2无法稳定工作在100MHz外频下，便被AMD标成66MHz外频（就是所谓的OEM的K6－2），在市面上以较低的价格出售。有的网友买回家之后发现，在100MHz外频下工作十分稳定。这是因为按照CPU的标准规格，双电压的CPU I/O电压应该是3.3V(由K6－2的铝壳上可以看出)，但是市面上支持双电压的主板一般将I/O电压设定为3.5V，而某些主板，允许你将I/O电压提高到3.6V或更高的3.8V/4.0V，因此66MHz外频的K6－2也可跑100MHz外频，且非常稳定。但一旦将CPU的外频提高到112MHz或是更高时，CPU就开始不胜负荷，所以66MHz外频跟100MHz外频还是有差别的，不然为何要便宜几十大洋呢？
　　5.有网友问我，K6－2的电压可调多高？因为超频公式中，提高电压可提高超频成功几率这一条，在K6－2的身上得到了淋漓尽致的发挥，适当的跳高电压可使266MHz的K6－2超到400MHz。但从安全考虑（一块CPU至少得让它干上一年吧），如果要调高电压，建议不要超过2.5V，若还是超不到你希望的频率，可考虑换块CPU。因为现在有太多的K6－2/266可以不加电压或在2.4V下稳稳超到400MHz！我的K6－2/266买得早，要2.8V才能在400MHz下打游戏，现在我让它跑350MHz，电压降到2.0V，这样可减少大量的发热！2.4V的电压应该是安全的，因为K6－Ⅲ就工作在2.4V下！
　　6.由于目前K6－2良好的超频性，266MHz最少可超到333MHz，加上又没有锁倍频，立刻被Remark者钻了空子，AMD发现此问题后虽有了一些措施，但Remark者也有各种相应的对策。盛传市面上的K6－2/300都是用266MHz Remark而来的，于是到处都是打假经验，例如:激光刻痕的深浅，背面的格子，但这都非绝对的公理，只能说是有可能。在选择方面，我发现越新出品的K6－2似乎越好超，我了解有好几块左上角编号为：7461913；左下是26050的K6－2/266在2.4V都稳定地工作在400MHz下！编号为26351的也不错！还有就是被商家Remark当300MHz来卖的K6－2/266的超频能力都极好，有一些不加电压也能上400MHz！
　　在辨别Remark方面，我至今没有找到万无一失的好方法，我觉得最好是找你熟悉的经销商购买，不要光看商家的牌子。不过买块能超400MHz的266MHz，即使是被Remark成300MHz的，也没有太大的损失！而上面我提到，被商家Remark当K6－2/300来卖的K6－2/266的超频能力都极好。
　　7.最近从一位西安网友处得知，用主板(如微星MS－5184)带的DMI Brower查看硬件信息,似乎可以辨认CPU是否被Remark,以下是他的真实经历：他买了一块散装的K6－2/300，买时还是带着放大镜去的，用了各种识别真假K6－2的方法,先摸铝片,是涩的,看来好像没有打磨铝片,右上角刻着300MHz(不太清楚),左下角是金26351,右下角是金N,铝片做得很工整,周边切得很齐,不像是后来车出来的伪造铝片。买回后直接上350MHz(100×35),在350MHz下装Win98和一批软件、游戏,大概用了4个小时，很稳,然后跳400MHz(100×4),不加电压直接上,进Win98没一点问题,跑了一个小时左右,没问题,就是比350MHz时稍热一些(散热片是14W的大风扇,又加了硅胶)。这一切都让他以为这是真的300MHz K6－2。但运行DMI Brower后,告诉我们这还是个266MHz的K6－2。(信息如下)
　　＊＊＊＊＊＊PROCESSOR INFORMATION(TYPE4)＊＊＊＊＊＊
　　Length:1ah
　　Handle:4h
　　Socket Designation:Socket 7
　　Processor Type:(03h)Central Processor
　　Processor Family:(19h)K5 Family
　　Processor Manufacturer:AMD
　　Processor ID:0000058ch 08021bfh
　　Processor Version:AMD－K6
　　Voltage:(02h)33V Supportedv
　　External Clock:(0064h)100MHz
　　Max Speed:(010ah)266MHz
　　Current Speed:(015eh)350MHz
　　Status:(41h)CPU Socket Populated,CPU Enabled
　　Processor Upgrade:(04h)ZIF Socket
　　＊＊＊＊＊＊MEMORY CONTROLLER INFORMATION(TYPE5)＊＊＊＊＊＊
　　不过，这方法我没试过，所以不能在此下结论，但如果真是这么准的话，是否也算是检测AMD的CPU的最科学的方法呢（Intel已有了，看来AMD现在也有了，该轮到奸商头疼了！）？是否有用，大家不妨试试。
(上海　沈　吟)
