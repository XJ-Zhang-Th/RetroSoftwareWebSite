　　问：为什么我的ICQ一连接，就自动连接到ICQ主页上，很讨厌,请问如何解决？
　　答：简单设置一下就可以解决这个问题，方法是：ICQ－>Preferences－>connection，取消“Launch Default Web Browser When Connection is detect。”选项就可以了。
　　问：如何快速下载软件？
　　答：这是一个很大的课题啊，也不是老牛三言两语所能讲清楚的。一般来说，你到你ISP的访问速度应该是最快的。所以要快速下载文件，你可以先到你ISP的主页上查看是否有你需要的文件，有则直接下载之；否则，请到国内各大软件下载网站查找。总的一个原则是能在国内骨干网上下载的，决不去国外，能在ISP站点下载的，决不跑其他地方。还有，一般169站点的下载速度比163站点快；如果实在无法用上面的方法解决，还有一种称之为“小挪移”的软件下载服务，由服务器到ISP服务器的下载速度，应该比你直接下载要强不知道多少倍！而且服务器后台传输的时候，你还可以断线，请到这个地址：http://www.taurus.com.cn查看详细介绍。
　　问：最近不知何故，当我的猫正确连接之后，有时浏览器却无法打开任何页面？
　　答：如果排出你机器设置错误的可能，你默认DNS服务器出问题的可能较大。可以咨询一下你的ISP或者换个DNS服务器设置。
　　问：哪里有可以记录下载量的软件？
　　答：一些常见的下载软件均能够记录下载量：如NetAnts和Go!Zilla。新出的JetCar也不错。
　　问：每次用离线浏览器Offline Explorer下载网站后每一个文件都会有一个扩展名为 primary 的文件。请问怎样可以不产生这种文件？
　　答：没有办法。用Win9X的查找功能先把这些primary文件全找出来，然后全选，一起删除之，这可能是最方便的方法了。
