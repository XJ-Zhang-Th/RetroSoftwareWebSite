值得推荐的十大基本工具软件

　　要把电脑用起来，就要必须精心地选择和学习各种不同的软件。首先，必须先给电脑安装操作系统，操作系统就是电脑总司令，任何电脑的功能部件、软件都必须在它的指挥下工作。
　　操作系统软件以外，还有教育软件、游戏软件、专业应用软件。而辅助计算机工作的软件一般就统称为工具软件。工具软件多如繁星，虽然干不了大事，但就像日常生活中的锅碗瓢盆一样重要，本文将给大家推荐十款极其优秀的基本工具软件，它们涵盖了软件使用的各个方面，使用它们，会让你使用计算机事半功倍。

　　● Windows Commander 3.5 ―― 文件管理

　　 操作系统本身提供管理文件的功能，Win98的资源管理器都可以进行文件查看、复制、删除等操作,然而Win98的这些功能用起来还很不方便。Windows Command 3.5功能很多，对于文件的查找、复制、移动等都相当方便，有很强的文件内容查看能力，可以直接阅读十六进制、二进制和纯文本格式的文件，还支持一些网络功能。

　　● WinZip 7.0――压缩工具

　　压缩工具是将文件数据进行一种重新的编码排列（这个过程称为压缩），使之更少的占用磁盘空间。通俗地说就是一盘磁带录制普通两盘磁带的歌。WinZip可以将文件压缩为流行的ZIP压缩格式，或者将ZIP等多种格式的压缩文件解压还原为原来的文件格式。WinZip可以用向导方式来引导用户使用，能和操作系统很好地配合进行压缩文件的管理和进行文件的压缩、分割和解压等操作，也具有配合网络应用等高级的功能。WinZip是众多压缩软件中的老大，长期居于软件下载榜前几位。是初学者踏入Internet网的必备软件。

　　●AcdSee 2.3――看图工具

　　看图工具是属于文件工具中的查看类，Win98中提供了多种工具来查看文件内容（一般称为打开文件），如笔记本打开通用字符文件（ASCII码）、写字板打开简单的格式文本、媒体播放器打开数字编码声音、影像文件，但我们还是需要AcdSee这个高性能的看图工具。为适应不同的应用场合，图像有很多编码储存的方法（称为图像格式），而AcdSee 2.3可以查看二十多种图像格式文件，并且打开图像文件的速度非常快 ，并且还有图像格式转换、图像文件管理等功能。

　　● 金山词霸Ⅲ――翻译工具

　　《金山词霸Ⅲ》是一个在屏幕上用鼠标指词翻译的工具，它包含了英汉、汉语词典，具有即指即译，即时发音的功能，其它的如东方快车、网际金典、朗道英汉词典等也是类似的工具软件。这些工具可以帮助我们更好地使用英文软件和资料。

　　● 南极星2.0――转换工具

　　由于中国大陆与中国台湾、香港对汉字在电脑内的编码不同（称为汉字内码不同），因此像用Big5编码的汉字资料就不能在我们平常用的电脑系统中正确显示出来，而南极星2.0就是一个将不同编码汉字正确显示出来的工具软件。南极星2.0能在屏幕上将不同编码的汉字正确地显示出来，也能够正确显示日文、韩文，还提供汉字内码转换、不同内码汉字的输入、日文的输入等多种功能。

　　● WinAmp 2.05――MP3音乐播放工具

　　MP3是一种在电脑中存储声音的文件格式，由于有高质量音质、占用存储空间少的优点，因此，有很多MP3文件格式的音乐流传。而WinAmp就是相应的解压缩MP3声音文件再进行播放的软件，WinAmp将MP3格式的音乐文件还原并播放出来。WinAmp能快速、高质地对MP3文件进行解码、播放，可精确地调整音乐的频率、音质，能按指定的音乐曲目顺序播放以及具有混音、循环播放等等功能。

　　● 超级解霸 5.0――VCD播放工具

　　VCD播放工具就是用电脑来实现VCD机功能的软件，而《超级解霸》不但提供了大多数VCD播放机的功能――如环绕立体声、卡拉OK、亮度调节等，其独有的防读死功能和预读功能使这个国产软件特别适合中国国情。而且可以对VCD中的声音、图像进行截取、录制等工作。XingPlay等也是很好的VCD播放工具。

　　● 网络蚂蚁 1.0――网络下载工具

　　在国际互连网上，有数不清的免费软件和资料，我们可以将这些软件资料通过网络复制到我们自己的磁盘上保存和使用（称为下载），网络蚂蚁是下载工具，可以将要下载的文件分成几个部分来同时下载（多点连接下载），也可以分几次下载一个文件（断点续传），这样可以节约许多网络费用。

　　● McAfee VirusScan 4.02――杀毒工具

　　电脑病毒是能自我隐藏、自我传播复制、暗中进行破坏的程序。杀毒工具就是查找，清除这些程序的程序。VirusScan是国际知名的杀毒程序，能随时监测病毒程序的活动，每个月都提供最新的杀病毒数据，更关键的是可以免费试用（即可以通过国际互连网下载该程序）。当然也可以选用国产的商业杀毒工具如KV300，瑞星，Kill98等等。

　　● EditPlus――文本编辑工具

　　初级电脑用户用Win98自带的NotePad可以完成一般文本编辑工作，不过使用Eeitplus你会觉得更方便。首先,它采用了一个开放的开发方式，允许使用者开发相关的插件和模板，在它的主页上很多使用者提供了插件和模板。其次，它实现了同浏览器的无缝连接，它调用IE3以上版本的核心，可以马上把EditPlus变成一个小型的浏览器。监视剪贴板的功能也非常好用。当然，该软件由一个韩国人开发，其双字节的支持自然是非常优秀的。
　　以上介绍的十款基础工具软件对一个想提高自己电脑水平的初学者来说是非常值得推荐的，如果你能熟练的使用上述十款软件，我们相信你的电脑水平达到中级水平了，我们的Step by step栏目将对上述十个软件逐个介绍。
