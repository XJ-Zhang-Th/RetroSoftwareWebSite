登录注册大野狼

　　最近在网上发现一只大野狼，真是狼吞虎咽，大吃搜索引擎登录注册。感觉真是爽!
　　SubmitWolf PRO 是一个专业的Web 页面自动注册工具，在它的数据库中包含了超过1200 个搜索站点。使用这个软件可以完全自动化地将你的站点注册到1200多个搜索站点中，从而增加你站点的知名度。下面介绍它的使用。
　　1. 创建一个新的Project
　　在第一次使用时，你需要创建一个新的Project，或者打开一个以前使用过的注册记录修改后重新提交。只有打开或创建一个新的Project，才能将你的站点的详细信息和URL输入，并提交你的站点。
　　2. 输入详细信息
　　选择Details标签页。现在你可将你的个人站点或是你公司网址的详细信息输入SubmitWolf 的对话框中。在这里你可以将你认为重要的和有用的信息输入。
　　首先是URL里要填入你的网址，在后面的空里填上你所在国家的代码。中国的代码是 cn。
　　其次你需要输入Username和Password 。这是用来修改已经提交到搜索站点中的信息。
　　URL Title 你可以为你的站点起一个好听的名字。
　　Keywords 你要为你的站点选择相应的关键词。并在每个词之间用“，"隔开。这里要用英文。
　　Description 大多数的搜索站点允许你提交一个200到250字的站点描述。
　　Selecting Site Types 有一部分搜索引擎是按站点的类型检索的，因此需要对你站点的类型做一个选择。
　　Language 用英文输入你站点的语言类型。
　　Contact Details 标签页中输入的是联系方法，因此没必要都填（如图）。
　　3. 选择搜索引擎Engines
　　如图，选择搜索引擎Engines页你能看到 SubmitWolf 的搜索站点库，其中包含了超过1200个搜索站点。如果你没注册，那么你只能提交到10个最大的搜索站点，其他的则不能使用。SubmitWolf公司会经常更新他们的这个搜索站点库，如果你是高级用户也可以点击“S”键，这里是自动提交搜索站点的源程序，你可以修改源程序使效率更高，提交时间更短。 　　否则你选择Select All然后在点击上方的Submit按钮就可以了。接下来就是SubmitWolf的事了,你要做的就是等待结果。
　　4. SubmitWolf的获得
　　这个软件你可以从http://www.submitwolf.com上下载得到。我的主页上（http://taoboy.163.net）有更详细的介绍。我的电子邮件是taoboy@88998.com，希望大家跟我交流。
