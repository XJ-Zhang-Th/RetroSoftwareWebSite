初级班（第 十 期）结业考试模拟题


　　编者按：电脑报函授学校初级班（第十期）结业考试定于3月中旬进行。结业考试的试题我们将在近期内寄出，请学员们注意查收。本期模拟试题的题型、难度与结业试题相近(参考答案详见下期)。中级班的模拟试题在本报第九期本版登出。
　　一、填空题（每1小题2分，共20分）
　　1． 十进制数316表示成二进制数是______。
　　2． 在WPS中，打印命令使用的功能键是_____。
　　3． 代数式a2＋abc＋(3x)2用BASIC语言写出的表达式是___________。
　　4． 在FoxBASE中，系统默认的数据库文件扩展名为____，命令（程序)文件扩展名为_____，索引文件的扩展名为____。
　　5． 存储器管理的三个功能是____、____和____。
　　6． 中央处理器（CPU）的四个主要功能是____、____、____和____。
　　7． 汉字操作系统一般由____、____、____和____等组成。
　　8． Windows的窗口可分为____和____两类。
　　9． 计算机内的数据类型有____、____、____和____四种。
　　10．在DOS系统中，建立一个新子目录的命令是____。
　　二、选择题（每1小题2分，共20分）
　　1．一个完整的计算机系统应包括____。
　　A 主机、键盘和显示器 　B 计算机及其外部设备
　　C 系统硬件和系统软件 　D 硬件系统和软件系统
　　2． 微型计算机的外部设备包括____、____和____。
　　A CPU 　　B 输入设备　C 时钟部件
　　D 显示器　E 键盘　　　F 打印机
　　3． 在微型计算机的汉字系统中，一个汉字的内码占____个字节。
　　A 1　B 2
　　C 3　D 4
　　4． 软盘加上写保护后，对它可以进行的操作是____。
　　A 只能读盘，不能写盘　B 既可读盘又可写盘
　　C 只能写盘，不能读盘　D 不能读盘，也不能写盘
　　5． 使用高级语言编写的程序，一般要先用____进行录入和修改，形成程序文件，再用____产生目标模块，最后把有关的目标模块以及库子程序模块用____进行处理形成____。
　　A 编译程序 B 编辑程序
　　C 汇编程序 D 连接程序
　　E 可执行程序 F 源程
　　6.操作系统的作用是____。
　　A 把源程序编译为目标程序
　　B 便于进行目录管理
　　C 控制和管理吸引资源的使用
　　D 实现软硬件的转换
　　7． DOS系统中，负责处理输入和输出的系统文件是____。
　　A IBMDOS.COM B IBMBIOS.COM
　　C BIOS.COM D COMMAND.COM
　　8．在下列文件中，对大多数文字编辑软件都具有通用性的是____。
　　A WPS的文书文件 B WPS的系统文件
　　C WPS的临时文件 D WPS的非文书文件
　　9． 复制数据库ABC.DBF的库结构的命令是____。
　　A COPY STRU <文件名>
　　B COPY TO <文件名> STRUCTURE
　　C DISKCOPY STRUCTURE
　　D CREATE <文件名1> FROM<文件名2>
　　10．FoxBASE中的SET DELETED OFF的作用是____。
　　A 禁止对数据库记录的删除
　　B 显示带删除标记的记录
　　C 使带删除标记的记录隐藏起来
　　D 删除标记对任何命令无影响
　　三、程序题（每1小题20分，共40分）
　　1．下面程序的功能是____。
　　LET I=1
　　DO
　　SET COLOR MOD(1,3)＋1
　　PLOT AREA:RND,RND;RND,RND;RND,RND
　　LET I=I＋1
　　LOOP UNTIL KEY INPUT
　　END

　　 ＄
　　 ＄＄＄
　　 ＄＄＄＄＄
　　 ＄＄＄＄＄＄＄
　　 ＄＄＄＄＄
　　 ＄＄＄
　　 ＄
　　四、论述题（20分）
　　以你应用数据库解决单位或自己实际问题的一个事例，谈谈你对数据库学习和应用的体会。（字数不少于800字）
