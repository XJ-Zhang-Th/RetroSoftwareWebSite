个性化Painter 5.0

　　无论是使用哪一个软件，都要根据自己的创作目的对软件的基本参数和工作环境作一些必要的设置，以便操作起来符合自己的习惯，应用自如，作出具有自身特色的作品。 　　本例将说明Painter 5.0的一些基本设置操作。并简单介绍滤镜处理方法。

　　一、基本设置

　　1．在Painter 5.0中，单击Edit下拉菜单中的“Preferences”命令(属性设置对话框)，弹出菜单，从中可选择相应的作图环境参数设置命令。
　　2．弹出菜单“General”（一般参数设置）：在该对话框中可以设置鼠标指针的显示形状，其形状有十字标和小三角形。在该对话框中还可以设置内定的“Libraries”资（料库），包括“Brushes”（画笔）、“Papers”（纸纹）、“Selections”（选区）、“Floaters”（浮动层）、“Color Set”（颜色板）。
　　3．该对话框其它的设置有“Auto－Save scripts”（自动保存绘画记录）的期限设置，它可决定标尺和文字等尺寸单位的“Units”（图像单位）设定，运行时“Temp File Volume”（临时文件存盘）的设定和“Feather”（像素的羽化）效果设置（其缺省值为16）等。此外还有其它预置选项。
　　4．建立快捷键：通过选择“Preferences”(属性设置)菜单中的“Function Keys”命令，可出现对话框，同时上面将显示当前的快捷键以及它对应的当前功能。
　　Painter可以将“Shift”键和其它功能键（F1，F2，F3等）相结合使用（选择Shift项前的复选框）或单独使用功能键来建立快捷键。完成快捷键选择后，直接在菜单中或浮动控制板中选择特定的功能执行命令即可。
　　“Customize Keys”对话框中的“Set”可用来存储用户新建立的快捷键。“Summary”可显示全部设定的快捷键，以供用户随时进行检索。
　　5．Undo命令设置：Painter在“Preferences”(属性设置)菜单下提供了Undo（重做）对话框，读者可在其中设定软件重做的步数。一般最大步数为32步，但随着Undo的步数的增大，占用的计算机内存也就越多，导致运行速度变慢。

　　二、插入滤镜

　　1．插入式光栅滤镜：为了与其它的图像处理软件如Photoshop相结合，使Painter 5.0的功能变得更为强大，在Painter 5.0中还提供了插入式（Plug－Ins）的滤镜接口，用户可通过选择“Preferences”(属性设置)菜单下的Plug－Ins命令来进行设置，软件将弹出“浏览文件夹”选择框，找到所需要的滤镜后，单击“确定”，即完成了滤镜的外挂。
　　2.要使滤镜发挥作用，则必须关闭Painter 5.0，然后重新启动。进入Painter的窗口后，可在“Effects”下拉菜单中看到新挂上的滤镜命令。
　　下面简单介绍一下滤镜的使用，首先从其它的文件中引入一张图片，选择Painter中的“Select”下拉菜单中的“All”命令，把整幅图作为操作对象进行处理。本例从“Effects”下拉菜单中选择“Splash”命令，然后设置好自己所需要的有关参数，按确定后即可。
　　4．此外，在Painter 5.0的“Preferences”(属性设置)菜单中，还为使用者提供了浮动控制板的外观设计，读者可根据个人的爱好和风格进行相应的设定，直到你满意为止。
