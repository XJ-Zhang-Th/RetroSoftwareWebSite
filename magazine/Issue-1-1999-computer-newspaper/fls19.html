最新Windows Commander 4.01使用概述
●张熙平

　　Windows Commander一直是被大家所推崇的功能强大的Windows文件管理器。最近又推出了最新的4.01版本，让我们看看这个工具新增功能的特点及其操作。

　　一、Windows Commander 4.01 新增功能特点

　　・提供了自动比较与手动切换比较不同位置文件的功能。
　　・在复制文件时，若目标路径下已存在同名文件，提供安全的换名保存功能。
　　・在同步目录功能中，提供指定目录下的子目录同时比较/同步功能。
　　・增加了对BMP格式的图形文件观察支持，在Lister窗口中支持十六进制字符扫描。
　　・支持将一压缩包中打包的文件直接复制到另一压缩包中。支持复制对象到NUL、PRN、LPTx、COMx等硬件端口。
　　・增强了创建文件关联中针对文件类型编辑功能。
　　・支持将传输的文件放到操作列表中，然后集中下载功能。
　　・查找指定区域中的相同文件。
　　・支持用FTP功能传输文件时，连结失败自动重连功能。
　　・在文件复制过程中，若目标盘容量小于当前复制的文件，则自动跳过当前文件。
　　・在FTP传输过程中支持发送特殊命令等，连线操作支持代理服务器。
　　・支持更加详细的文件列表显示信息。且文件列表信息各列的宽度自动调整，使用户看起来更直观。
　　・提供增强的文件选择功能和增强的文件过滤功能，提供增强的文件查找功能。
　　・在Lister文件内容观察窗口中提供自动换行功能。

　　二、Windows Commander常用操作例解

　　1.对象选择
　　Windows Commander提供了两种对象选择方式，即兼容NC的对象选择方式和Windows自身提供的对象选择方案。前者表示使用鼠标右键选择对象，使用鼠标左键操作；后者表示使用鼠标左键选择对象，单击鼠标右键给出系统提供的鼠标右键菜单。
　　如需改变当前的对象选择方式，可单击程序界面菜单条中的Configuration | Options命令，在关联菜单中切换到“Operation”标签项，在该界面中的“Mouse selection mode”项中设置相应的选择方案即可。
　　2.改变对象的属性与创建时间
　　首先在文件窗口中选择需进行操作的文件/文件夹（程序允许一次修改多个对象属性）。单击程序界面菜单条中的File | Change attributes命令，程序会给出“Change attributes”设置界面。在其中设置相应的对象属性/创建日期。需要说明的是，Windows Commander不提供对文件夹的创建日期进行修改功能。
　　3.压缩文件
　　Windows Commander支持六种格式的压缩包，程序内建Zip压缩功能，对于其他格式，需要使用相应的压缩程序才能实现。在此我们以Zip格式为例说明。首先选择需要压缩的对象，按下“Alt + F5”键，程序给出“Pack files”设置界面。在其中设置创建的压缩包名称以及保存路径，在“Packer”项中选择需创建的压缩包格式ZIP，如果需要以全路径方式压缩文件，应选择使用“Also pack path names (only recursed)”复选框，如果要压缩选定文件夹中的子文件夹，应选择使用“Recursively pack subdirectories”复选框，如果要创建分卷压缩文件，应选择使用“Multiple disk archive”复选框，如果要创建自扩展格式的Zip压缩包，可选择使用“Create self extracting ZIP archive”复选框。全部设置结束后，单击OK按钮确定即可。
　　4.压缩文件管理
　　Windows Commander提供了非常强大的压缩包文件管理功能。在程序界面中，用户可象使用文件夹一样使用压缩包和其中的文件，即：使用鼠标双击压缩包文件图标，程序即可在当前窗口中显示压缩文件列表，你也可象使用普通的文件一样对压缩包中打包的文件进行操作。压缩文件运行时，程序会首先给出一个对话框，单击其中的“Unpack and execute”按钮，程序会自动扩展该文件到临时路径并运行它。如果改变了压缩文件内容，退出压缩包时，程序给出一个提示对话框，询问用户是否进行压缩包文件的更新，根据自己的需要选择即可。
　　5.扩展压缩包
　　首先选择需扩展的压缩包文件，然后按下“Alt + F9”键，程序会给出“Unpack files”设置界面。在其中设置压缩文件扩展的目的路径。如果需要以全路径方式进行压缩包文件的扩展，可选择使用“Unpack path names if stored with files”复选框。最后单击程序界面中的OK按钮即可。
　　如果只想扩展压缩包中的部分文件，可首先打开相应的压缩包，在其中选择需进行扩展的压缩文件，之后按下“Alt + F9”键，并按以上提供的方法操作就可以了。
　　6.文件/文件夹管理
　　首先选择需要操作的目标对象，然后根据不同的要求按照以下提供的方法操作：
　　(1)复制：使用鼠标拖曳所选择的对象图标到目标文件夹图标上，松开鼠标左键，程序给出关联的确认对话框，询问用户是否需要将所选择的文件/文件夹复制到当前路径，单击界面中的“OK”按钮确定。此外，也可以使用程序提供的对象拖曳功能实现对象的快速复制，这样更加简单一些。
　　(2)移动：按下“F6”键，并在关联的对话框中设置目标文件夹。如果这个路径很长，可单击界面中的“Tree”按钮，并在关联的系统文件夹列表窗口中选择。
　　(3)换名：使用鼠标按住需换名操作的文件图标不动，稍停一会儿，文件名会变为一个文本框，其中也有一个光标，现在输入需要修改的新名字，然后击回车键确认即可。此外，使用“F6”键也可以实现文件/文件夹的换名。
　　(4)删除：选择好需要操作的对象后，按下“F8”键，并在关联的对话框中确认就可以了。程序默认将删除的文件保存到系统的回收站中，这样可以安全恢复。
　　(5)新建文件夹：按下“F7”键，程序会给出一个对话框，然后在其中键入需要创建的文件夹名就可以了。程序支持创建深层次的文件夹，如需在e:＼mytools路径下创建名为“document”的子文件夹，可直接在“F7”键关联窗口中键入“e:＼mytools＼document”，而无须首先切换到e:＼mytools路径下。
　　(6)文件的拆分与合并：单击界面菜单条中的File | Split File命令，程序给出关联的“Split”对话框，在其中的文本框中设置保存拆分文件的路径，在“Bytes per file”下拉文本框中选择每个分卷文件的容量。单击OK按钮确定。对于该工具拆分的文件，其中包括*.crc文件，这个文件只能由程序自己识别。使用鼠标双击该文件图标，程序会自动进行拆分文件的合并。
　　7.观察文件内容
　　Windows Commander的文件观察功能提供了对文本文件、二进制文件、十六进制文件和BMP图形文件以及部分多媒体文件的支持。观察方法非常简单，选中相应的文件后，按下功能键“F3”即可。在程序界面菜单中，也提供了比较简单的诸如查找字符串、复制文本内存、设置字符显示方式等功能。
　　8.编辑文本文件
　　首先，在程序界面的系统文件列表窗口中选择需进行编辑的文件。然后按下功能键“F4”即可。
　　同样，Windows Commander提供了使用外部编辑器功能，设置方法是：首先单击程序界面中的Configuration | Options命令，并在关联的设置界面中选择“Edit/View”项，切换到该标签项。在其中的“Editor for F4”项设置可供使用的文件编辑器，之后保存设置并退出。
　　9.查找文件
　　首先，按下快捷键“Alt + F7”，程序给出关联的“Find Files”对话框，在“General”标签项中的“Search for”文本框中输入需要查找的对象，在“Search in”项中设置可扫描的范围。如果要扫描驱动器，可单击界面中的“Drives”按钮，并在关联驱动器列表窗口中选取，如果要扫描压缩包中的文件，可选择使用“Search archives”复选框。在该界面中的“Advanced”项中提供了扫描限制设置，在“Load/Save”项中提供了保存扫描设置信息功能。之后单击“Start search”按钮，程序开始扫描操作，并将扫描结果以列表形式显示在界面下部的列表框中。
　　10．同步目录Synchronize Directories
　　这项功能在新版本程序中有所改进，具体步骤如下：
　　单击程序界面菜单条中的Commands | Synchronize Dirs命令，程序给出“Synchronize Directories”设置界面。在界面上边的两个文本框中分别键入需要比较的两个目标文件夹，然后单击界面中的“Compare”按钮。程序会将比较结果以列表形式显示在界面下方的列表框中，并以不同的颜色区分，非常直观。同时在这个列表框中，程序也给出了文件移动方向提示，你可根据自己的需要选择确实需要同步的文件，然后单击界面中的“Synchronize”按钮。此外，在该界面中程序也提供了简单的比较控制，其中包括比较指定文件夹下的子文件夹、比较时忽略文件创建日期等，根据自己的需要选择。
　　11．定义界面语言
　　Windows Commander是英文软件，但是现在有中文界面汉化包可供下载（下载地址是http://zxs.netwalk.net.cn/patch/z-wincmd401.zip），将下载的文件展开，复制到程序安装路径下的Language路径下后，并进行简单设置，界面就变为中文的了。具体设置方法是：单击程序界面菜单条中的Configuration | Options命令，然后切换到“Language”标签项，在该界面中的列表框中选择中文界面语言即可。当设置使用中文界面语言后，Windows Commander的汉字显示效果并不是很好，此时可以切换到“Fonts”标签项，设置显示字库为宋体，字符大小为9。
　　12．FTP功能
　　如果需要链接到指定的FTP服务器，可按下“Ctrl + F”组合键，此时程序会给出“Connect to ftp server”对话框，在其中的“Connect to”列表框中选择需要连接到的FTP服务器，然后单击“Connect”按钮确定。当连接成功后，就可以象使用本机文件一样使用FTP服务器中的文件了。
　　如果需要创建一个新的连接地址，可单击“Connect to ftp server”对话框中的“New URL”按钮，然后在关联的对话框中键入需要连接到的FTP地址，设置是否使用匿名登录“anonymous connection”、是否使用防火墙（代理服务器）“Use firewall (proxy server)”，之后单击OK按钮确定。
　　13．自定义按钮工具条
　　Windows Commander提供了编辑界面按钮条功能，具体方法如下：首先单击程序界面菜单条中的Configuration | button项，程序给出“Change button bar”设置界面。单击“Append”按钮，程序在“Button bar”列表框中给出一个新的按钮（深蓝色的方块），使用鼠标拖动该按钮到一个合适的按钮位置。单击“Command”下拉文本框，在其中选择需添加的项目。如果需添加系统中安装的应用程序的快捷按钮，单击该文本框右边的“>>”按钮，并在关联界面中选择。在“Parameters”文本框中添加该程序启动时的运行参数、在“Start path”文本框中添加该程序启动路径、在“Icon file”文本框中键入程序图标文件、在“Tooltip”文本框中添加按钮提示。最后单击OK按钮确定，回到程序主界面，此时可以看到新增加的按钮已经显示在界面按钮条中了。
